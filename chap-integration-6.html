    Exercises 8.4 Exercises

    1.

    Let \(f\) be a non-negative function on a measure space \((X,\mathcal{F},\mu)\text{,}\) and let \(E \subseteq X\) be a measurable set with \(\mu(E) = 0\text{.}\) Prove
    🔗
    \begin{equation*} \int_X f \,d\mu = \int_{X\backslash E} f\,d\mu. \end{equation*}
    🔗

    2.

    Let \((X,\mathcal{F},\mu)\) be a measure space, and let \(f\) and \(g\) be non-negative measurable functions on \(X\text{.}\) If \(f \leq g\) then
    🔗
    \begin{equation*} \int_X f \,d\mu \leq \int_X g \,d\mu. \end{equation*}
    🔗

    3.

    Let \((X,\mathcal{F})\) be a measurable space. Fix \(x \in X\) and let \(\delta_x\) be the Dirac measure at \(x\text{.}\) If \(f\) is a non-negative measurable function on \(X\text{,}\) what is the value of \(\int_X f \,d \delta_x\text{?}\)
    🔗
    🔗

    4.

    Let \(c\) be the counting measure on the subsets of \(\mathbb{N}\text{,}\) and let \(f\) be a (necessarily measurable) function from \(\mathbb{N}\) to \(\mathbb{R}\text{.}\)
    🔗
    Show that \(f\) is integrable with respect to \(c\) if and only if the series \(\sum_{n=1}^\infty f(n)\) converges absolutely to a finite number. Further show that if \(f\) is integrable then
    🔗
    \begin{equation*} \int_\mathbb{N} f \,dc = \sum_{k=1}^\infty f(n). \end{equation*}
    🔗

    5.

    Let \((X,\mathcal{F},\mu)\) be a measure space, and let \(f\) be a non-negative measurable function on \(X\text{.}\) Let \(E \in \mathcal{F}\) be a measurable set such that \(f(x)>0\) for all \(x \in E\text{.}\) Show that, if \(\mu(E)>0\) then
    🔗
    \begin{equation*} \int_E f\,d\mu > 0. \end{equation*}
    🔗

    6.

    Prove Corollary Corollary 8.1.18: Let \((X,\mathcal{F},\mu)\) be a measure space, let \((f_n)_n\) be a sequence of non-negative measurable functions, and let \(f = \sum_{n=1}^\infty f_n.\) Prove
    🔗
    \begin{equation*} \int_X f \, d\mu = \sum_{k=1}^\infty \int_X f_n\, d\mu. \end{equation*}
    🔗

    7.

    Let \((X,\mathcal{F},\mu)\) be a measure space. Suppose \(f\) and \(g\) are integrable functions such that
    🔗
    \begin{equation*} \int_A f\ d\mu = \int_A g\ d\mu \end{equation*}
    for all measurable sets \(A \subseteq X\text{.}\) Show that \(f = g\) almost everywhere with respect to \(\mu\text{.}\) (HINT: for show that if \(f \geq 0\) and \(\int_Ef\,d\mu = 0\) for all \(E \in F\) then \(f = 0\) a.e.).
    🔗
    🔗

    8.

    Let \((X,\mathcal{F},\mu)\) be a measure space.
    🔗
    1. Let \(f\) be a non-negative measurable function on \(X\text{.}\) Define \(\nu \colon \mathcal{F} \rightarrow [0,\infty]\) by
      \begin{equation*} \nu(E) = \int_E f\, d\mu. \end{equation*}
      Show that \(\nu\) is a measure on \((X,\mathcal{F})\) satisfying \(\nu(E) = 0\) when \(\mu(E) = 0\text{.}\)
      🔗
      🔗
      🔗
    2. Let \(g\) an integrable function on \(X\) with respect to \(\mu\text{.}\) Define \(\sigma \colon \mathcal{F} \rightarrow \mathbb{R}\) by
      \begin{equation*} \sigma(E) = \int_E g \,d\mu. \end{equation*}
      Show that \(\sigma\) is a signed-measure on \((X,\mathcal{F})\) satisfying \(\sigma(E) = 0\) when \(\mu(E) = 0\text{.}\)
      🔗
      🔗
      🔗
