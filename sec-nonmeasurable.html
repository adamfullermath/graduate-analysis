    Section 3.2 Borel sets, Lebesgue measurable sets, and subsets of \(\mathbb{R}\text{.}\)

    Let \(\mathcal{B}\) be the Borel sets of \(\mathbb{R}\text{,}\) and let \(\mathcal{M}\) be the Lebesgue measurable sets of \(\mathbb{R}\text{.}\) We have seen that the following inclusions hold:
    🔗
    \begin{equation*} \mathcal{B} \subseteq \mathcal{M} \subseteq \mathcal{P}(\mathbb{R}). \end{equation*}
    In this section we will see that both these inclusions are strict. I.e. we will see that there exist Lebesgue measurable sets which are not Borel; and there exists subsets of \(\mathbb{R}\) which are not Lebesgue measurable.
    🔗
    Another question arising from Section 3.1 is the following: are there any uncountable Lebesgue measurable sets with \(0\) Lebesgue measure? We know that countable sets have measure \(0\text{,}\) but are the countable sets all of the Lebesgue measure \(0\) sets? We will see presently, that there are uncountable sets with Lebesgue measure \(0\text{.}\)
    🔗
    We will begin with proving the existence of subsets of \(\mathbb{R}\) which are not Lebesgue measurable. The following theorem was published by Giuseppe Vitali (1875 - 1932) in 1905.
    🔗

    Theorem 3.2.1. Vitali.

    Let \(E \subseteq \mathbb{R}\) satisfy \(m^*(E)>0\text{.}\) Then there exists \(C \subseteq E\) such that \(C\) is not Lebesgue measurable.
    🔗
    🔗

    Proof.

    By the countable monotonicity of \(m^*\text{,}\)
    🔗
    \begin{equation*} m^*(E) \leq \sum_{n=1}^\infty m^*(E \cap [-n,n]). \end{equation*}
    Thus, \(E\) contains a bounded subset with positive Lebesgue outer measure. We can thus assume that \(E\) is bounded.
    🔗
    Define a relation \(\sim_\mathbb{Q}\) on \(E\) by the following rule: if \(x,y\in E\) then \(x \sim_\mathbb{Q} y\) if and only if there is a rational number \(q \in \mathbb{Q}\) such that \(x - y = q\text{.}\) It is not hard to see that \(\sim_\mathbb{Q}\) is an equivalence relation on \(E\) (Exercise 3.4.5). Create a set \(C \subseteq E\) by choosing a representative element from each equivalence class of the equivalence relation \(\sim_\mathbb{Q}\text{.}\) Thus \(C \subseteq E\) satisfies: if \(x \in E\) then there exists a unique \(q_x \in \mathbb{Q}\) such that \(x+q_x \in C\text{.}\) It follows that the collection of sets \(\{C + q\}_{q \in \mathbb{Q}}\) are disjoint and cover \(E\text{.}\) Further, as \(E\) is bounded (and there are infinitely many rational numbers between any two rational numbers), there is a bounded, infinite set \(\Lambda \subseteq \mathbb{Q}\) such that
    🔗
    \begin{equation*} E \subseteq \bigcup_{\lambda \in \Lambda} C + \lambda. \end{equation*}
    Suppose now that \(C\) is Lebesgue measurable. Then, since \(C \subseteq E\text{,}\) \(m(C) < \infty\text{.}\) Further, \(C + \lambda\) is Lebesgue measurable for each \(\lambda \in \Lambda\text{,}\) and \(m(C + \lambda) = m(C)\) (Exercise 3.4.2). Thus, \(\bigcup_{\lambda \in \Lambda} C + \lambda\) Lebesgue measurable, as it is a countable union of Lebesgue measurable sets. We also, have that \(\bigcup_{\lambda \in \Lambda} C + \lambda\) is bounded, since \(E\) and \(\Lambda\) are bounded. Thus
    🔗
    \begin{equation*} \infty > m(\bigcup_{\lambda \in \Lambda} C + \lambda) = \sum_{\lambda\in \Lambda} m(C+ \lambda) = \sum_{\lambda\in \Lambda} m(C). \end{equation*}
    Since \(\Lambda\) is an infinite set, it follows that \(m(C) = 0\text{.}\)
    🔗
    Now, by the countable submonotonicity of the Lebesgue outer measure \(m^*\) we have
    🔗
    \begin{equation*} m^*(E) \leq \sum_{\lambda\in\Lambda} m(C+\lambda) = 0. \end{equation*}
    This contradicts, \(m^*(E) > 0\text{,}\) and so \(C\) is not Lebesgue measurable.
    🔗
    🔗

    Remark 3.2.2.

    Theorem 3.2.1 tells us that there are many subsets off \(\mathbb{R}\) which are not Lebesgue measurable. Still, there is something unsatisfactory about the proof. In the proof we construct a set \(C\) which is not Lebesgue measurable, but what does \(C\) look like? It is important to note that the construction of \(C\) given here relies on the axiom of choice. We choose and element from each equivalence class of \(\sim_\mathbb{Q}\text{.}\) This is not, however, a shortcoming of the proof. It is a necessity. There are models of set theory where the axiom of choice is not assumed, where it can be shown that all subsets of the real line are Lebesgue measurable. This is an interesting topic, but is too far into the weeds of set theory for what we are trying to do. We will (happily!) stick to the standard Zermelo–Fraenkel axioms of set theory plus the axiom of choice (ZFC). Thus, we are living in a world where Theorem 3.2.1 holds. This won’t be the last time that the axiom of choice plays a significant role in these notes; but it is perhaps the most famous and consequential.
    🔗
    🔗
    We will now embark on the construction of an uncountable subset of \(\mathbb{R}\) which is Lebesgue measurable with Lebesgue measure \(0\text{.}\)
    🔗

    Example 3.2.3.

    Let \(C_0 = [0,1]\text{.}\) Let \(C_1 = [0,\frac{1}{3}] \cup [\frac{2}{3},1]\text{.}\) Note, that \(C_1\) is \(C_0 \backslash (\frac{1}{3},\frac{2}{3})\text{.}\) I.e. \(C_1\) is \(C_0\) with the middle third of its interval removed. To construct \(C_2\) we remove the (open) middle third from each of the two disjoint intervals in \(C_1\text{.}\) That is
    🔗
    \begin{equation*} C_2 = \left[0,\frac{1}{9}\right] \cup \left[\frac{2}{9},\frac{1}{3}\right]\cup\left[\frac{2}{3},\frac{7}{9}\right]\cup\left[\frac{8}{9},1\right]. \end{equation*}
    We construct \(C_3\) by removing the open middle thirds from each of the 4 intervals in \(C_2\text{.}\) We continue in this way: \(C_n\) is constructed from \(C_{n-1}\) by removing the open middle thirds from each of the \(2^{n-1}\) closed intervals in \(C_{n-1}\text{.}\)
    🔗
    🔗
    Let \(C = \bigcap_{n=0}^\infty C_n\text{.}\) We call \(C\) the Cantor set.
     1 
    The set \(C\) is also referred to as the Cantor middle-third set.
    Since each \(C_n\) is a closed set, \(C\) is also a closed set (as intersections of closed sets are closed sets). It follows that \(C\) is a Borel set, and hence \(C\) is Lebesgue measurable.
    🔗
    Let \(U_k\) be the open set which is removed from \(C_{k-1}\) to get \(C_k\text{.}\) Thus, \(U_1 = (1/3,2/3)\text{,}\) \(U_2 = (1/9,2/9) \cup (7/9,8/9)\text{,}\) etc. Let \(U = \bigcup_{k=1}^\infty U_k\text{.}\) Note that
    🔗
    \begin{equation*} m(U_k) = \frac{2^{k-1}}{3^k}. \end{equation*}
    As the sets \(\{U_k\}_{k=1}^\infty\) are disjoint, we have that
    🔗
    \begin{equation*} m(U) = \sum_{k=1}^\infty \frac{2^{k-1}}{3^k} = \frac{1}{3}\sum_{k=0}^\infty \frac{2^{k}}{3^k} = 1. \end{equation*}
    Now, as \([0,1] = U \cup C\text{,}\) and \(\ell([0,1])=1\text{,}\) it follows that \(m(C) = 0\text{.}\)
    🔗
    We will now look at the cardinality of \(C\text{.}\) Every number in \([0,1]\) has a binary expansion. If we do not consider finite binary expansions, and only consider infinite expansions, the binary expansions are unique.
     2 
    That is we will write the binary expansion \(0.1\) as \(0.011111\ldots\text{.}\)
    Suppose \(c \in C\text{.}\) Construct \(f(c) \in [0,1]\) be the following algorithm: if \(c\) is in the left half \(C_1\text{,}\) then the first digit in the binary expansion of \(f(c)\) is \(0\text{;}\) if \(c\) is in the right interval of \(C_1\) then the first digit of the binary expansion of \(f(c)\) is \(1\text{.}\) Now, in \(C_2\text{,}\) \(f(c)\) is in either the left subinterval, or the right subinterval of the interval \(c\) is in \(C_0\text{.}\) Let \(0\) be the second digit of \(f(c)\) if \(c\) is to the left, \(1\) be the second digit if \(c\) is to the right. Continuing in this way we get \(f(c)\) as a binary expansion. The map \(f\) is a bijection from \(C\) to \([0,1]\text{,}\) hence \(C\) and \([0,1]\) have the same cardinality. As \([0,1]\) is uncountable
     3 
    This is traditionally proved using the famed diagonal argument of Cantor. We also know that \([0,1]\) is uncountable since \(m([0,1]) = 1\) and countable sets have Lebesgue measure \(0\) by Theorem 3.1.3.
    , it follows that \(C\) is uncountable.
    🔗
    🔗
    We thus shown have that the Cantor set \(C\) has Lebesgue measure \(0\) and is uncountable. Hence the converse of Theorem 3.1.3 does not hold. All countable sets have Lebesgue measure \(0\text{,}\) but not all Lebesgue measure \(0\) sets are countable.
    🔗
    We want to prove the existence of Lebesgue measurable sets which are not Borel sets. In order to do this we first introduce a function, the Cantor-Lebesgue function. Whilst we introduce it now for the purpose of finding a set which is not Borel, the Cantor-Lebesgue function is of interest in its own right.
    🔗

    Example 3.2.5.

    Let \(C\) be the Cantor set, from Example 3.2.3, and let \(U = [0,1]\backslash C\text{.}\) We first define a function \(\varphi\) on \(U\text{.}\) Recall that \(U = \bigcup_{k=1}^\infty U_k\) where \(U_k\) is the open set removed from \(C_k\) at the \(kth\) step of the construction of the Cantor set \(C\text{.}\)
    🔗
    Let \(\varphi(x) = 1/2\) for \(x \in U_1 = (1/3,2/3)\text{.}\) For \(x \in U_2\) define \(\varphi(x)\) by
    🔗
    \begin{equation*} \varphi(x) = \begin{cases} \frac{1}{4} & x \in (1/9,2/9) \\ \frac{3}{4} & x \in (7/9,8/9). \end{cases} \end{equation*}
    Continue in this way, so that for \(x \in U_k\)
    🔗
    \begin{equation*} \varphi(x) = \frac{2m-1}{2^k} \end{equation*}
    when \(x\) is in the \(m\)th interval in \(U_k\) (See Figure~\ref{fig: cantor lebesgue}).
    🔗
    🔗
    We extend \(\varphi\) to all of \([0,1]\) by setting \(\varphi(0)=0\text{,}\) and for \(x \in C\backslash\{0\}\text{,}\)
    🔗
    \begin{equation*} \varphi(x) = \sup\{\varphi(y) \colon y \in [0,x) \cap U \}. \end{equation*}
    We call \(\varphi\) the Cantor-Lebesgue function Note that \(\varphi\) is continuous. Indeed, \(\varphi\) is clearly continuous on \(U\text{,}\) since \(\varphi\) is constant on open intervals in \(U\text{.}\) For \(x \in C\text{,}\) and \(K\) large enough, we can find \(a_k < x < b_k\) such that
    🔗
    \begin{equation*} \varphi(b_k) - \varphi(a_k) < \frac{1}{2^k} \end{equation*}
    for all \(k\geq K\text{.}\) It follows that \(\varphi\) does not have a jump discontinuity at \(x\text{.}\) As \(\varphi\) is increasing, this shows that \(\varphi\) is continuous at \(x\text{.}\)
    🔗
    🔗

    Lemma 3.2.7.

    Let \(C\) be the Cantor set. Then there is a set \(A \subseteq C\) such that \(A\) is Lebesgue measurable but \(A\) is not a Borel set.
    🔗
    🔗

    Proof.

    Let \(\varphi\) be the Cantor-Lebesgue function from Example 3.2.5. The function \(\varphi\) is an increasing, continuous function with \(\varphi(0) = 0\) and \(\varphi(1) = 1\text{.}\) It follows from the Intermediate Value Theorem that \(\varphi\) maps \([0,1]\) surjectively onto \([0,1]\text{.}\) Define a function \(\psi \colon [0,1] \rightarrow [0,2]\) by
    🔗
    \begin{equation*} \psi(x) = \varphi(x) + x. \end{equation*}
    Then \(\psi\) is a continuous, strictly increasing function with \(\psi([0,1]) = [0,2]\text{.}\)
    🔗
    Let \(C \subseteq [0,1]\) be the Cantor set, and let \(U = [0,1] \backslash U\text{.}\) By Example 3.2.3, \(m(C) = 0\text{,}\) and hence \(m(U) = 1\text{.}\) Applying \(\psi\) to \(C\) and \(U\) we get disjoint sets \(\psi(C)\) and \(\psi(U)\) satisfying
    🔗
    \begin{equation*} [0,2] = \psi(C) \cup \psi(U). \end{equation*}
    Recall that \(\varphi\) is constructed so that there are disjoint open intervals \(\{I_k\}_{k=1}^\infty\) such that \(U = \bigcup_k I_k\) and \(\varphi\) is constant on each \(I_k\text{.}\) Let \(c_k = \varphi(x)\) when \(c \in I_k\text{.}\) Then, \(\psi(I_k) = c_k + I_k\text{.}\) Hence \(\psi(U) = \bigcup_k(c_k + I_k)\) is a countable union of intervals, and hence \(\psi(U)\) is Lebesgue measurable. Further
    🔗
    \begin{equation*} m(\psi(U)) = \sum_{k=1}^\infty \ell(c_k + I_k) = \sum_{k=1}^\infty \ell(c_k + I_k) = m(U) = 1. \end{equation*}
    As \([0,2]\) and \(\psi(U)\) are Lebesgue measurable, it follows that \(\psi(C)\) is Lebesgue measurable, and \(m(\psi(C)) = m([0,2]) - m(\psi(U)) = 1.\)
    🔗
    Theorem 3.2.1 tells us that there is a set \(B \subseteq \psi(C)\) which is not Lebesgue measurable.
    🔗
    Since \(\psi\) is strictly increasing bijection from \([0,1]\) to \([0,2]\) it has a continuous inverse \(\psi^{-1} \colon [0,2] \rightarrow [0,1]\) (Exercise 3.4.6). Let \(A = \psi^{-1}(B) \subseteq C\text{.}\) As \(\psi(A) = B\) is not Lebesgue measurable, and hence not a Borel set, it follows from Exercise 3.4.7 that \(A\) is not a Borel set. As \(A \subseteq C\) and \(m(C) = 0\text{,}\) \(A\) is a Lebesgue measurable since \((\mathbb{R}, \mathcal{M}, m)\) is a complete measure space.
    🔗
    🔗

    Theorem 3.2.8.

    The \(\sigma\)-algebra of Borel subsets of \(\mathbb{R}\) is strictly contained in the \(\sigma\)-algebra of Lebesgue measurable subsets of \(\mathbb{R}\text{.}\)
    🔗
    🔗

    Proof.

    That the Borel sets \(\mathcal{B}\) are contained in the Lebesgue measurable sets \(\mathcal{M}\) is Theorem 3.1.5. Lemma 3.2.7 gives the existence of a Lebesgue measurable set which is not Borel.

    Corollary 3.2.9.

    The measure space \((\mathbb{R},\mathcal{B},m)\) of the Lebesgue measure restricted to the Borel sets of \(\mathbb{R}\) is not a complete measure space.
    🔗
    🔗

    Proof.

    The Cantor set \(C\) is closed, and hence a Borel set. Further \(m(C)=0\text{.}\) Lemma 3.2.7 shows that there is a set \(A \subseteq C\) which is not a Borel set, hence \((\mathbb{R},\mathcal{B},m)\) is not a complete measure space.
