    Section 1.1 Banach spaces

    Almost everything we cover can just as easily be proved over the field of complex numbers \(\mathbb{C}\text{.}\) Whilst vector spaces can, of course, be defined over other fields, the analysis we are doing is rarely done on fields other than \(\mathbb{R}\) and \(\mathbb{C}\text{.}\) We will thus stick to using the field \(\mathbb{R}\) without comment; suitable adjustments for the field \(\mathbb{C}\) can be done, but won’t be done here.
    🔗

    Definition 1.1.1.

    Let \(V\) be a (real) vector space. A norm on \(V\) is a function \(\|\cdot\| \colon V \rightarrow \mathbb{R}\) satisfying
    🔗
    1. \(\|v\|\geq 0\) for all \(v \in V\text{,}\) and \(\|v\| = 0\) if and only if \(v = 0\text{;}\)
      🔗
      🔗
    2. \(\|\alpha v\| = |\alpha| \|v\|\) for all \(v \in V\) and \(\alpha \in \mathbb{R}\text{;}\) and
      🔗
      🔗
    3. \(\|v+w\| \leq \|v\| + \|w\|\) for all \(v,w\in V\) (the triangle inequality).
      🔗
      🔗
    If \(V\) is a vector space with a norm, we call \(V\) a normed vector space.
    🔗
    🔗

    Example 1.1.2.

    The field \(\mathbb{R}\) itself is a real vector space. The absolute value \(|\cdot|\) defines a norm on \(\mathbb{R}\text{.}\)
    🔗
    More generally, let \(V = \mathbb{R}^n\text{.}\) We can define a norm on \(\mathbb{R}^n\) by
    🔗
    \begin{equation*} \|v\|_2 = \left(\sum_{i=1}^n v_i^2\right)^{1/2} \end{equation*}
    for \(v = (v_1,\ldots,v_n) \in \mathbb{R}^n\text{.}\) This is called the Euclidean norm or the \(2\)-norm on \(\mathbb{R}^n\text{.}\) That \(\mathbb{R}^n\) is a norm is left as an exercise, Exercise 1.4.2.
    🔗
    🔗

    Example 1.1.3.

    Denote by \(C_b(\mathbb{R})\) all continuous, bounded functions on \(\mathbb{R}\text{.}\) Then \(C_b(\mathbb{R})\) is a vector space. Define \(\|\cdot\|_\infty\) on \(C_b(\mathbb{R})\) by
    🔗
    \begin{equation*} \|f\|_\infty = \sup_{x \in \mathbb{R}} |f(x)|. \end{equation*}
    Then \(\|\cdot\|_\infty\) defines a norm on \(C_b(\mathbb{R})\) called the infinity norm or the \(\sup\)-norm.
    🔗
    More generally, if \(X\) is a topological space, the \(C_b(X)\text{,}\) the space of all continuous bounded functions on \(\mathbb{R}\text{,}\) is a vector space and
    🔗
    \begin{equation*} \|f\|_\infty = \sup_{x\in X}|f(x)| \end{equation*}
    defines a norm on \(C_b(X)\text{.}\)
    🔗
    Denote by \(C(X)\) all continuous functions on a topological space \(X\text{.}\) In general, \(\|\cdot\|_\infty\) will not define a norm on \(C(X)\) (since we could hace \(\sup_{x\in X}|f(x)|=\infty\) for some \(f \in C(X)\)). If \(X\) is compact then all continuous functions on \(X\) are bounded. In this case \(C(X) = C_b(X)\text{.}\)
    🔗
    🔗
    If \(V\) is a normed vector space, then we can define a metric, or distance, on \(V\text{.}\) The distance between two vectors \(u,v\in V\) is given by \(\|u-v\|.\) With this metric, we can then study convergence of sequences, just as we do in the real line.
    🔗

    Definition 1.1.4.

    Let \(V\) be a normed vector space, and let \(v\) be a vector in \(V\text{,}\) and let \((v_n)_n\) be a sequence of vectors in \(V\text{.}\) The sequence \((v_n)_n\) is convergent with limit \(v\) with respect to \(\|\cdot\|\text{,}\) or convergent to \(v\) if
    🔗
    \begin{equation*} \lim_{n \rightarrow \infty} \|v - v_n\| = 0. \end{equation*}
    That is, the sequence \((v_n)_n\) converges to \(v\) if for all \(\varepsilon >0\) there is an \(N\) such that if \(n\geq N\text{,}\) then
    🔗
    \begin{equation*} \|v - v_n\|<\varepsilon. \end{equation*}
    🔗
    We can similarly define Cauchy sequences in normed vector spaces.
    🔗

    Definition 1.1.5.

    Let \(V\) be a normed vector space, and let \((v_n)_n\) be a sequence of vectors in \(V\text{.}\) The sequence \((v_n)_n\) is a Cauchy sequence in \(V\) with respect to \(\|\cdot\|\) if for all \(\varepsilon >0\) there is an \(N\) such that if \(n,m\geq N\text{,}\) then
    🔗
    \begin{equation*} \|v_n- v_m\|<\varepsilon. \end{equation*}
    🔗
    Roughly speaking, a sequence of vectors is convergent if they get closer and closer to some vector; and a sequence of vectors is Cauchy if the vectors get closer and closer to each other. The following proposition should thus not come as a surprise. The proof is analogous to the real-line case.
    🔗

    Proposition 1.1.6.

    Let \(V\) be a normed vector space, and let \((v_n)_n\) be a sequence of vectors in \(V\) which converge to some \(v \in V\text{.}\) Then \((v_n)_n\) is a Cauchy sequence.
    🔗
    🔗
    Whilst all convergent sequences are Cauchy, a Cauchy sequence is not necessarily convergent.
    🔗

    Example 1.1.7.

    Let \(V = \mathbb{Q}^n\) and let \(\|\cdot\|_2\) be the \(2\)-norm (restricted from the \(2\)-norm on \(\mathbb{R}^n\text{.}\)) Take a sequence \((a_n)_n\) of rationals which converge to \(\sqrt{2}\text{.}\) E.g. let
    🔗
    \begin{equation*} a_1 = 1.4,\ a_2 = 1.41,\ a_3 = 1.414,\ a_4 = 1.4142, \ldots. \end{equation*}
    For each \(k\text{,}\) let
    🔗
    \begin{equation*} v_k = (a_k,a_k,\ldots,a_k) \in \mathbb{Q}^n. \end{equation*}
    Then \((v_k)_k\) is a Cauchy sequence in \(\mathbb{Q}^n\text{,}\) but does not converge in \(\mathbb{Q}^n\text{.}\) If we view \((v_k)_k\) as a sequence in \(\mathbb{R}^n\text{,}\) then \((v_k)_k\) converges to \((\sqrt{2},\ldots,\sqrt{2}) \notin \mathbb{Q}^n\text{.}\)
    🔗
    🔗
    Recall that all Cauchy sequences in \(\mathbb{R}\) converge. That is, \(\mathbb{R}\) is complete. The completeness of \(\mathbb{R}\) is a fundamental property on which a lot of analysis relies. We are similarly, primarily interested in normed vector spaces in which Cauchy sequences are convergent.
    🔗

    Definition 1.1.8.

    If \(V\) is a normed vector space in which every Cauchy sequence is convergent, then we call \(V\) a Banach space. That is, a Banach space a normed vector space which is complete with respect to the metric induced by the norm.
    🔗
    🔗

    Example 1.1.9.

    The vector space \(\mathbb{R}^n\) with the \(2\)-norm is a Banach space, see Exercise 1.4.4.
    🔗
    🔗

    Example 1.1.10.

    Let \(I \subseteq \mathbb{R}\) be an interval. The normed vector space \(C_b(I)\) with norm \(\|\cdot\|_\infty\) (see Example 1.1.3) is a Banach space, see Exercise 1.4.5
    🔗
    🔗

    Example 1.1.11.

    Let \(I \subseteq \mathbb{R}\) be an interval. A continuous function \(f \colon I\rightarrow \mathbb{R}\) vanishes at infinity if for all \(\varepsilon>0\) there is a closed interval \(K \subseteq I\) such that \(|f(x)|<\varepsilon\) for \(x \in I \backslash K\text{.}\) We denote the collection of all continuous functions on \(I\) that vanish at infinity by \(C_0(I)\text{.}\)
    🔗
    The space \(C_0(I)\) is a vector subspace of \(C_b(I)\text{,}\) thus \(C_0(I)\) is a normed vector space with the \(\sup\)-norm \(\|\cdot\|_\infty\text{.}\) With this norm \(C_0(I)\) is a Banach space, see Exercise 1.4.6.
    🔗
    🔗

    Example 1.1.12.

    Let \(I \subseteq \mathbb{R}\) be an interval. Let \(C_c(I)\) denote all continuous real-valued functions on \(I\) which are compactly supported. That is, a continuous function \(f\) on \(I\) is in \(C_c(I)\) if there is a compact set \(K \subseteq I\) such that \(f(x) = 0\) for \(x \in I \backslash K\text{.}\) Then \(C_c(I)\) a vector space of functions with
    🔗
    \begin{equation*} C_c(I) \subseteq C_0(I) \subseteq C_b(I). \end{equation*}
    However, in general, \(C_c(I)\) is not a Banach space, see Exercise 1.4.9 and Exercise 1.4.10.
    🔗
    🔗
    In many cases, a natural norm on a vector space arises from an inner-product. We will now recall the definition of an inner-product here and explain how we can get a norm from an inner-product.
    🔗

    Definition 1.1.13.

    Let \(V\) be a real-inner product. An inner-product on \(V\) is a bilinear, real-valued map
    🔗
    \begin{equation*} \langle \cdot,\cdot\rangle \colon V \times V \rightarrow \mathbb{R} \end{equation*}
    such that \(\langle v,v\rangle \geq 0\) for all \(v \in V\text{,}\) and \(\langle v,v\rangle =0\) if and only if \(v =0\text{.}\)
    🔗
    If \(V\) is a vector space with an inner-product, then \(V\) is called an inner-product space.
    🔗
    🔗
    If \(V\) is an inner-product space, then we can make \(V\) into a normed vector space. Indeed, if we define
    🔗
    \begin{equation*} \|v\| = \langle v,v\rangle ^{1/2} \end{equation*}
    then it can be shown that this defines a norm on \(V\text{.}\) We call this norm the inner-product norm. We will leave proving the general case that an inner-product norm is always a norm to the Exercises. When they arise in the examples we are interested in, it will be clear that an inner-product norm is a norm.
    🔗

    Definition 1.1.14.

    Let \(V\) be an inner-product space. If \(V\) is complete, i.e. \(V\) is a Banach space, with respect to the inner-product norm, then \(V\) is Hilbert space.
    🔗
    🔗

    Example 1.1.15.

    The dot-product \(\cdot\) on \(\mathbb{R}^n\) is an inner-product and the Euclidean-norm/\(2\)-norm on \(\mathbb{R}^n\) is the inner-product norm induced by the dot-product. As \(\mathbb{R}^n\) is a Banach space with the \(2\)-norm (Example 1.1.9), \(\mathbb{R}^n\) is a Hilbert space.
    🔗
    🔗
    A vector space can have more than one norm. It is possible that two different norms may behave the same when it comes to topological concerns: e.g. convergence of sequences. We introduce the idea of equivalent norms now to deal with this instance.
    🔗

    Definition 1.1.16.

    Let \(V\) be a real vector space, and let \(\|\cdot\|_1\) and \(\|\cdot\|_2\) be two norms on \(V\text{.}\) The norm \(\|\cdot\|_2\) is equivalent to \(\|\cdot\|_2\) if there are scalars \(c,C>0\) such that
    🔗
    \begin{equation*} c \|v\|_1 \leq \|v\|_2 \leq C\|v\|_1 \end{equation*}
    for all \(v \in V\text{.}\)
    🔗
    🔗
    The relation of one norm being equivalent to another does, as the name suggests, form an equivalence relation on the family of all norms on a vector space \(V\text{.}\)
    🔗

    Proposition 1.1.17.

    Let \(V\) be a real vector space, and let \(\|\cdot\|_1\) and \(\|\cdot\|_2\) be two norms on \(V\text{.}\) The norm \(\|\cdot\|_1\) is equivalent to \(\|\cdot\|_2\) if and only if the norm \(\|\cdot\|_2\) is equivalent to \(\|\cdot\|_1\text{.}\)
    🔗
    🔗

    Proof.

    The proof is left as an exercise, Exercise 1.4.8
    🔗
    🔗
    Let \(V\) be a vector space with norms \(\|\cdot\|_1\) and \(\|\cdot\|_2\text{.}\) If \(\|\cdot\|_1\) is equivalent to \(\|\cdot\|_2\) then the norm \(\|\cdot\|_2\) is equivalent to \(\|\cdot\|_1\) by Proposition 1.1.17. In this case we will simply say that \(\|\cdot\|_1\) and \(\|\cdot\|_2\) are equivalent.
    🔗
    Properties like convergence do not rely on the exact value of the norms; analytic definitions usually use inequalities, not equalities. Thus, when a vector space has two equivalent norms, it does not usually matter which one we work with. We formalize this in the following theorem.
    🔗

    Theorem 1.1.18.

    Let \(V\) be a real vector space and let \(\|\cdot\|_1\) and \(\|\cdot\|_2\) be equivalent norms on \(V\text{.}\) Let \((v_n)_n\) be a sequence of vectors in \(V\text{.}\) Then
    🔗
    1. \((v_n)_n\) is a Cauchy sequence with respect to the norm \(\|\cdot\|_1\) if and only if \((v_n)_n\) is a Cauchy sequence with respect to the norm \(\|\cdot\|_2\text{;}\) and
      🔗
      🔗
    2. \((v_n)_n\) is convergent with respect to \(v_0 \in V\) with respect to \(\|\cdot\|_1\) if and only if \((v_n)_n\) is convergent with respect to \(v_0 \in V\) with respect to \(\|\cdot\|_2\text{.}\)
      🔗
      🔗
    In particular, \(V\) is a Banach space under the norm \(\|\cdot\|_1\) if and only if \(V\) is a Banach space under the norm \(\|\cdot\|_2\text{.}\)
    🔗
    🔗

    Proof.

    We will prove (i). Part (ii) is proved similarly. Let \(c,C>0\) be constants such that \(c\|\cdot\|_1\leq \|\cdot\|_2\leq C\|\cdot\|_1\text{.}\) Suppose \((v_n)_n\) is a Cauchy sequence with respect to the norm \(\|\cdot\|_1\text{.}\)
    🔗
    Take \(\varepsilon>0\text{.}\) There exists \(N >0\) such that, if \(n,m\geq N\) then
    🔗
    \begin{equation*} \|v_n - v_m\|_1 < \varepsilon/C. \end{equation*}
    Thus, if \(n,m\geq M\) then
    🔗
    \begin{equation*} \|v_n - v_m\|_2 \leq C\|v_n - v_m\|_1<\varepsilon. \end{equation*}
    Thus \((v_n)_n\) is Cauchy with respect to \(\|\cdot\|_2\text{.}\)
    🔗
    As \(\|\cdot\|_2\) is also equivalent to \(\|\cdot\|_1\text{,}\) a similar argument gives that if \((v_n)_n\) is Cauchy with respect to \(\|\cdot\|_2\) then \((v_n)_n\) is Cauchy with respect to \(\|\cdot\|_1.\)
    🔗
    🔗

    Definition 1.1.19.

    Let \(V\) be a normed vector space, and let \(B \subseteq V\) be a non-empty subset. A set \(S \subseteq B\) is dense in \(B\) if for all \(v \in B\) and \(\varepsilon>0\) there exists \(s \in S\) such that
    🔗
    \begin{equation*} \|v-w\|<\varepsilon. \end{equation*}
    A Banach space \(V\) is called separable if there is countable set \(S \subseteq V\) which is dense in \(V\text{.}\)
    🔗
    🔗
    Many Banach spaces that we study will be separable. Heuristically, a Banach space being separable means that it is not too big. Non-separable spaces tend to be harder to understand.
    🔗

    Example 1.1.20.

    The rationals \(\mathbb{Q}\) are dense in \(\mathbb{R}\) and the rationals are countable, and so \(\mathbb{R}\) is separable. More generally, \(\mathbb{Q}^n\) is a countable dense subset of \(\mathbb{R}^n\) (with the \(2\)-norm), and so \(\mathbb{R}^n\) is separable.
    🔗
    🔗
    The following results show that infinite dimensional Banach spaces can be separable. We first show that the polynomials are dense in the Banach space of continuous functions on a closed interval \([a,b]\text{.}\) This result is known as the Weierstrass Approximation Theorem. We then show that there is a countable subset of polynomials which is dense in \(C([a,b])\text{,}\) and so \(C([a,b])\) is a separable Banach space, with the \(\sup\)-norm.
    🔗

    Theorem 1.1.21. Weierstrass Approximation Theorem.

    Let \([a,b]\subseteq \mathbb{R}\) be a closed interval, let \(f\) be a continuous function \(f\colon [a,b]\rightarrow \mathbb{R}\text{,}\) and let \(\varepsilon>0\text{.}\) There exists a polynomial \(p\) such that
    🔗
    \begin{equation*} \|f-p\|_\infty < \varepsilon, \end{equation*}
    where \(\|\cdot\|_\infty\) is the \(\sup\)-norm on \(C([a,b])\text{.}\)
    🔗
    🔗

    Proof.

    First assume that \(f\) is a continuous function on \([0,1]\text{.}\) I.e., assume \([a,b]=[0,1]\text{.}\) We can further assume that \(f \neq 0\text{,}\) since if \(f = 0\) then \(f\) is already a polynomial and we can take \(p = f\text{.}\)
    🔗
    For each \(n\) and \(0\leq k \leq n\text{,}\) let
    🔗
    \begin{equation*} p_{n,k}(x) = \binom{n}{k}x^k(1-x)^{n-k}, \end{equation*}
    where \(\binom{n}{k}\) is the binomial coefficient \(n\) choose \(k\text{.}\)
    🔗
    We will need several identities involving \(p_{n,k}\text{.}\) Firstly, by the Binomial Theorem,
    🔗
    \begin{equation*} \begin{aligned} 1 = (x + (1-x))^n = \sum_{k=0}^n \begin{pmatrix}n\\l\end{pmatrix}x^k(1-x)^{n-k}=\sum_{k=1}^n p_{n,k}(x). \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation} \sum_{k=1}^n p_{n,k}(x) = 1.\tag{1.1} \end{equation}
    Taking the derivative of (1.1) with respect to \(x\) we find
    🔗
    \begin{equation*} \begin{aligned} 0 &= \sum_{k=0}^n \binom{n}{k}\left[kx^{k-1}(1-x)^{n-k} -(n-k)x^k(1-x)^{n-k-1}\right]\\ &= \sum_{k=0}^n \left[\frac{k}{x}p_{n,k}(x) + \frac{k}{1-x}p_{n,k}(x) -\binom{n}{k}nx^k(1-x)^{n-k-1}\right]\\ &= \sum_{k=0}^n \left[\frac{k}{x(1-x)}p_{n,k}(x)\right] -\frac{n}{1-x}(x + (1-x))^n\\ &= \frac{k}{x(1-x)}\sum_{k=0}^n p_{n,k}(x) - \frac{nx}{x(1-x)}. \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation} \sum_{k=0}^n kp_{n,k}(x) = nx.\tag{1.2} \end{equation}
    Taking the derivative of (1.2) with respect to \(x\) we find
    🔗
    \begin{equation*} \begin{aligned} n &= \sum_{k=0}^n k\binom{k}{n}\left[kx^{k-1}(1-x)^{n-k} -(n-k)x^k(1-x)^{n-k-1}\right]\\ &=\sum_{k=0}^n\left[\frac{k^2}{x}p_{n,k}(x) - \frac{n}{1-x}kp_{n,k}(x) +\frac{k^2}{x-1}p_{n,k}(x) \right] \\ &= \sum_{k=0}^n\left[\frac{k^2}{x(x-1)}p_{n,k}(x)\right] - \frac{n}{x-1}nx, \end{aligned} \end{equation*}
    where (1.2) was used in the last equality. It follows that
    🔗
    \begin{equation} \sum_{n=0}^n k^2p_{n,k}(x) = nx(1-x) + n^2 x^2.\tag{1.3} \end{equation}
    Hence, using (1.1), (1.2) and (1.3), we have that
    🔗
    \begin{equation*} \begin{aligned} \sum_{k=0}^n(k-nx)^2p_{n,k}(x) &= \sum_{k=0}^n(k^2 - 2nkx + n^2x^2)p_{n,k}(x)\\ &= nx(1-x) +n^2x^2 -2nx(nx) + n^2x^2. \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation} \sum_{k=0}^n(k-nx)^2p_{n,k}(x) = nx(1-x).\tag{1.4} \end{equation}
    Consider the polynomials
    🔗
    \begin{equation*} B_n(x) = \sum_{k=0}^n f\left(\frac{k}{n}\right) \binom{n}{k}x^k(1-x)^{n-k} =\sum_{k=0}^n f\left(\frac{k}{n}\right) p_{n,k}(x). \end{equation*}
    We will show that, for large \(n\text{,}\) \(B_n\) approximates \(f\text{.}\)
    🔗
    Since \(f\) is continuous on the closed interval \([0,1]\text{,}\) \(f\) is uniformly continuous. Take \(\varepsilon >0\) and choose \(\delta>0\) such that if \(|x-y|<\delta\) then \(|f(x)-f(y)|<\varepsilon/2.\) Choose \(N\) so that \(N > \frac{\|f\|_\infty}{\varepsilon\delta^2}\text{,}\) and fix \(n \geq N\text{.}\) For \(x \in [0,1]\text{,}\) let
    🔗
    \begin{equation*} C_x = \{ k \colon 1\leq k \leq n,\ \left|k/n -x\right|\geq \delta\} \end{equation*}
    and let
    🔗
    \begin{equation*} D_x = \{ k \colon 1\leq k \leq n,\ \left|k/n -x\right|<\delta\}. \end{equation*}
    Note that, if \(k \in A_x\) for some \(x \in [0,1]\text{,}\) then
    🔗
    \begin{equation*} \frac{(k-nx)^2}{\delta^2n^2}\geq 1. \end{equation*}
    Thus,
    🔗
    \begin{equation} \begin{aligned} \sum_{k \in C_x}p_{n,k}(x) &\leq \sum_{k \in C_x}p_{n,k}(x) \frac{(k-nx)^2}{\delta^2n^2}\\ &\leq \sum_{k=0}^n p_{n,k}(x) \frac{(k-nx)^2}{\delta^2n^2} \\ &= \frac{1}{n^2\delta^2} \sum_{k=0}^n(k-nx)^2p_{n,k}(x) \\ &= \frac{x(1-x)}{n\delta^2}\\ &\leq \frac{1}{4n\delta^2}, \end{aligned}\tag{1.5} \end{equation}
    where the last equality comes from (1.4) and the last inequality follows since \(x(1-x)\leq 1/4\text{,}\) for \(x\in[0,1]\text{.}\)
    🔗
    Now, for \(n\geq N\) and \(x \in [0,1]\) we have
    🔗
    \begin{align*} |B_n(x) - f(x)| =\amp \left| B_n(x) - f(x)\sum_{k=0}^n p_{n,k}(x)\right| \amp\text{ by }\knowl{./knowl/xref/eq-weierstrass-1.html}{\text{(1.1)}}\\ =\amp\left| \sum_{k=0}^n f\left(\frac{k}{n}\right)p_{n,k}(x) - f(x)\sum_{k=0}^n p_{n,k}(x)\right|\\ \leq\amp \sum_{k=0}^n\left|f\left(\frac{k}{n}\right) - f(x)\right|p_{n,k}(x) \\ =\amp \sum_{k\in C_x} \left|f\left(\frac{k}{n}\right) - f(x)\right|p_{n,k}(x)\\ \amp+ \sum_{k\in D_x} \left|f\left(\frac{k}{n}\right) - f(x)\right|p_{n,k}(x)\\ \leq\amp \frac{2\|f\|_\infty}{4n\delta^2} + \sum_{k\in D_x} \left|f\left(\frac{k}{n}\right) - f(x)\right|p_{n,k}(x) \amp \text{by }\knowl{./knowl/xref/eq-weierstrass-5.html}{\text{(1.5)}}\\ <\amp \frac{\|f\|_\infty}{2n\delta^2} + \frac{\varepsilon}{2} \sum_{k\in D_x}p_{n,k} \\ \leq\amp \frac{\|f\|_\infty}{2n\delta^2} + \frac{\varepsilon}{2} \amp\text{by }\knowl{./knowl/xref/eq-weierstrass-1.html}{\text{(1.1)}}\\ <\amp \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon, \end{align*}
    where the last inequality comes from the choice of \(n \geq N > \|f\|_\infty/\varepsilon\delta^2\)
    🔗
    We have proved the result when \([a,b] = [0,1]\text{.}\) Consider an arbitrary closed interval \([a,b]\) and take \(f \in C([a,b])\text{.}\) The polynomial
    🔗
    \begin{equation*} \varphi \colon x \mapsto a + x(b-a) \end{equation*}
    is a homeomorphism from \([0,1]\) to \([a,b]\) with inverse
    🔗
    \begin{equation*} \varphi^{-1} \colon x \mapsto \frac{x-a}{b-a} \end{equation*}
    By the above argument, given \(\varepsilon>0\) there is a polynomial \(p\) on \([0,1]\) such that
    🔗
    \begin{equation*} \|f\circ \psi - p\|_\infty<\varepsilon. \end{equation*}
    Letting \(q\) be the polynomial on \([a,b]\) defined by \(q = p\circ\varphi^{-1}\text{.}\) Then
    🔗
    \begin{equation*} \|f-q\|_\infty = \|f\circ\varphi -q\|_\infty < \varepsilon. \end{equation*}
    Hence the polynomials are dense in \(C([a,b])\text{.}\)
    🔗
    🔗
    That \(C([a,b])\) is a separable space is a straightfoward corollary to the Weierstrass Approximation Theorem.
    🔗

    Corollary 1.1.22.

    Let \([a,b]\) a closed bounded interval in \(\mathbb{R}\text{.}\) Then the continuous functions on \([a,b]\) with the \(\sup\)-norm is a separable Banach space.
    🔗
    🔗

    Proof.

    That \(C([a,b])\) is a Banach space is discussed in Example 1.1.11. If \(a=b\text{,}\) then \(C([a,b]) = \mathbb{R}\text{,}\) which is separable.
    🔗
    Assume \(b>a\text{.}\) Take any \(f \in C([a,b])\) and \(\varepsilon>0\) By the Weierstrass Approximation Theorem, there is a polynomial \(p\) on \([a,b]\) such that
    🔗
    \begin{equation*} \|f- p\|_\infty < \varepsilon/2. \end{equation*}
    Suppose \(p\) has degree \(n\text{.}\) Then there are scalars \(a_0,\ldots,a_n\text{,}\) with \(a_0 \neq 0\text{,}\) such that
    🔗
    \begin{equation*} p(x) = a_0 x^n + a_1x^{n-1} + \ldots + a_{n-1}x + a_n. \end{equation*}
    Choose rationals \(q_0,\ldots,q_n\) such that
    🔗
    \begin{equation*} |a_k -q_k| < \frac{\varepsilon}{2(n+1) (|a|+|b|)^k}. \end{equation*}
    Let \(q\) be the polynomial
    🔗
    \begin{equation*} q(x) = q_0 x^n + q_1x^{n-1} + \ldots + q_{n-1}x + q_n. \end{equation*}
    Then
    🔗
    \begin{equation*} \begin{aligned} \|q-p\|_\infty &= \sup_{x\in[a,b]}|p(x) - q(x)|\\ &\leq \sup_{x\in[a,b]}\sum_{k=0}^n|a_k - q_k||x|^k\\ &\leq \sup_{x\in[a,b]}\sum_{k=0}^n|a_k - q_k|(|a|+|b|)^k\\ &< \varepsilon/2. \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation*} \|f-q\|_\infty \leq \|f-p\|_\infty + \|p-q\|_\infty < \varepsilon. \end{equation*}
    We have shown that the polynomials with rational coefficients are dense in \(C([a,b])\text{.}\) The cardinality of the polynomials with rational coefficients is
    🔗
    \begin{equation*} \bigcup_{n=1}^\infty \mathbb{Q}^n, \end{equation*}
    which is countable. Thus \(C([a,b])\) is separable.
