    Section 8.1 Integration of non-negative functions

    Definition 8.1.1.

    Let \((X,\mathcal{F},\mu)\) be a measure space. Let \(\varphi = \sum_{k=1}^n a_n \chi_{E_n}\) be a non-negative simple function on \(X\) where \(\{a_1,\ldots,a_n\} = \varphi(X)\) are distinct scalars and \(\{E_k\}_{k=1}^n\) are disjoint measurable functions. We define the integral of \(\varphi\) with respect to \(\mu\), denote \(\int_X \varphi d\mu\) as
    🔗
    \begin{equation*} \int_X \varphi\, d\mu = \sum_{k=1}^n a_k \mu(E_k). \end{equation*}
    If \(E \subseteq X\) is a measurable set we define the integral of \(\varphi\) over \(E\) as
    🔗
    \begin{equation*} \int_E \varphi\,d\mu = \int_X \varphi\chi_E\,d\mu. \end{equation*}
    🔗
    Note that, the condition that \(\varphi\) is non-negative is necessary in order to guarantee that \(\sum_{k=1}^n a_k \mu(E_k)\) is defined. If \(\mu(E_1) = \infty\) and \(\mu(E_2) = \infty\) then
    🔗
    \begin{equation*} (1) \mu(E_1) + (-1) \mu(E_2) \end{equation*}
    is not defined.
    🔗

    Lemma 8.1.2.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(\varphi\) be a non-negative simple function. If
    🔗
    \begin{equation*} \varphi = \sum_{k=1}^n b_k \chi_{F_k} \end{equation*}
    where \(b_1,\ldots,b_n\) are (not necessarily distinct) scalars and \(\{F_k\}_{k=1}^n\) are disjoint measurable sets, then
    🔗
    \begin{equation*} \int_X \varphi\, d\mu = \sum_{k=1}^n b_k \mu(F_k). \end{equation*}
    🔗

    Proof.

    Let \(\{a_1,\ldots,a_m\} = \varphi(X)\) be distinct scalars and \(E_1,\ldots, E_m\) be disjoint sets such that \(\varphi = \sum_k a_k \chi_{E_k}\text{.}\) Assume \(X = \bigcup_{k=1}^n F_k\text{,}\) (by adding \(0\) to \(\{b_1,\ldots,b_m\}\) and \(X \backslash \bigcup_k F_k\) to \(\{F_1,\ldots,F_n\}\) if necessary).
    🔗
    Note that for each \(1\leq j \leq n\text{,}\) \(b_j \in \{a_1,\ldots,a_m\}\text{.}\) Thus, for \(1\leq k \leq m\)
    🔗
    \begin{equation*} E_k = \bigcup_{j\colon b_j=a_k}F_j. \end{equation*}
    Hence
    🔗
    \begin{equation*} \mu(E_k) = \sum_{j \colon b_j = a_k} \mu(F_j). \end{equation*}
    Hence
    🔗
    \begin{equation*} \begin{aligned} \int_X \varphi\,d\mu &=\sum_{k=1}^m a_k \mu(E_k)\\ &= \sum_{k=1}^m a_k \sum_{j \colon b_j = a_k}\mu(F_j) \\ &= \sum_{k=1}^m \sum_{j \colon b_j = a_k}b_j\mu(F_j) \\ &= \sum_{j=1}^n b_j \mu(F_j). \end{aligned} \end{equation*}
    🔗

    Theorem 8.1.3.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(\varphi\) and \(\psi\) be non-negative simple functions on \(X\text{.}\) Then, for any scalars \(\alpha, \beta \geq 0\text{,}\)
    🔗
    \begin{equation*} \int_X(\alpha \varphi + \beta \psi) \,d\mu = \alpha \int_X\varphi \,d\mu + \beta \int_X \psi\, d\mu. \end{equation*}
    Further, integration is monotonic: if \(\varphi \leq \psi\) then
    🔗
    \begin{equation*} \int_X \varphi\, d\mu \leq \int_X \psi\, d\mu. \end{equation*}
    🔗

    Proof.

    That \(\int_X \alpha \varphi\, d\mu = \alpha \int_X \varphi \,d\mu\) is straightforward.
    🔗
    Let \(\varphi\) and \(\psi\) be simple functions. Let \(\{a_1,\ldots,a_m\} = \varphi(X)\text{,}\) \(\{b_1,\ldots,b_n\} = \psi(X)\text{,}\) \(E_i = \varphi^{-1}(\{a_i\})\) for \(1 \leq i \leq m\text{,}\) and \(F_j = \psi^{-1}(\{b_j\})\text{.}\) Then
    🔗
    \begin{equation*} \varphi + \psi = \sum_{i=1}^m \sum_{j=1}^n(a_i + b_j) \chi_{E_i \cap F_j}. \end{equation*}
    By Lemma Lemma 8.1.2,
    🔗
    \begin{equation*} \begin{aligned} \int_X(\varphi + \psi)\, d\mu &= \sum_{i=1}^m \sum_{j=1}^n (a_i + b_j) \mu(E_i \cap F_j) \\ &= \sum_{i=1}^m\sum_{j=1}^n a_i\mu(E_i \cap F_j) + \sum_{j=1}^n\sum_{i=1}^m b_j\mu(E_i \cap F_j) \\ &= \sum_{i=1}^m a_i \sum_{j=1}^n \mu(E_i \cap F_j) + \sum_{j=1}^n b_j\sum_{i=1}^m \mu(E_i \cap F_j) \\ &= \sum_{i=1}^m a_i \mu(E_i) + \sum_{j=1}^n b_j \mu(F_j) \\ &= \int_X \varphi\,d\mu + \int_X\psi\,d\mu. \end{aligned} \end{equation*}
    Now suppose that \(\varphi\) and \(\psi\) are non-negative simple functions with \(\varphi \leq \psi\text{.}\) Then \(\psi - \varphi \geq 0\text{.}\) It follows that
    🔗
    \begin{equation*} 0 \leq \int_X(\psi - \varphi)\,d\mu = \int_X \psi\,d\mu - \int_X\varphi\,d\mu. \end{equation*}
    Hence \(\int_X \psi\,d\mu \geq \int_X \varphi\,d\mu\text{.}\)
    🔗
    🔗

    Remark 8.1.4.

    Let \(E_1,\ldots,E_n\) be a collection of measurable sets, let \(a_1,\ldots,a_n\) be non-negative real numbers, and let \(\varphi = \sum_{k=1}^n a_k \chi_{E_k}.\) By Theorem Theorem 8.1.3,
    🔗
    \begin{equation*} \int_X \varphi \, d\mu = \sum_{k=1}^n a_k \int_X \chi_{E_k}\,d\mu = \sum_{k=1}^n a_k \mu(E_k). \end{equation*}
    That is, if \(\varphi = \sum_{k=1}^n a_k \chi_{E_k}\) is a simple function we do not need the sets \(\{E_k\}_{k=1}^n\) to be disjoint in order to easily calculate \(\int_X \varphi \,d\mu.\)
    🔗
    🔗
    The following corollary is immediate on noting that
    🔗
    \begin{equation*} \chi_{A\cup B} = \chi_A + \chi_B \end{equation*}
    for disjoint sets \(A\) and \(B\text{.}\)
    🔗

    Corollary 8.1.5.

    If \(\varphi\) non-negative simple function on a measure space \((X,\mathcal{F},\mu)\) and \(A, B \subseteq X\) are disjoint sets, then
    🔗
    \begin{equation*} \int_{A \cup B} \varphi \,d\mu = \int_A \varphi\, d\mu + \int_B\varphi\, d\mu. \end{equation*}
    🔗
    The next corollary will be used frequently. It tells us that the behavior of a function on a set of measure \(0\) does not affect the integral.
    🔗

    Corollary 8.1.6.

    If \(\varphi\) is a simple function on a measure space \((X,\mathcal{F},\mu)\) and \(E \subseteq X\) satisfies \(\mu(E)= 0\) then
    🔗
    \begin{equation*} \int_X \varphi\,d\mu = \int_{X\backslash E} \varphi\, d\mu. \end{equation*}
    🔗
    We will now define the integral of non-negative measurable functions.
    🔗

    Definition 8.1.7.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be a non-negative measurable function on \(X\text{.}\) Define the integral of \(f\) with respect to \(\mu\) by
    🔗
    \begin{equation*} \int_X f\,d\mu = \sup\left\{ \int_X \varphi\,d\mu \colon 0\leq \varphi \leq f,\ \varphi\text{ simple.}\right\} \end{equation*}
    If \(E \subseteq X\) is a measurable set we define the integral of \(f\) over \(E\) as
    🔗
    \begin{equation*} \int_E f\,d\mu = \int_X f\chi_E\,d\mu. \end{equation*}
    A non-negative measurable function \(f\) on \(X\) is integrable if
    🔗
    \begin{equation*} \int_X f\,d\mu < \infty. \end{equation*}
    When \(X \subseteq \mathbb{R}\) is a Lebesgue measurable set, \(\mathcal{F}\) is the Lebesgue measurable subsets of \(X\text{,}\) and \(\mu = m\) is the Lebesgue measure, we refer to the integral with respect to \(m\) as the Lebesgue integral. If \(f\) is an integrable function with respect to \(m\) we say that \(f\) is Lebesgue integrable.
    🔗
    🔗
    The proof of the following two propositions are left to Exercise Exercise 8.4.1 and Exercise Exercise 8.4.2, respectively.
    🔗

    Proposition 8.1.8.

    Let \((X,\mathcal{F},\mu)\) be a measure space, let \(f\) be non-negative measurable function on \(X\text{,}\) and let \(E \subseteq X\) be a measurable set with \(\mu(E) = 0\text{.}\) Then
    🔗
    \begin{equation*} \int_X f\, d\mu = \int_{X\backslash E} f\, d\mu. \end{equation*}
    🔗

    Proposition 8.1.9. Monotonicity of integration.

    Let \((X,\mathcal{F},\mu)\) be a measure space, and let \(f\) and \(g\) be non-negative measurable functions on \(X\text{.}\) If \(f \leq g\) then
    🔗
    \begin{equation*} \int_X f \,d\mu \leq \int_X g \,d\mu. \end{equation*}
    🔗
    Note that it may be possible for the integral of a non-negative measurable function to be infinite. That is \(f\geq 0\) does not need to integrable in order to calculate \(\int_X f\,d\mu\text{.}\) If \(\varphi =\sum_{k=1}^n a_k \chi_{E_k}\) is a simple function (where \(E_1,\ldots,E_n\) are measurable sets), then \(\int_X \varphi\, d\mu = \infty\) if and only if \(\mu(E_k) = \infty\) for some \(k\text{.}\) If \(f \geq 0\text{,}\) it is possible to have \(\int_X f \,d\mu = \infty\) even if \(\int_X \varphi \,d\mu < \infty\) for all simple function \(0 \leq \varphi \leq f\text{.}\) The following example illustrates this point.
    🔗

    Example 8.1.10.

    Consider the function \(f(x) = \int\frac{1}{x}\) on the interval \((0,\infty)\text{.}\) Let \(m\) be the Lebesgue measure on \((0,\infty)\text{.}\) We will show \(\int_{(0,\infty)} f \,dm = \infty\) whilst \(\int_{(0,\infty)}\varphi\,dm <\infty\) for all simple \(0 \leq \varphi \leq f\text{.}\)
    🔗
    First, suppose \(\varphi = \sum_{k=1}^n a_k \chi_{E_k}\) is a simple function with \(0 \leq \varphi \leq f\text{.}\) For each \(k\text{,}\) \(a_k \leq \frac{1}{x}\) for all \(x \in E_k\text{.}\) Since \(\lim_{x \rightarrow \infty}\frac{1}{x} = 0\text{,}\) it follows that if \(a_k > 0\text{,}\) then \(E_k\) is a bounded set. Thus \(m(E_k) < \infty\) for each \(k\) when \(a_k >0\text{,}\) and so \(\int_{(0,\infty)}\varphi\,dm < \infty\text{.}\)
    🔗
    For each \(n\) define a simple function
    🔗
    \begin{equation*} \phi_n(x) = \begin{cases} \frac{1}{\lceil x\rceil}& x \leq n \\ 0 & x >n \end{cases}, \end{equation*}
    where \(\lceil\cdot\rceil\) denotes the ceiling function (i.e. round to the nearest integer above). Then \(\phi_n \leq f\) for each \(n\text{.}\)
    For each \(n\) we have
    🔗
    🔗
    \begin{equation*} \int_{(0,\infty)} \phi_n\,dm =\sum_{k=1}^n \frac{1}{k}, \end{equation*}
    Thus, for each \(n\text{,}\)
    🔗
    \begin{equation*} \int_{(0,\infty)}f \,dm \geq \sum_{k=1}^n \frac{1}{k} \end{equation*}
    and so
    🔗
    \begin{equation*} \int_{(0,\infty)}f \,dm \geq \sum_{k=1}^\infty \frac{1}{k}. \end{equation*}
    As the harmonic series \(\sum_k \frac{1}{k}\) is not convergent, it follows that
    🔗
    \begin{equation*} \int_{(0,\infty)} f\,d\mu = \infty. \end{equation*}
    🔗
    If \(f\) is integrable, then \(f\) will be, in some sense, well-behaved. We make this explicit in Theorem Theorem 8.1.13 after the following important lemma.
    🔗

    Lemma 8.1.12. Chebyshev’s Inequality.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be a non-negative measurable function. Then, for each real number \(\lambda > 0\text{,}\)
    🔗
    \begin{equation*} \mu(\{x \in X \colon f(x) \geq \lambda\}) \leq \frac{1}{\lambda} \int_X f d\mu. \end{equation*}
    🔗

    Proof.

    Fix \(\lambda >0\) and let
    🔗
    \begin{equation*} X_\lambda = \{x \in X \colon f(x) \geq \lambda \}. \end{equation*}
    Note that, since \(f\) is measurable, \(X_\lambda\) is measurable. Let \(\varphi = \lambda \chi_{X_\lambda}\text{.}\) Then \(\varphi\) is simple and \(\varphi \leq f\text{.}\) Further \(\int_X \varphi\,d\mu = \lambda \mu(X_\lambda)\text{.}\) Thus
    🔗
    \begin{equation*} \lambda \mu(X_\lambda) \leq \int_X f\, d\mu. \end{equation*}
    Dividing the above inequality by \(\lambda\) gives the desired result.
    🔗
    🔗

    Theorem 8.1.13.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be a non-negative measurable function such that \(\int_X f\, d\mu < \infty\text{.}\) Then \(f\) is finite almost everywhere and the set
    🔗
    \begin{equation*} \{x \in X \colon f(x)>0 \} \end{equation*}
    is \(\sigma\)-finite.
    🔗
    🔗

    Proof.

    Let \(X_\infty = \{x \in X \colon f(x) = \infty\}\text{.}\) To show that \(f\) is finite a.e. we must show that \(\mu(X_\infty) = 0\text{.}\) For each \(n\text{,}\) let \(X_n = \{x \in X \colon f(x) \geq n\}\) and let \(\phi_n = n \chi_{X_n}\text{.}\) Then \(\phi_n \leq f\) and so
    🔗
    \begin{equation*} n \mu(X_n) = \int_X \phi_n\,d\mu \leq \int_X f\, d\mu<\infty. \end{equation*}
    Thus
    🔗
    \begin{equation*} \mu(X_n) \leq \frac{1}{n}\int_X f\,d\mu \end{equation*}
    and so \(\lim_n \mu(X_n) = 0\text{.}\) Note that \(X_\infty = \bigcap_n X_n\text{,}\) and so by the continuity of measure
    🔗
    \begin{equation*} \mu(X_\infty) = \lim_{n\rightarrow \infty} \mu(X_n) = 0, \end{equation*}
    and hence \(f\) is finite a.e.
    🔗
    Let \(X_{>0} = \{x \in X \colon f(x) > 0\}\) Note that
    🔗
    \begin{equation*} X_{>0} = \bigcup_{n=1}^\infty \{x \in X \colon f(x) \geq \frac{1}{n}\}. \end{equation*}
    By Chebshev’s Inequality,
    🔗
    \begin{equation*} \mu\left(\left\{x \in X \colon f(x) \geq \frac{1}{n}\right\}\right)\leq n \int_X f\,d\mu<\infty. \end{equation*}
    Thus \(X_{>0}\) is a countable union of sets of finite measure. That is, \(X_{>0}\) is \(\sigma\)-finite.
    🔗
    🔗
    In Section Section 3.2 we denounced the Riemann integral for not behaving well with limits of functions. In particular, we have no significant theorems on the relationship between the Riemann integral and pointwise limits of functions. So how does our integral behave with pointwise limits, or pointwise a.e. limits? Explicitly, if \((f_n)_n\) is a sequence of non-negative measurable functions converging pointwise a.e. to a measurable function \(f\text{,}\) do we have
    🔗
    \begin{equation*} \lim_{n\rightarrow \infty} \int_X f_n \,d\mu = \int_X f \,d\mu? \end{equation*}
    The answer is, of course: not necessarily. We give some counter examples here.
    🔗

    Example 8.1.14.

    Let \(f = 0\) on \(\mathbb{R}\) and let \(m\) be the Lebesgue measure.. Then \(\int_{\mathbb{R}}f\,dm = 0.\) We define sequences \((f_n)_n\text{,}\) \((g_n)_n\) and \((h_n)_n\) on \(\mathbb{R}\) by
    🔗
    \begin{equation*} \begin{aligned} f_n &= \chi_{[n,n+1]}; \\ g_n &= 2^{n(\bmod 2)}\chi_{[n,n+1]};\text{ and}\\ h_n &= n \chi_{[n,n+1]}. \end{aligned} \end{equation*}
    Each of the sequences \((f_n)_n\text{,}\) \((g_n)_n\) and \((h_n)_n\) converge pointwise to \(f = 0\text{.}\) Further
    🔗
    \begin{equation*} \begin{aligned} \int_{\mathbb{R}} f_n\,dm &= 1;\\ \int_{\mathbb{R}} g_n\,dm &= \begin{cases} 1 & n\text{ even},\\ 2 & n\text{ odd};\end{cases}\text{ and}\\ \int_{\mathbb{R}} h_n\,dm &= n. \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation*} \begin{aligned} \lim_{n\rightarrow \infty}\int_{\mathbb{R}} f_n\,dm &= 1; \\ \lim_{n\rightarrow \infty}\int_{\mathbb{R}} g_n\,dm & \text{ does not exist; and}\\ \lim_{n\rightarrow \infty}\int_{\mathbb{R}} h_n\,dm &= \infty. \end{aligned} \end{equation*}
    In conclusion, the answer to () can be no, for a variety of reasons. The limit on the left-hand side can be finite, it may not exist, or it may be infinite. In each of these cases, () may not hold.
    🔗
    🔗
    The obvious moral to take away from Example Example 8.1.14 is that, in order for there to be a positive answer to () we need some additional conditions on the functions \((f_n)_n\) and \(f\text{.}\) There is, however, a less obvious, but important point that is worth observing. We first recall some terminology for sequences of numbers. Let \((x_n)_n\) be a sequence of real numbers. The limit superior of \((x_n)_n\) is defined as
    🔗
    \begin{equation*} \liminf_n x_n = \lim_n \inf\{x_k \colon k \geq n\} = \sup_n\{ \inf\{x_k \colon k \geq n\}\} \end{equation*}
    and the limit superior is defined as
    🔗
    \begin{equation*} \limsup_n x_n = \lim_n \sup\{x_k \colon k \geq n\} = \inf_n\{ \sup\{x_k \colon k \geq n\}\}. \end{equation*}
    The limit inferior and limit superior always exist (with \(-\infty\) and \(\infty\) as possible values). It is always the case that
    🔗
    \begin{equation*} \liminf x_n \leq \limsup x_n. \end{equation*}
    Further \(\liminf x_n = \limsup x_n\) if and only if \((x_n)\) is a convergent sequence, in which case
    🔗
    \begin{equation*} \liminf x_n = \lim x_n = \limsup x_n. \end{equation*}
    Returning to Example Example 8.1.14. Whilst we do not have that \(\int_{\mathbb{R}} f \,dm\) is a limit of the integrals of the functions which converge pointwise to \(f\text{,}\) we do have the following inequalities:
    🔗
    \begin{equation*} \begin{aligned} 0 = \int_{\mathbb{R}} f \,dm &\leq \liminf_n \int_{\mathbb{R}} f_n \,dm =1 \\ 0 = \int_{\mathbb{R}} f \,dm &\leq \liminf_n \int_{\mathbb{R}} g_n \,dm =1 \\ 0 = \int_{\mathbb{R}} f \,dm &\leq \liminf_n \int_{\mathbb{R}} h_n \,dm =\infty. \end{aligned} \end{equation*}
    That these inequalities hold for each of our sequences is not a coincidence.
    🔗

    Lemma 8.1.15. Fatou’s Lemma.

    Let \((X,\mathcal{F},\mu)\) be a measure space, and let \((f_n)_n\) be a sequence of non-negative measurable functions which converge pointwise a.e. to a measurable function \(f\text{.}\) Then
    🔗
    \begin{equation*} \int_X f\,d\mu \leq \liminf \int_X f_n\,d\mu. \end{equation*}
    🔗

    Proof.

    Let \(E \subseteq X\) be a measure \(0\) set such that \((f_n)_n\) converges to \(f\) pointwise on \(X \backslash E\text{.}\) By Proposition Proposition 8.1.8, replacing \(X\) with \(X \backslash E\) does not change the value of any integrals. We can thus assume that \((f_n)_n\) converges pointwise to \(f\) everywhere on \(X\text{.}\)
    🔗
    Let \(\varphi\) be a simple function satisfying \(0 \leq \varphi \leq f\text{.}\) We will show that
    🔗
    \begin{equation*} \int_X \varphi\,d\mu \leq \liminf \int_X f_n\, d\mu. \end{equation*}
    First, assume that \(\int_X \varphi\,d\mu = \infty\text{.}\) In this case, there is a measurable set \(E \subseteq X\) and a real number \(a > 0\) such that \(\mu(E) = \infty\) and \(\varphi(x) = a\) for all \(x \in E\text{.}\) For each \(n\) let
    🔗
    \begin{equation*} E_n = \left\{x \in X \colon f_k(x) \geq \frac{a}{2} \text{ for }k\geq n\right\}. \end{equation*}
    Note that \(E \subseteq \bigcup_n E_n\) since, for \(x \in E\text{,}\) \(\lim_n f_n(x) = f(x) \geq a\text{.}\)
    🔗
    As the sets \(\{E_n\}_n\) are ascending (and measurable) we have, by the continuity of measure,
    🔗
    \begin{equation*} \infty = \mu(E) \leq \lim_n \mu(E_n). \end{equation*}
    By Chebyshev’s Inequality,
    🔗
    \begin{equation*} \mu(E_n)) \leq \frac{2}{a} \int_X f_n\,d\mu \end{equation*}
    and hence \(\lim_n \int_X f_n\, d\mu = \infty.\) Thus
    🔗
    \begin{equation*} \int_X f\, d\mu\geq \int_X \varphi\, d\mu = \infty = \lim_n \int_X f_n\, d\mu = \liminf_n \int_X f_n\, d\mu. \end{equation*}
    Thus \(\int_X f \,d\mu = \liminf_n \int_X f_n\, d\mu.\)
    🔗
    Assume now that \(\int_X \varphi \, d\mu < \infty\text{.}\) Let \(P = \{x \in E \colon \varphi(x) > 0\}.\) Since \(\varphi\) is simple and \(\int_X \varphi\,d\mu < \infty\) we must have that \(\mu(P)<\infty\text{.}\) Fix \(\varepsilon >0\text{.}\) For each \(n\text{,}\) let
    🔗
    \begin{equation*} P_n = \{x \in P \colon f_k(x) > (1-\varepsilon)\varphi(x)\text{ for }k\geq n\}. \end{equation*}
    The sequence \(\{P_n\}_n\) is an ascending sequence of measurable sets with \(P = \bigcup_n P_n\text{.}\) Hence \(\{P\backslash P_n\}_n\) is a descending sequence of measurable sets with intersection \(\emptyset\text{.}\) By the continuity of measure, since \(\mu(P)<\infty\text{,}\)
    🔗
    \begin{equation*} \lim_n \mu(P \backslash P_n) = 0. \end{equation*}
    Choose \(N\) such that \(\mu(P\backslash P_n) < \varepsilon\) for all \(n \geq N\text{.}\) Let \(M\) be the maximum value \(\varphi\) takes. For any \(n \geq N\) we have that
    🔗
    \begin{equation*} \begin{aligned} \int_X f_n \, d\mu &\geq \int_{P_n} f_n \, d\mu \\ &\geq \int_{P_n}(1-\varepsilon) \varphi\, d\mu \\ &= (1-\varepsilon) \left[\int_P \varphi\,d\mu - \int_{P\backslash P_n} \varphi\, d\mu\right] \\ &\geq (1-\varepsilon)\int_P \varphi\,d\mu -\int_{P\backslash P_n} \varphi\, d\mu\\ &= (1-\varepsilon)\int_X \varphi\,d\mu -\int_{P\backslash P_n} \varphi\, d\mu\\ &\geq (1-\varepsilon)\int_X \varphi\,d\mu -M\mu(P\backslash P_n)\\ &>(1-\varepsilon)\int_X \varphi\,d\mu -M\varepsilon \\ &= \int_X \varphi\,d\mu -\varepsilon\left(\int_X\varphi\,d\mu + M\right). \end{aligned} \end{equation*}
    Summarizing the above, for \(n\geq N\text{,}\) we have
    🔗
    \begin{equation*} \int_X f_n \,d\mu \geq \int_X \varphi\,d\mu -\varepsilon\left(\int_X\varphi\,d\mu + M\right). \end{equation*}
    Hence
    🔗
    \begin{equation*} \liminf_n \int_X f_n \,d\mu \geq \int_X \varphi\,d\mu -\varepsilon\left(\int_X\varphi\,d\mu + M\right). \end{equation*}
    Since \(\varepsilon>0\) is arbitrary, and since \(\int_X \varphi\,d\mu<\infty\) and \(M< \infty\text{,}\) it follows that
    🔗
    \begin{equation*} \int_X \varphi\,d\mu \leq \liminf_n \int_X f_n \,d\mu. \end{equation*}
    Now, taking a supremum over all simple \(\varphi\) with \(0 \leq \varphi \leq f\) gives the desired result.
    🔗
    🔗
    Fatou’s Lemma will prove to be a powerful tool. We give now our first application of Fatou’s Lemma, and our first theorem where a pointwise limit of functions implies a limit of integrals.
    🔗

    Theorem 8.1.16. Monotone Convergence Theorem.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \((f_n)_n\) be an increasing sequence of non-negative measurable functions on \(X\text{.}\) Define a function \(f\) on \(X\) by \(f(x) = \lim_n f_n(x).\) Then \(f\) is measurable and
    🔗
    \begin{equation*} \lim_n \int_X f_n\,d\mu = \int_X f\,d\mu. \end{equation*}
    🔗

    Proof.

    Note that \(\lim_n f_n(x)\) exists for all \(x\) since the sequence \((f_n)_n\) is increasing. Thus \(f\) is a well-defined function on \(X\text{.}\) Further \(f\) is measurable by Proposition Proposition 7.2.3 and \(f\) is non-negative since each \(f_n\) is non-negative.
    🔗
    By the monotonicity of integration, we have that \((\int_X f_n\,d\mu)_n\) is an increasing sequence, and hence convergent. Further, since \(f_n \leq f\) for each \(n\text{,}\)
    🔗
    \begin{equation*} \lim_n \int_X f_n\,d\mu \leq \int_X f\,d\mu. \end{equation*}
    By Fatou’s Lemma, Lemma Lemma 8.1.15,
    🔗
    \begin{equation*} \int_X f\,d\mu \leq \liminf_n \int_X f_n \,d\mu = \lim_n \int_Xf_n\,d\mu. \end{equation*}
    Thus
    🔗
    \begin{equation*} \lim_n \int_X f_n\,d\mu = \int_X f\,d\mu. \end{equation*}
    🔗
    It is reasonable to ask whether `increasing’ can be replaced by `decreasing’ in the Monotone Convergence Theorem. That is, if \((f_n)_n\) is a decreasing sequence of non-negative measurable functions converging pointwise to \(f\text{,}\) is it true that
    🔗
    \begin{equation*} \lim_n \int_X f_n\,d\mu = \int_X f\,d\mu? \end{equation*}
    The following example shows that we cannot expect the statement to hold for decreasing sequences.

    Example 8.1.17.

    Consider the sequence of measurable functions \((f_n)_n = (\chi_{[n,\infty)})_n\) on the real line. The sequence \((f_n)_n\) is decreasing, and converges pointwise to the function \(f = 0\text{.}\) However, for each \(n\text{,}\)
    🔗
    \begin{equation*} \int_\mathbb{R} f_n\,dm = m(([n,\infty)) = \infty, \end{equation*}
    and so
    🔗
    \begin{equation*} \infty = \lim_n \int_\mathbb{R} f_n\,dm \neq \int_\mathbb{R} f\,dm = 0. \end{equation*}

    Corollary 8.1.18.

    Let \((X,\mathcal{F},\mu)\) be a measure space, let \((f_n)_n\) be a sequence of non-negative measurable functions, and let \(f = \sum_{n=1}^\infty f_n.\) Then
    🔗
    \begin{equation*} \int_X f \, d\mu = \sum_{k=1}^\infty \int_X f_n\, d\mu. \end{equation*}
    🔗

    Remark 8.1.19.

    Let \(f\) be a non-negative measurable function on a measure space \((X,\mathcal{F},\mu)\text{.}\) By the definition of the integral, \(\int_X f\,d\mu\) can be approximated by integrals of simple functions below \(f\text{.}\) By the Simple Approximation Theorem, \(f\) can be (pointwise) approximated by simple functions below \(f\text{.}\) The Monotone Convergence Theorem allows us to put these two approximations together. Explicitly, by the Simple Approximation Theorem, there is an increasing sequence of simple functions \((\phi_n)_n\text{,}\) \(0\leq \phi_n \leq f\text{,}\) which converge pointwise to \(f\text{.}\) By the Monotone Convergence Theorem
    🔗
    \begin{equation*} \lim_n \int_X \phi_n\,d\mu = \int_X f\,d\mu. \end{equation*}

    We end this subsection by showing that integration is linear.
    🔗

    Theorem 8.1.20.

    Let \((X,\mathcal{F},\mu)\) be a measure space, let \(f\) and \(g\) be non-negative measurable functions, and let \(\alpha\text{,}\) \(\beta\) be non-negative real-numbers. Then
    🔗
    \begin{equation*} \int_X(\alpha f + \beta g)\,d\mu = \alpha \int_X f\,d\mu + \beta \int_X g\,d\mu. \end{equation*}
    🔗

    Proof.

    By the Simple Approximation Theorem there are increasing sequences of non-negative simple functions \((\phi_n)_n\) and \((\psi_n)_n\) such that \((\phi_n)_n\) converges to \(f\) pointwise and \((\psi_n)_n\) converges to \(g\) pointwise. Thus \((\alpha\phi_n + \beta\psi_n)_n\) is an increasing sequence of non-negative simple functions which converge pointwise to \(\alpha f +\beta g\text{.}\) Using the Monotone Covergence Theorem and Theorem Theorem 8.1.3 and the Monotone Convergence Theorem again we get
    🔗
    \begin{equation*} \begin{aligned} \int_X (\alpha f + \beta g)\,d\mu &= \lim_n \int_X(\alpha \phi_n + \beta \psi_n)\,d\mu \\ &= \alpha \lim_n\int_X \phi_n\,d\mu + \beta \lim_n \int_X \psi_n \,d\mu \\ &= \alpha \int_X f\,d\mu + \beta \int_X g\, d\mu. \end{aligned} \end{equation*}
