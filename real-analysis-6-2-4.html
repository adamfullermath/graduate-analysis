    Section 1.2 \(L^p\)-norms

    Denote by \(\mathcal{F}\) be the collection of all extended real-valued measurable functions on \(X\) which are finite almost everywhere. By Theorem II.5.1.8, \(\mathcal{F}\) is a vector space. Let \(\sim\) be the equivalence relation: \(f\sim g\) if and only if \(f = g\) almost everywhere, for \(f,g \in \mathcal{F}\text{.}\) By Exercise II.5.3.6, \(\sim\) is an equivalence relation on \(\mathcal{F}\text{.}\) Let \(\mathcal{F}/\sim\) be the equivalence classes of \(\sim\) in \(\mathcal{F}\text{.}\) For each \(f \in \mathcal{F}\) denote by \([f]\) the equivalence class of \(f\) under the equivalence relation \(\sim\text{.}\) The set \(\mathcal{F}/\sim\) inherits a vector space structure from \(\mathcal{F}\) by the operations
    🔗
    \begin{equation*} \begin{aligned} [f] + [g] &= [f+g],\ \text{and}\\ \alpha[f] &= [\alpha f] \end{aligned} \end{equation*}
    for \(f,g \in \mathcal{F}\) and \(\alpha \in \mathbb{R}\) (see Exercise 1.4.11).
    🔗

    Definition 1.2.1.

    Let \((X,\mathcal{M},\mu)\) be a measure space, let \(\mathcal{F}\) be all finite almost everywhere measurable functions on \(\mathcal{F}\text{,}\) and let \(\sim\) be the equivalence relation of equivalence almost everywhere on \(\mathcal{F}\text{.}\)
    🔗
    For \(1<p<\infty\text{,}\) define
    🔗
    \begin{equation*} L^p(X,\mu) = \{[f] \in \mathcal{F}/\sim \colon |f|^p \text{ integrable}\}. \end{equation*}
    We call \(L^p(X,\mu)\) the \(L^p\)-space of \((X,\mathcal{M},\mu)\).
    🔗
    🔗

    Remark 1.2.2.

    Note, that \(L^p(X,\mu)\) is well-defined. Indeed, if \(f,g \in \mathcal{F}\) and \(f = g\) a.e., then \(|f|^p =|g|^p\) a.e. and so \(|f|^p\) is integrable if and only if \(|g|^p\) is integrable.
    🔗
    🔗
    We will also define the \(L^p\)-space for \(p = \infty\text{.}\)
    🔗

    Definition 1.2.3.

    Let \((X,\mathcal{M},\mu)\) be a measure space, let \(\mathcal{F}\) be all finite almost everywhere measurable functions on \(\mathcal{F}\text{,}\) and let \(\sim\) be the equivalence relation of equivalence almost everywhere on \(\mathcal{F}\text{.}\) A function \(f \in \mathcal{F}\) is essentially bounded if there is a constant \(M \geq 0\) such that \(|f| \leq M\) almost everywhere. In this case, we call \(M\) an essential bound for \(f\text{.}\) Define \(L^\infty(X,\mu)\) as
    🔗
    \begin{equation*} L^\infty(X,\mu) = \{[f]\in \mathcal{F}/\sim \colon f \text{ essentially bounded}\}. \end{equation*}
    We call \(L^\infty(X,\mu)\) the \(L^\infty\)-space for \((X,\mathcal{M},\mu).\)
    🔗
    🔗

    Remark 1.2.4.

    Note, that \(L^\infty(X,\mu)\) is well-defined. Indeed, if \(f,g \in \mathcal{F}\) and \(f = g\) a.e., then \(f\) is essentially bounded if and only if \(g\) is essentially bounded.
    🔗
    🔗

    Remark 1.2.5.

    Let \(1\leq p \leq \infty\text{.}\) The set \(L^p(X,\mu)\) is defined as a subset of \(\mathcal{F}/\sim\text{.}\) Thus, the elements of \(L^p(X,\mu)\) are not functions, but equivalence classes of functions. We will, however, often abuse language and refer to the elements of \(L^p(X,\mu)\) as functions. We will often abuse notation, and write, for \(f \in \mathcal{F}\text{,}\) \(f \in L^p(X,\mu)\) to mean that \([f]\in L^p(X,\mu)\text{.}\)
    🔗
    🔗
    We will show that each \(L^p\)-space is a Banach space under an appropriate norm. To do this, we must first show that each \(L^p\)-space is a vector space.
    🔗

    Proposition 1.2.6.

    Let \((X,\mathcal{M},\mu)\) be a measure space, and let \(1\leq p \leq \infty\text{.}\) Then \(L^p(X,\mu)\) is a vector subspace of \(\mathcal{F}/\sim.\)
    🔗
    🔗

    Proof.

    First, assume that \(p<\infty\text{.}\) Take \(f \in L^p(X,\mu)\) and a scalar \(\alpha \in \mathbb{R}\text{.}\) Then
    🔗
    \begin{equation*} \int_X |\alpha f|^p \,d\mu = |\alpha|^p \int_X|f|^p\,d\mu<\infty. \end{equation*}
    Hence \(\alpha f \in L^p(X,\mu)\text{.}\)
    🔗
    Note that if \(a,b\in\mathbb{R}\text{,}\) then \(|a+b|^p\leq 2^p(|a|^p + |b|^p).\) Indeed,
    🔗
    \begin{equation*} |a+b| \leq |a| + |b| \leq 2\max\{|a|,|b|\}, \end{equation*}
    and hence
    🔗
    \begin{equation*} |a+b|^p \leq 2^p \left(\max\{|a|,|b|\}\right)^p \leq 2^p(|a|^p + |b|^p). \end{equation*}
    Thus, if \(f,g \in L^p(X,\mu)\text{,}\) then
    🔗
    \begin{equation*} \int_X |f+g|^p \,d\mu \leq 2^p \int_X |f|^2\,d\mu+2^p \int_X |g|^2\,d\mu<\infty. \end{equation*}
    Hence \(f+g \in L^p(X,\mu)\text{.}\)
    🔗
    That \(L^\infty(X,\mu)\) is a vector space is left as an exercise, Exercise 1.4.12.
    🔗
    🔗
    We now define a norm on \(L^p(X,\mu)\text{.}\) To show that these are indeed norms will take some work. Fix \(1\leq p < \infty\text{.}\)\ For \(f \in L^p(X,\mu)\) let
    🔗
    \begin{equation*} \|f\|_p = \left( \int_X |f|^p\,d\mu)\right)^{1/p}. \end{equation*}
    When \(p=\infty\) we define things differently. For \(f \in L^\infty(X,\mu),\) define
    🔗
    \begin{equation*} \|f\|_\infty = \inf \{M \colon M \text{ an essential bound for }f\}. \end{equation*}
    For \(1\leq p \leq \infty\) we call \(\|\cdot\|_p\) the \(p\)-norm on \(L^2(X,\mu)\text{.}\)
    🔗
    The introduction of the \(p\)-norms has given us some overlapping notation for \(\|\cdot\|_2\) and \(\|\cdot\|_\infty\text{.}\) We address this in the following remarks.
    🔗

    Remark 1.2.7.

    Let \(I \subseteq \mathbb{R}\) be a Lebesgue measurable set and let \(m\) be the Lebesgue measure on \(I\text{.}\) We have the \(\sup\)-norm \(\|\cdot\|_\infty\) on \(C_b(I)\text{,}\) and we have the \(\infty\)-norm \(\|\cdot\|_\infty\) on \(L^\infty(I,m)\text{.}\) Note that any continuous function on \(I\) is measurable with respect to Lebesgue measure \(m\text{,}\) by Corollary II.5.1.5. Thus, if \(f \in C_b(I)\) then \(f \in L^\infty(I,m)\text{,}\) and the \(\sup\)-norm and the \(\infty\)-norm of \(f\) coincide. Thus the \(\sup\)-norm is a restriction of an \(\infty\)-norm to continuous bounded functions.
    🔗
    🔗

    Remark 1.2.8.

    We also have two \(2\)-norms: \(\|\cdot\|_2\) on \(\mathbb{R}^n\) and \(\|\cdot\|_2\) on \(L^2(X,\mu)\text{.}\) Let \(A = \{1,2,3,\ldots,n\}\) and let \(c\) be the counting measure on the subsets of \(A\text{.}\) In this case, the finite a.e. measurable functions on \(A\) are all functions \(f \colon A \rightarrow \mathbb{R}\text{.}\) Further, two functions \(f\) and \(g\) on \(A\) are equal a.e. if and only if \(f =g\) everywhere. Letting \(\mathcal{F}\) be the collection of all finite a.e. measurable functions on \(A\) as above, we have that \(L^p(A,c) = \mathcal{F}\) for all \(1\leq p \leq \infty.\) Note that \(\mathcal{F}\) is isomorphic to \(\mathbb{R}^n\) via the isomorphism
    🔗
    \begin{equation*} \psi\colon f \mapsto (f(1),f(2),\ldots,f(n)). \end{equation*}
    Take any \(f \in L^2(A,c)=\mathcal{F}\text{.}\) It is not hard to see that
    🔗
    \begin{equation*} \|f\|_2 = \|\psi(f)\|_2. \end{equation*}
    Thus, the \(2\)-norm on \(\mathbb{R}^n\) is an example of a \(2\)-norm on a \(L^2\)-space.
    🔗
    🔗
    Have not yet shown that \(\|\cdot\|_p\) is a norm on \(L^p(X,\mu)\text{.}\) We will work towards this for the rest of this section. We begin with the simplest cases: \(p=1\) and \(p=\infty\text{.}\)
    🔗

    Proposition 1.2.9.

    Let \((X,\mathcal{M},\mu)\) be a measure space. Then \(\|\cdot\|_1\) is a norm on \(L^1(X,\mu)\) and \(\|\cdot\|_\infty\) is a norm on \(L^\infty(X,\mu)\text{.}\)
    🔗
    🔗

    Proof.

    It is clear that \(\|f\|_1\geq 0\) for all \(f \in L^1(X,\mu)\text{.}\) Suppose, \(f \in L^1(X,\mu)\) and \(\|f\|_1 = 0\text{.}\) Then
    🔗
    \begin{equation*} \int_X |f|\,d\mu =0. \end{equation*}
    Let \(X_+ = \{x \in X \colon f(x) \neq 0\}\) and \(X_0 = \{x \in X \colon f(x) = 0\}\text{.}\) Then
    🔗
    \begin{equation*} \begin{aligned} 0 = \int_X|f|\,d\mu = \int_{X_0}|f|\,d\mu + \int_{X_+}|f|\,d\mu = \int_{X_+}|f|\,d\mu. \end{aligned} \end{equation*}
    Hence, by Exercise II.6.4.5, \(\mu(X_+) = 0\text{.}\) That is, \(f = 0\) a.e., and so \(f = 0\) in \(L^1(X,\mu)\text{.}\) Therefore \(\|f\|_1 = 0\) if and only if \(f = 0\text{.}\)
    🔗
    If \(f \in L^1(X,\mu)\) and \(\alpha \in \mathbb{R}\text{,}\) then
    🔗
    \begin{equation*} \|\alpha f\|_1 = \int_X |\alpha f|\,d\mu = |\alpha| \int_X |f|\,d\mu = |\alpha| \|f\|_1. \end{equation*}
    Finally, suppose \(f,g\in L^1(X,\mu)\text{.}\)Then
    🔗
    \begin{equation*} \begin{aligned} \|f+g\|_1 &= \int_X|f+g|\,d\mu \\ &\leq \int_X(|f| + |g|)\,d\mu = \|f\|_1+ \|g\|_1. \end{aligned} \end{equation*}
    Thus \(\|\cdot\|_1\) satisfies the triangle inequality. We have, therefore, shown that \(\|\cdot\|_1\) is a norm on \(L^1(X,\mu)\text{.}\)
    🔗
    We now consider \(\|\cdot\|_\infty\) on \(L^\infty(X,\mu)\text{.}\) Again, \(\|f\|_\infty \geq 0\) for all \(f \in L^\infty(X,\mu)\text{.}\) If \(\|f\|_\infty=0\) then \(|f(x)| \leq 0\) for almost all \(x \in X\text{.}\) Thus \(\|f\|_\infty =0\) if and only if \(f = 0\) in \(L^\infty(X,\mu)\text{.}\)
    🔗
    It is straightforward to show that \(\|\alpha f\|_\infty = |\alpha| \|f\|_\infty\) for all \(\alpha \in \mathbb{R}\) and \(f \in L^\infty(X,\mu)\text{.}\) It remains to show that \(\|\cdot\|_\infty\) satisfies the triangle inequality. Take \(f,g \in L^\infty(X,\mu)\text{,}\) and let \(M\) be an essential bound for \(f\) and \(N\) be an essential bound for \(g\text{.}\) Thus, there exist measurable sets \(E_f\) and \(E_g\) in \(X\) with \(\mu(E_f) = \mu(E_g) = 0\text{,}\) \(|f(x)|\leq M\) for all \(x\in X\backslash E_f\text{,}\) and \(|g(x)|\leq N\) for all \(x\in X\backslash E_g\text{.}\) Thus, if \(x \in X\backslash(E_f \cup E_g)\text{,}\) then
    🔗
    \begin{equation*} |f(x) + g(x)| \leq |f(x)| + |g(x)| \leq M+N. \end{equation*}
    As \(\mu(E_f \cup E_g) =0\text{,}\) it follows that \(M+N\) is an essential bound for \(f+g\text{.}\) Hence
    🔗
    \begin{equation*} \begin{aligned} &\inf \{M \colon M \text{ an essential bound for }f\} + \inf \{N \colon N \text{ an essential bound for }g\}\\ &= \inf \{M+N \colon M \text{ an ess. bound for }f,\ N \text{ an ess. bound for }g\}\\ &\geq \inf \{K \colon K \text{ an essential bound for }f+g\}.\\ \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation*} \|f\|_\infty + \|g\|_\infty \geq \|f+g\|_\infty. \end{equation*}
    Therefore \(\|\cdot\|_\infty\) is a norm on \(L^\infty(X,\mu)\text{.}\)
    🔗
    🔗
    In order to show that \(\|\cdot\|_p\) is a norm on \(L^p(X,\mu)\) for \(1<p<\infty\text{,}\) we first need to introduce some new definitions and prove some inequalities.
    🔗

    Definition 1.2.10.

    For \(1<p<\infty\text{,}\) we define \(q\) to be conjugate of \(p\) when
    🔗
    \begin{equation*} q = \frac{p}{p-1}. \end{equation*}
    We define the conjugate of \(1\) to be \(\infty\text{,}\) and the conjugate of \(\infty\) to be \(1\text{.}\)
    🔗
    🔗

    Remark 1.2.11.

    Note that if \(1< p <\infty\) and \(q\) is conjugate to \(p\text{,}\) then \(q\) is the unique number such that
    🔗
    \begin{equation*} \frac{1}{p} + \frac{1}{q} =1. \end{equation*}
    In particular, \(2\) is conjugate to \(2\text{,}\) and if \(p\neq 2\) then \(q \neq p\text{.}\)
    🔗
    🔗

    Lemma 1.2.12. Young’s Inequality.

    Let \(1< p < \infty\) and \(q\) be conjugate to \(p\text{.}\) If \(a,b>0\) then
    🔗
    \begin{equation*} ab \leq \frac{a^p}{p}+\frac{b^q}{q}. \end{equation*}
    🔗

    Proof.

    The proof we present here is based on the convexity of the function \(f(x) = e^x\text{.}\) By convexity we mean that, for \(u,v \in \mathbb{R}\) and \(\lambda \in [0,1]\)
    🔗
    \begin{equation} f((1-\lambda)u + \lambda v) \leq (1-\lambda)f(u) + \lambda f(v).\tag{1.6} \end{equation}
    That is, the straight line segment from \((u,f(u))\) to \((v,f(v))\) is above the curve \((t,f(t))\) with \(u \leq t \leq v\text{.}\) Intuitively, this can be understood since \(f''(x) = e^x > 0\) for all \(x\text{.}\) A formal proof of the inequality (1.6) is eked out in Exercise 1.4.13.
    🔗
    🔗
    Let \(u = \ln(a^p)\text{,}\) \(v = \ln(b^q)\text{,}\) and \(\lambda = \frac{1}{p}.\) Note that, since \(p>1\text{,}\) \(0<\lambda <1\text{.}\) Further, since \(q\) is conjugate to \(p\text{,}\) \(1-\lambda = \frac{1}{q}\text{.}\) Thus, by (1.6),
    🔗
    \begin{equation*} \begin{aligned} \exp\left(\frac{1}{p}\ln(a^p) + \frac{1}{q}\ln(b^q)\right) &\leq \frac{1}{p}\exp(\ln(a^p)) + \frac{1}{q}\exp(\ln(b^q))\\ &= \frac{a^p}{p} + \frac{b^q}{q}. \end{aligned} \end{equation*}
    Thus, noting that \(\ln(a) = \frac{1}{p}\ln(a^p)\) and \(\ln(b) = \frac{1}{q}\ln(b^q)\text{,}\)
    🔗
    \begin{equation*} \begin{aligned} ab &= e^{\ln(ab)} = e^{\ln(a) + \ln(b)} \\ &= \exp\left(\frac{1}{p}\ln(a^p) + \frac{1}{q}\ln(b^q)\right) \\ &\leq \frac{a^p}{p} + \frac{b^q}{q}, \end{aligned} \end{equation*}
    which proves the desired inequality.
    🔗
    🔗

    Theorem 1.2.14. Hölder’s Inequality.

    Let \((X,\mathcal{M},\nu)\) be a measure space, let \(p\) be any value \(1< p < \infty\) and let \(q\) be conjugate to \(p\text{.}\) If \(f,g\) are measurable functions on \(X\) then
    🔗
    \begin{equation*} \int_X |fg|\,d\mu \leq \left(\int_X |f|^p\,d\mu\right)^{1/p} \left(\int_X |g|^q\,d\mu\right)^{1/q}. \end{equation*}
    In particular, if \(f \in L^q(X,\mu)\) and \(g \in L^q(X,\mu)\) then \(fg \in L^1(X,\mu)\) and
    🔗
    \begin{equation*} \|fg\|_1 \leq \|f\|_p \|g\|_q. \end{equation*}
    🔗

    Proof.

    The inequality holds if either \(f\) or \(g\) are zero almost everywhere in \(X\text{.}\) We can thus assume that neither \(f\) or \(g\) is \(0\) a.e. If either \(|f|^p\) or \(|g|^q\) is not intergable, i.e. if
    🔗
    \begin{equation*} \int_X |f|^p\,d\mu =\infty \text{ or } \int_X |g|^q\,d\mu =\infty, \end{equation*}
    then the desired inequality holds trivially.
    🔗
    We can thus assume that \(f \in L^{p}(X,\mu)\) and \(g \in L^q(X,\mu)\) are non-zero. For any non-zero \(h \in L^p(X,\mu)\) and non-zero \(k \in L^q(X,\mu)\) we have
    🔗
    \begin{equation*} |h(x)||k(x)| \leq \frac{|h(x)|^p}{p} + \frac{|k(x)|^q}{q} \end{equation*}
    for almost all \(x \in X\text{,}\) by Young’s Inequality, Lemma 1.2.12. Applying this inequality to \(h = \frac{1}{\|f\|_p}f\) and \(k = \frac{1}{\|g\|_q}g\) we get
    🔗
    \begin{equation*} \begin{aligned} \frac{1}{\|f\|_p\|g\|_q}\int_X |fg|\,d\mu &= \int_X \left(\frac{|f|}{\|f\|_p} \frac{|g|}{\|g\|_q} \right)\,d\mu \\ &\leq \int_X\left( \frac{1}{p}\frac{|f|^p}{\|f\|^p_p} + \frac{1}{q}\frac{|g|}{\|g\|^q_q}\right)\,d\mu \\ &= \frac{1}{p}+\frac{1}{q} = 1. \end{aligned} \end{equation*}
    Thus \(\int_X|fg|,d\mu \leq \|f\|_p \|g\|_q.\)
    🔗
    🔗
    A special case of Hölder’s inequality is when \(p = q =2.\) When \(p = q = 2\text{,}\) we call Hölder’s inequality the Cauchy-Schwarz inequality. For posterity, we state this special case here. A special case of the Cauchy-Schwarz inequality is in Exercise 1.4.1. An alternate proof of the Cauchy-Schwarz inequality, which does not rely on Hölders inequality, is described in Exercise 1.4.14.
    🔗

    Theorem 1.2.15. Cauchy-Schwarz Inequality.

    Let \((X,\mathcal{M},\mu)\) be a measure space. If \(f,g \in L^2(X,\mu)\text{,}\) then \(fg\) is integrable and
    🔗
    \begin{equation*} \left| \int_X fg \,d\mu \right| \leq \|f\|_2 \|g\|_2. \end{equation*}
    🔗
    The following corollary to Hölder’s inequality will be used frequently.
    🔗

    Corollary 1.2.16.

    Let \(1< p < \infty\) and let \(q\) be conjugate to \(p\text{.}\) If \(f \in L^p(X,\mu)\text{,}\) then there exists \(g \in L^q(X,\mu)\) with \(\|g\|_q=1\) such that
    🔗
    \begin{equation*} \int_X fg\,d\mu = \|f\|_p. \end{equation*}
    🔗

    Proof.

    If \(f = 0\text{,}\) then the result holds for any \(g \in L^q(X,\mu)\) with \(\|g\|_q = 1\text{.}\) Let \(f \in L^p(X,\mu)\) be non-zero. Define the sign function for \(f\) by
    🔗
    \begin{equation*} \operatorname{sgn}(x) = \begin{cases} 1 & \text{if }x>0, \\ 0 & \text{if }x=0,\\ -1 & \text{if }x<0,\end{cases} \end{equation*}
    and let \(\alpha = \|f\|_p^{1-p}.\) Let \(g(x) = \alpha\ \operatorname{sgn}(f(x))|f|^{p-1}.\) Since \(f\) and \(\operatorname{sgn}(f)\) are measurable functions, \(g\) is measurable. Noting that \(q(p-1) = p\text{,}\) we have
    🔗
    \begin{equation*} \begin{aligned} \int_X|g|^q\,d\mu &= \alpha^q \int_X |f|^{q(p-1)}\,d\mu\\ &= \alpha^q \int_X |f|^p\,d\mu\\ &= \|f\|_p^{q(1-p)} \|f\|_p^p =1. \end{aligned} \end{equation*}
    Thus, \(g \in L^q(X,\mu)\) and \(\|g\|_q =1\text{.}\) Further,
    🔗
    \begin{equation*} \begin{aligned} \int_X fg \,d\mu &= \alpha \int_X |f||f|^{p-1}\,d\mu \\ & =\alpha \|f\|_p^p = \|f\|_p, \end{aligned} \end{equation*}
    completing the proof.
    🔗
    🔗
    We come now to our final named inequality of this section.
    🔗

    Theorem 1.2.17. Minkowski’s Inequality.

    Let \((X,\mathcal{M},\mu)\) be a measure space and take \(p\) with \(1<p<\infty\text{.}\) If \(f,g \in L^p(X,\mu)\text{,}\) then
    🔗
    \begin{equation*} \|f + g\|_q \leq \|f\|_q + \|g\|_q. \end{equation*}
    🔗

    Proof.

    Let \(q\) be conjugate to \(p\text{.}\) Take \(f,g \in L^p(X,\mu)\text{.}\) By Corollary 1.2.16, there exists \(h \in L^q(X,\mu)\) with \(\|h\|_q = 1\) such that
    🔗
    \begin{equation*} \|f+g\|_p = \int_X(f+g)h\,d\mu. \end{equation*}
    Thus
    🔗
    \begin{equation*} \begin{aligned} \|f+g\|_p &= \int_X(f+g)h\,d\mu = \left|\int_X(f+g)h\,d\mu\right|\\ &\leq \int_X|fh|\,d\mu + \int_X|gh|\,d\mu \\ &\leq \|f\|_p + \|g\|_p, \end{aligned} \end{equation*}
    with the last inequality coming from two applications of Hölder’s inequality, and the fact that \(\|h\|_q=1\text{.}\)
    🔗
    🔗
    We now have all we need to show that \(\|\cdot\|_p\) is a norm for \(1<p<\infty\text{.}\)
    🔗

    Theorem 1.2.18.

    Let \((X,\mathcal{M},\mu)\) be a measure space. For each \(p\text{,}\) \(1\leq p \leq \infty\text{,}\) \(\|\cdot\|_p\) is a norm on \(L^p(X,\mu)\text{.}\)
    🔗
    🔗

    Proof.

    The cases when \(p=1\) and \(p= \infty\) are covered in Proposition 1.2.9. For \(1<p<\infty\text{,}\) Minkowski’s inequality, Theorem 1.2.17, shows that \(\|\cdot\|_p\) satisfies the triangle inequality. That \(\|\cdot\|_p\) satisfies the other properties of a norm is straightforward.
    🔗
    🔗

    Remark 1.2.19.

    Let \((X,\mathcal{M},\mu)\) be a measure space. If \(f,g \in L^2(X,\mu)\text{,}\) then the Cauchy-Schwarz inequality (or Hölder’s inequality) says that \(fg\) is integrable. We can thus define a map
    🔗
    \begin{equation*} L^2(X,\mu) \times L^2(X,\mu) \rightarrow \mathbb{R} \end{equation*}
    by
    🔗
    \begin{equation*} \langle f,g\rangle = \int_X fg\,d\mu. \end{equation*}
    This map is bilinear, by the linearity properties of integration. Further
    🔗
    \begin{equation*} \langle f,f\rangle = \int_X f^2\,d\mu = \int_X |f|^2\,d\mu =\|f\|_2^2. \end{equation*}
    Hence \(\langle \cdot,\cdot\rangle \) is an inner-product on \(L^2(X,\mu)\) and the \(2\)-norm is the induced inner-product norm on \(L^2(X,\mu)\text{.}\)
    🔗
    🔗
    Given a measure space \((X,\mathcal{M},\mu)\) we now have infinitely many vector space \(L^p(X,\mu)\text{.}\) What is the relationship between these vector spaces? When do we have \(L^q(X,\mu) \subseteq L^p(X,\mu)\text{?}\) When \(\mu(X)<\infty\) we have the following:
    🔗

    Theorem 1.2.20.

    Let \((X,\mathcal{M},\mu)\) be a finite measure space. If \(1\leq p < q \leq \infty\text{,}\) then \(L^q(X,\mu) \subseteq L^p(X,\mu)\text{.}\)
    🔗
    🔗

    Proof.

    Take \(f \in L^q(X,\mu)\text{.}\) If \(q = \infty\text{,}\) then
    🔗
    \begin{equation*} \int|f|^p\,d\mu \leq \|f\|_\infty^p\mu(X)<\infty, \end{equation*}
    and hence \(f \in L^p(X,\mu)\text{.}\)
    🔗
    Suppose now that \(1<q<\infty\text{.}\) Note that
    🔗
    \begin{equation*} \frac{p}{q} + \frac{q-p}{q} = 1, \end{equation*}
    and hence \(\frac{q}{q-p}\) is the conjugate of \(\frac{q}{p}\text{.}\) Applying Hölder’s inequality with the conjugates \(\frac{q}{q-p}\) and \(\frac{q}{p}\) we have
    🔗
    \begin{equation*} \begin{aligned} \int_X|f|^p\,d\mu &= \int_X|f|^p\cdot \chi_X\,d\mu \\ &\leq \left(\int_X (|f|^p)^{q/p}\,d\mu \right)^{p/q}\left(\int_X \chi_X^{q/(q-p)}\,d\mu\right)^{(q-p)/q}\\ &= \left(\int_X |f|^q\,d\mu \right)^{p/q} (\mu(X))^{(q-p)/q}\\ &= \|f\|_q^p (\mu(X))^{(q-p)/q} <\infty. \end{aligned} \end{equation*}
    Hence, \(f \in L^p(X,\mu)\text{.}\)
    🔗
    🔗
    The following example shows that the containment in Theorem 1.2.20 can be strict.
    🔗

    Example 1.2.21.

    Let \(m\) be the Lebesgue measure on the interval \((0,1) \subseteq \mathbb{R}\text{.}\) By Theorem 1.2.20, if \(1\leq p < q \leq \infty\text{,}\) then \(L^q((0,1),m)) \subseteq L^p((0,1),m)\text{.}\) Let \(f(x) = \frac{1}{\sqrt{x}}.\) Then \(f \in L^1((0,1),m)\) with
    🔗
    \begin{equation*} \|f\|_1 = \int_0^1 \frac{1}{\sqrt{x}}\,dm =2. \end{equation*}
    However \(f \notin L^2((0,1),m)\) since
    🔗
    \begin{equation*} \int_{(0,1)}\frac{1}{x}\,dm =\lim_{a\rightarrow 0^+} (\ln(1) - \ln(a)) = \lim_{a\rightarrow 0^+} (1 - \ln(a)) = \infty. \end{equation*}
    More generally, if \(1\leq p < q< \infty\) and
    🔗
    \begin{equation*} f(x) = x^{-(1/q)} \end{equation*}
    then \(f \in L^p((0,1),\mu)\) but \(f \notin L^q((0,1),m).\) Also, note that \(f\) is not bounded, and so \(f \notin L^\infty((0,1),m).\) Thus, if \(1\leq p<q<\infty\) then
    🔗
    \begin{equation*} L^q((0,1),m) \subsetneq L^p((0,1),m). \end{equation*}
    🔗
    When \(\mu(X)=\infty\) the conclusion of Theorem 1.2.20 does not necessarily hold. In fact, we may not have containments in any direction.
    🔗

    Example 1.2.22.

    Let \(m\) be the Lebesgue measure on \((0,\infty)\text{.}\) Suppose \(1\leq p < q \leq \infty\text{.}\) Let \(f(x) = x^{-1/p}\) for \(x\geq 1\text{,}\) and \(f(x) = 0\) for \(x \in (0,1)\text{.}\) Since \(f\) is bounded, we have that \(f \in L^\infty([1,\infty),m)\text{.}\) If \(q < \infty\text{,}\) then, since \(q/p>1\text{,}\) \(f \in L^q((0,\infty),m)\text{.}\) However, \(f \notin L^p((0,\infty),m)\) since \(1/x\) is not integrable on \([1,\infty)\text{.}\) Thus
    🔗
    \begin{equation*} L^q((0,\infty),m) \not\subseteq L^p((0,\infty),m). \end{equation*}
    Using the examples from Example 1.2.21 we can also show that
    🔗
    \begin{equation*} L^p((0,\infty),m) \not\subseteq L^q((0,\infty),m). \end{equation*}
    Explicitly, for \(q< \infty\text{,}\) if \(f(x) = x^{-1/q}\) for \(x \in (0,1)\) and \(f(x) = 0\) for \(x >0\text{,}\) then \(f \in L^p((0,\infty),m)\) but \(f \notin L^q((0,\infty),m).\)
    🔗
    🔗

    Subsection \(\ell^p\)-spaces and \(\mathbb{R}^n\)

    We pronounce \(\ell^p(A)\) as `little \(lp\)’. When \(A = \mathbb{N}\text{,}\) we simply write \(\ell^p\) for \(\ell^p(\mathbb{N})\text{.}\) The space \(\ell^p\) thus consists of all sequences of real-numbers \((a_n)_n\) such that
    🔗
    \begin{equation*} \sum_{n=1}^\infty |a_n|^p < \infty, \end{equation*}
    when \(1\leq p < \infty\text{;}\) and \(\ell^\infty\) consists of all bounded sequences.
    🔗
    The following result shows that the conclusion of Theorem 1.2.20 can hold on non-finite measure spaces, when the measure is the counting measure. This should be compared with Example 1.2.22
    🔗

    Theorem 1.2.23.

    Let \(1\leq p<q\leq \infty\text{,}\) then \(\ell^q(A) \subseteq \ell^p(A)\) and \(\|f\|_q \leq \|f\|_p\) for all \(f \in \ell^p(A).\)
    🔗
    🔗

    Proof.

    Take any non-negative function \(g\) on \(A\text{.}\) If \(g\) is not bounded, then there is an \(M> 0\) and a sequence \((a_n)_n\) in \(A\) such that \(g(a_n)\geq M\) for all \(n\geq 0\text{.}\) Thus
    🔗
    \begin{equation*} \int_A g\,dc \geq \sum_{n=1}^\infty g(a_n) = \infty. \end{equation*}
    Hence \(g\) is not integrable. Therefore, if \(f \in \ell^p(A)\text{,}\) then \(f\) is bounded. Take \(f\in\ell^p(A)\) and \(\varepsilon >0\text{.}\) There exists an \(a \in A\) such that
    🔗
    \begin{equation*} \|f\|_\infty < f(a) + \varepsilon. \end{equation*}
    Thus
    🔗
    \begin{equation*} \int_A |f|^p\, dc \geq |f(a)|^p > (\|f\|_\infty - \varepsilon)^p. \end{equation*}
    As this holds for all \(\varepsilon>0\text{,}\) it follows that \(\|f\|_p \geq \|f\|_\infty\text{.}\)
    🔗
    Suppose now that \(1\leq p <q < \infty\text{.}\) Then
    🔗
    \begin{equation*} \begin{aligned} \left(\int_A |f|^q\,dc\right)^{1/q} &\leq \left(\int_A |f|^p|f|^{q-p}\,dc\right)^{1/q}\\ &\leq \|f\|_\infty^{(q-p)/p}\left(\int_A |f|^p\,dc\right)^{1/q}\\ &= \|f\|_\infty^{(q-p)/q} \|f\|_p^{p/q}\\ &\leq \|f\|_p. \end{aligned} \end{equation*}
    Thus \(f \in \ell^q(A)\) and \(\|f\|_q\leq \|f\|_p.\)
    🔗
    🔗
    The following example shows that the inclusion of \(\ell^p(A) \subseteq \ell^q(A)\) from Theorem 1.2.23 can be strict.
    🔗

    Example 1.2.24.

    Take \(1 \leq p<q <\infty\) and let \(a_n = \left(\frac{1}{n}\right)^{1/p}.\) Then \((a_n)_n \in \ell^q\) but \((a_n)_n\notin\ell^p.\)
    🔗
    Any non-zero constant sequence is in \(\ell^\infty\) but not \(\ell^p\) for \(1\leq p <\infty\text{.}\) E.g., if \(a_n = 1\) for all \(n\text{,}\) then \((a_n)_n \in \ell^{\infty}\) but \((a_n)_n \notin \ell^p\) for any \(p<\infty\text{.}\)
    🔗
    🔗
    We turn our attention now to finite dimensional vector spaces. Let \(A = \{1,\ldots,n\}\text{.}\) In Remark 1.2.8 we observed that, for all \(1\leq p \leq \infty\text{,}\) \(\ell^p(A)\) is the real-valued functions on \(A\text{.}\) Further, we can identify \(\ell^p(A)\) with \(\mathbb{R}^n\) via the isomorphism
    🔗
    \begin{equation*} f \mapsto (f(1),f(2),\ldots,f(n)). \end{equation*}
    Under this identification, we have a norms \(\|\cdot\|_p\) on \(\mathbb{R}^n\) for \(1\leq p<\infty\text{.}\) These norms are given by
    🔗
    \begin{equation*} \|(v_1,\ldots,v_n)\|_p = \left( \sum_{i=1}^n |v_i|^p\right)^{1/p} \end{equation*}
    for \(1\leq p < \infty\text{,}\) and
    🔗
    \begin{equation*} \|(v_1,\ldots,v_n)\|_\infty = \max_{1\leq i \leq n}|v_i|. \end{equation*}
    When \(n>1\) these norms are all different, as illustrated in Figure~\ref{fig: p-norm}. However, the norms are all equivalent.
    🔗

    Theorem 1.2.26.

    The norms \(\{\|\cdot\|_p\}_{1\leq p\leq \infty}\) on \(\mathbb{R}^n\) are all equivalent.
    🔗
    🔗

    Proof.

    Take any \(v=(v_1,\ldots,v_n) \in \mathbb{R}^n\text{.}\) Then
    🔗
    \begin{equation*} \|v\|_1 = \sum_{i=1}^n|v_i| \leq n \max_{1\leq i \leq n}\{|v_i|\}= n\|v\|_\infty, \end{equation*}
    and
    🔗
    \begin{equation*} \|v\|_\infty \leq \sum_{i=1}^n|v_i| = \|v\|_1. \end{equation*}
    Thus, the norms \(1\)-norm and the \(\infty\)-norm are equivalent.
    🔗
    Let \(i_0\) be such that \(\max\{|v_i|\} = |v_{i_0}|\text{.}\) Then
    🔗
    \begin{equation*} \|v\|_\infty^p = |v_{i_0}|^p \leq \sum_{i=1}^n |v_i|^p. \end{equation*}
    Thus
    🔗
    \begin{equation*} \|v\|_\infty \leq \|v\|_p. \end{equation*}
    By Theorem 1.2.23, \(\|v\|_p \leq \|v\|_1\) for all \(1\leq p \leq \infty\text{.}\) Thus, for \(1<p<\infty\) we have
    🔗
    \begin{equation*} \|v\|_\infty \leq \|v\|_p \leq \|v\|_1 \leq n\|v\|_\infty. \end{equation*}
    Hence the \(p\)-norm is equivalent to the \(\infty\)-norm for all \(1\leq p \leq \infty\text{.}\) Therefore, all the \(p\)-norms are equivalent on \(\mathbb{R}^n\text{.}\)
