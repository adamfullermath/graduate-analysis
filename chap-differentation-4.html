    Section 9.2 Absolutely continuous functions

    Let \([a,b]\) be closed interval in \(\mathbb{R}\) and let \(m\) be the Lebesgue measure on \([a,b]\text{.}\) In this section we want to compare Borel measures on \([a,b]\) with the Lebesgue measure. If \(\nu\) is a Borel measure on \([a,b]\text{,}\) when is \(\nu \perp m\text{?}\) When is \(\nu \ll m\text{?}\) What can we learn from the Radon-Nikodym Theorem and the Lebesgue decomposition theorem?
    🔗
    We first give a quantitative description of absolute continuity.
    🔗

    Proposition 9.2.1.

    Let \((X,\mathcal{F},\mu)\) be a measure space. Suppose \(\nu\) is a finite signed-measure on \((X,\mathcal{F})\text{.}\) Then \(\nu \ll \mu\) if and only if for all \(\varepsilon >0\) there exists a \(\delta >0\) such that given \(E \in \mathcal{F}\text{,}\) if \(\mu(E) < \delta\) then \(\nu(E) < \varepsilon.\)
    🔗
    🔗

    Proof.

    First, assume the \(\varepsilon - \delta\) criterion does not hold. That is, we will assume that there is an \(\varepsilon > 0\) such that for all \(\delta > 0\) there is a set \(F_\delta\) with \(\mu(F_\delta)<\delta\) and \(\nu(F_\delta) \geq \varepsilon.\) There is thus a sequence of sets \(\{E_n\}_n^\infty\) such that
    🔗
    \begin{equation*} \mu(E_n) < \frac{1}{2^n} \text{ and }\nu(E_n)\geq \varepsilon_0. \end{equation*}
    For each \(n\text{,}\) let \(A_n = \bigcup_{k=n}^\infty E_k\text{.}\) Then \(\{A_n\}_{n=1}^\infty\) is a descending sequence of sets with
    🔗
    \begin{equation*} \mu(A_n) < \sum_{k=1}^n \frac{1}{2^k} = \frac{1}{2^{n-1}} \text{ and } \nu(A_n)\geq \varepsilon_0 \end{equation*}
    Let \(A = \bigcap_{n=1}^\infty A_n\text{.}\) Since \(A_n \subseteq A\) for all \(n\text{,}\)
    🔗
    \begin{equation*} \mu(A) \leq \mu(A_n) < \frac{1}{2^{n-1}} \end{equation*}
    for all \(n\text{.}\) Hence \(\mu(A) = 0\text{.}\) By the continuity of measure (since \(\nu\) is a finite measure)
    🔗
    \begin{equation*} \nu(A) = \lim_n\nu(A_n) \geq \varepsilon_0>0. \end{equation*}
    Hence \(\nu\) is not absolutely continuous with respect to \(\mu\text{.}\)
    🔗
    Conversely, suppose that \(\varepsilon -\delta\) criterion holds. Suppose \(\mu(A) = 0\text{.}\) Then, for all \(\varepsilon > 0\text{,}\) \(\nu(A)< \varepsilon\text{.}\) Hence \(\nu(A) = 0\text{.}\) Thus \(\nu\ll\mu\text{.}\)
     1 
    Note that we did not use the \(\nu\) was finite for this direction of the proof. That \(\nu\) was finite was only used in the first paragraph to apply the continuity of measure for intersections.
    🔗
    🔗

    Definition 9.2.2.

    Let \([a,b] \subseteq \mathbb{R}\) be a closed interval, let \(m\) be the Lebesgue measure on the Borel sets in \([a,b]\text{,}\) and let \(\nu\) be a Borel signed-measure on \([a,b]\text{.}\) We say that \(\nu\) is absolutely continuous if \(|\nu| \ll m\text{.}\) We say the \(\nu\) is singular if \(\nu_+ \perp m\) and \(\nu_- \perp m\text{,}\) where \(\nu = \nu_+ - \nu_-\) is the Jordan decomposition of \(\nu\text{.}\)
    🔗
    🔗
    We know that any Borel measure on \([a,b]\) determines a right-continuous function \(f\) of bounded variation on \([a,b]\text{;}\) and any right-continuous function \(f\) on \([a,b]\) determines a Borel measure, Theorem Theorem 6.2.14. Example Exercise 5.4.10 shows that we can get singular measures this way; Example Example 9.1.5 shows that we can absolutely continuous measures this way. The following definition is clearly leading to what functions give what types of measures.
    🔗

    Definition 9.2.3.

    Let \([a,b]\) be a closed interval and let \(f\colon [a,b] \rightarrow \mathbb{R}\) be a function. The function \(f\) is absolutely continuous if for all \(\varepsilon >0\) there is a \(\delta > 0\) such that if \(\{(a_i,b_i)\}_{i=1}^n\) are disjoint open intervals with
    🔗
    \begin{equation*} \sum_{i=1}^n(b_i - a_i) < \delta \end{equation*}
    then
    🔗
    \begin{equation*} \sum_{i=1}^n|f(b_i) - f(a_i)| < \varepsilon. \end{equation*}
    🔗
    It should be clear that an absolutely continuous function is continuous. The following example shows that all Lipschitz functions are absolutely continuous.
    🔗

    Example 9.2.4.

    Let \(f\) be a Lipschitz function on \([a,b]\). That is, assume there is a constant \(K\) such that \(|f(x) - f(y)|\leq K|x-y|\) for all \(x,y\in[a,b]\text{.}\) Take any \(\varepsilon>0\) and let \(\delta = \varepsilon/K\text{.}\) Let \(\{(a_i,b_i)\}_{i=1}^n\) be a collection of disjoint open intervals such that \(\sum_i(b_i-a_i)<\delta\text{.}\) Then
    🔗
    \begin{equation*} \begin{aligned} \sum_{i=1}^n|f(b_i) - f(a_i)| & \leq\sum_{i=1}^nK(b_i - a_i) \\ &< \varepsilon. \end{aligned} \end{equation*}
    Thus, \(f\) is absolutely continuous.
    🔗
    Assume now that \(f =0\text{.}\) Let \(\nu\) be the Borel signed-measure on \([a,b]\) induced by \(f\text{,}\) from Theorem Theorem 6.2.14. That is, if \(f = f_1 - f_2\) is the Jordan decomposition of \(f\text{,}\) then \(\nu = \nu_+ - \nu_-\) where \(\nu_+\) is the Carathéodory extension of the premeasure \(\nu_+((c,d]) = f_1(d) - f_1(c)\) and \(\nu_-\) is the Carathéodory extension of the premeasure \(\nu_-((c,d]) = f_2(d) - f_2(c)\text{,}\) for \((c,d] \subseteq [a,b]\text{.}\) Following a similar argument as in Example Example 9.1.5, we can show that \(\nu\) is absolutely continuous.
    🔗
    🔗
    The following theorem has a similar proof to the analogous statements for continuous functions and is left as an exercise.
    🔗

    Theorem 9.2.5.

    Let \(f\) and \(g\) be absolutely continuous functions on a closed interval \([a,b]\text{,}\) then
    🔗
    1. \(\alpha f + \beta g\) is absolutely continuous for all \(\alpha, \beta \in \mathbb{R}\text{;}\)
      🔗
      🔗
    2. \(fg\) is absolutely continuous;
      🔗
      🔗
    3. if \(g(x) \neq 0\) for all \(x \in [a,b]\text{,}\) \(f/g\) is absolutely continuous.
      🔗
      🔗
    🔗
    If we want to talk about Borel signed-measures constructed from absolutely continuous functions, we must first show that absolutely continuous functions have bounded variation.
    🔗

    Proposition 9.2.6.

    Let \(f\) be an absolutely continuous function on a closed interval \([a,b]\text{,}\) then \(f\) is of bounded variation.
    🔗
    Further, if \(f = f_1 - f_2\) is the Jordan decomposition of \(f\text{,}\) then \(f_1\) and \(f_2\) are absolutely continuous.
    🔗
    🔗

    Proof.

    Let \(\delta > 0\) satisfy
    🔗
    \begin{equation*} \sum_{k=1}^n |f(b_k) - f(a_k)| < 1 \end{equation*}
    whenever \(\{(a_k,b_k)\}_{k=1}^n\) are disjoint open intervals in \([a,b]\) with
    🔗
    \begin{equation*} \sum_{k=1}^n(b_k-a_k)<\delta. \end{equation*}
    Let \(K\) be an integer greater than \((b-a)/\delta\text{.}\) Then, for any partition \(P\) of \([a,b]\)
    🔗
    \begin{equation*} V(f,P) \leq K. \end{equation*}
    Thus, \(f\) has bounded variation, with total variation \(TV(f) \leq K\text{.}\)
    🔗
    Let \(f = f_1 - f_2\) be the Jordan decomposition of \(f\text{.}\) Recall
    🔗
    \begin{equation*} f_1 = \frac{1}{2}(T_f + f) \text{ and }f_2 = \frac{1}{2}(T_f - f), \end{equation*}
    where \(T_f\) is the function of total variation for \(f\text{.}\) Fix \(\varepsilon >0\text{.}\) Fix \(\delta>0\) such that if \(\{(a_i,b_i)\}_{i=1}^n\) are disjoint open intervals such that \(\sum_{i=1}^n(b_i-a_i) < \delta\) then
    🔗
    \begin{equation*} \sum_{i=1}^n |f(b_i) -f(a_i)|<\varepsilon/2. \end{equation*}
    Fix disjoint open intervals \(\{(a_i,b_i)\}_{i=1}^n\) with \(\sum_{i=1}^n(b_i-a_i) < \delta\text{.}\) For each \(i\text{,}\) let \(P_i\) be a partition of \([a_i,b_i]\text{.}\) Then
    🔗
    \begin{equation*} \sum_{i=1}^n V(f,P_i) < \varepsilon/2. \end{equation*}
    Thus
    🔗
    \begin{equation*} \begin{aligned} \varepsilon > \varepsilon/2 & \geq \sum_{i=1}^n TV(f|_{[a_i,b_i]}) \\ &= \sum_{i=1}^n T_f(b_i) - T_f(a_i). \end{aligned} \end{equation*}
    Hence \(T_f\) is absolutely continuous. It follows that \(f_1\) and \(f_2\) are absolutely continuous.
    🔗
    🔗
    The following proposition shows that what we have just found Lipschitz functions is not a happy coincidence. We have a one-to-one correspondence with absolutely continuous sign-measures and absolutely continuous functions.
    🔗

    Proposition 9.2.7.

    Let \([a,b]\) be a closed interval in \(\mathbb{R}\text{.}\) Let \(\nu\) be a signed Borel measure on \([a,b]\text{,}\) and let \(f(x) = \nu([a,x])\text{.}\) Then \(\nu\) is absolutely continuous with respect to the Lebesgue measure \(m\text{,}\) if and only if \(f\) is an absolutely continuous function and \(f(a) = 0\text{.}\)
    🔗
    🔗

    Proof.

    Suppose \(\nu \ll m\text{.}\) First suppose that \(\nu\) is a measure, i.e. \(\nu(E)\geq 0\) for all Borel sets \(E \subseteq [a,b]\text{.}\) Fix \(\varepsilon > 0\text{.}\) By Proposition Proposition 9.2.1, there exists \(\delta >0\) such that, if \(E \subseteq [a,b]\) is a Borel set with \(m(E) < \delta\text{,}\) then \(\nu(E) < \varepsilon\text{.}\) Let \(\{[a_k,b_k]\}_{k=1}^n\) be a family of disjoint intervals in \([a,b]\) such that \(\sum_k (b_k - a_k) < \delta\text{.}\) Note that, since \(\nu \ll m\text{,}\)
    🔗
    \begin{equation*} \nu((c,d)) = \nu([c,d]) = \nu((c,d]) \end{equation*}
    for any \(a\leq c \leq d < b.\) Thus
    🔗
    \begin{equation*} \begin{aligned} \varepsilon &> \nu\left(\bigcup_{k=1}^n (a_k,b_k]\right) \\ &= \sum_{k=1}^n \nu((a_k,b_k]) \\ &= \sum_{k=1}^n f(b_k) - f(a_k) \\ &= \sum_{k=1}^n |f(b_k) - f(a_k)|. \end{aligned} \end{equation*}
    Hence \(f\) is an absolutely continuous function.
    🔗
    Suppose now that \(\nu\) is a signed Borel measure. Let \(\nu = \nu_+ - \nu_-\) be the Jordan decomposition of \(\nu\) (Theorem Theorem 6.1.9). Then, by Proposition Proposition 6.2.15, the Jordan decomposition of \(f\) is given by \(f = f_1 - f_2\) where \(f_1\) is the cumulative distribution function of \(\nu_+\) and \(f_2\) is the cumulative distribution function of \(\nu_-\text{.}\) If \(\nu \ll m\) then, by definition, \(|\nu| = \nu_+ + \nu_-\) is absolutely continuous with respect to \(m\text{.}\) Hence, \(\nu_+ \ll m\) and \(\nu_- \ll m\text{.}\) It follows by the above argument that \(f_1\) and \(f_2\) are absolutely continuous, and hence \(f\) is absolutely continuous.
    🔗
    Suppose now that \(f\) is absolutely continuous and \(f(a)=0\text{.}\) Again, first assume that \(\nu\) is a measure, so that \(f\) is an increasing function. By the continuity of \(f\) and the continuity of measure, for any \(a\leq c < d < b\text{,}\)
    🔗
    \begin{equation*} \begin{aligned} \nu((c,d]) &= f(d) - f(c) \\ & = \lim_{x \rightarrow d^-}f(x) - f(c) \\ &= \lim_{x \rightarrow d^-} \nu((c,x]) \\ & = \nu((c,d)). \end{aligned} \end{equation*}
    Fix \(\varepsilon>0\text{.}\) Fix \(\delta >0\) so that if \(\{(a_k,b_k)\}_{k=1}^n\) are disjoint open intervals in \([a,b]\) with \(\sum_{k=1}^n (b_k - a_k) < \delta\text{,}\) then
    🔗
    \begin{equation*} \nu\left( \bigcup_{k=1}^n (a_k,b_k)\right) = \sum_{k=1}^n \left[ f(b_k) - f(a_k) \right] < \varepsilon/2. \end{equation*}
    Let \(U \subseteq [a,b]\) be an open set with \(m(U) < \delta\text{.}\) There are disjoint intervals \(\{(a_k,b_k)\}_{k=1}^\infty\) such that \(U = \bigcup_{k=1}^\infty (a_k,b_k)\text{.}\) Hence
    🔗
    \begin{equation*} \sum_{k=1}^\infty [b_k -a_k] < \delta. \end{equation*}
    Thus, for any \(n\text{,}\)
    🔗
    \begin{equation*} \nu\left( \bigcup_{k=1}^n (a_k,b_k) \right) < \varepsilon/2. \end{equation*}
    Hence \(\nu(U) \leq \frac{\varepsilon}{2} < \varepsilon\text{.}\)
    🔗
    Take now any Borel set \(E \subseteq [a,b]\) with \(m(E) < \delta/2\text{.}\) By the outer regularity of the Lebesgue measure, there is an open set \(U\) with \(E \subseteq U\) and \(m(U) < \delta\text{.}\) Hence \(\nu(E) \leq \nu(U) < \varepsilon.\) Hence, by Proposition Proposition 9.2.1, \(\nu\) is absolutely continuous with respect to \(m\text{.}\)
    🔗
    Now suppose \(\nu\) is a Borel signed measure, such that \(f(x) = \nu([a,x])\) is absolutely continuous. As before, let \(f_1\) and \(f_2\) be the cumulative distribution functions of \(\nu_+\) and \(\nu_-\text{,}\) respectively. As \(f\) is absolutely continuous, \(f_1\) and \(f_2\) are absolutely continuous by Proposition Proposition 9.2.6. By the above argument, it follows that \(\nu_+ \ll m\) and \(\nu_- \ll m\text{.}\) Hence, we have that \(\nu\) is absolutely continuous with respect to \(m\text{.}\)
