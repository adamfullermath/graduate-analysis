    Section 1.3 Change of variables

    In this section we will study how changes of variables in \(\mathbb{R}^n\) effect the integral. We begin by noting that all linear maps from \(\mathbb{R}^n\) to \(\mathbb{R}^n\) are continuous.
    🔗

    Lemma 1.3.1.

    Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be a linear map. Then \(T\) is continuous.
    🔗
    🔗

    Proof.

    Let \(e_1,\ldots,e_n\) be the canonical basis of \(\mathbb{R}^n\text{.}\) That is, let \(e_1= (1,0,\ldots,0)\text{,}\) \(e_2 = (0,1,0,\ldots,0)\text{,}\) and so on. Take \(x = (x_1,\ldots,x_n) \in \mathbb{R}^n\text{.}\) Then
    \begin{align*} \|T(x)\|_2 \amp= \| x_1 T(e_1) + \ldots + x_nT(e_n)\|_2 \\ \amp\leq \sum_{k=1}^n |x_k| \|T(e_k)\|_2 \\ \amp\leq \left(\sum_{k=1}^n\|T(e_k)\|_2^2\right)^{1/2} \|x\|_2, \end{align*}
    where the last inequality is an application of the Cauchy-Schwarz inequality. Letting \(M = \left(\sum_{k=1}^n\|T(e_k)\|_2^2\right)^{1/2}\text{,}\) we have shown that \(\|T(x)\|_2 \leq M \|x\|_2\) for all \(x \in \mathbb{R}^n\text{.}\) Hence, by Proposition III.2.1.1, \(T\) is continuous.
    🔗
    🔗

    Remark 1.3.2.

    Recall that the norm of a linear functional \(\phi\) on a normed vector space \(X\) is defined as
    \begin{equation*} \|\phi\|_\mathrm{op} = \inf\{M \colon |\phi(x)|\leq M \|x\| \text{ for all } x \in X \}. \end{equation*}
    We can use Lemma 1.3.1 to similarly define a norm on the space of linear maps from \(\mathbb{R}^n\) to \(\mathbb{R}^n\text{.}\) That is, if \(T\) is a linear map from \(\mathbb{R}^n\) to \(\mathbb{R}^n\) we can define \(\|T\|_\mathrm{op}\) as
    \begin{equation*} \|T\|_\mathrm{op} = \inf\{M \colon \|T(x)\|_2 \leq M \|x\|_2 \text{ for all } x \in \mathbb{R}^n\}. \end{equation*}
    We will not not make too much use of the fact that this is a norm in these notes, but it plays an important role in functional analysis.
    🔗
    🔗
    If \(f\) is a Lebesgue integrable function on \(\mathbb{R}^n\text{,}\) and \(T\) is an invertible linear map on \(\mathbb{R}^n\text{,}\) we want to compare the integrals \(\intd{\mathbb{R}^n}{f}{m_n}\) and \(\intd{\mathbb{R}^n}{f\circ T}{m_n}\text{.}\) It is not (yet) clear that the second integral is defined. In general, the composition of a measurable function and a continuous function need not be continuous.
     1 
    Note that we can not apply Proposition II.5.1.15 here.
    Thus, in order to proceed, we must first show that \(f \circ T\) is measurable. As we are ultimately interested in the case when \(T\) is an invertible linear map, we will restrict to this case (though conclusions of Lemma 1.3.3, Lemma 1.3.4, and Proposition 1.3.5 hold for all linear maps).
    🔗
    Let \(e_1,\ldots,e_n\) be the standard orthonormal basis of \(\bR^n\text{.}\) Recall that if \(T\) is an invertible linear map in \(\bR^n\text{,}\) then \(T\) can be written as product of matrices of the form:
     2 
    We are just using the fact that any invertible matrix can be row-reduced to the identity matrix.
    Type 1
    🔗
    there is an \(i\) such that \(Te_i = ce_i\) for some real \(c\neq 0\text{,}\) and \(Te_j = e_j\) for all \(j\neq i\text{;}\)
    Type 2
    🔗
    there is an \(i < n\) such that \(Te_i = e_i + e_{i+1}\text{,}\) and \(Te_k = e_k\) for all \(k\neq i\text{;}\)
    Type 3
    🔗
    there is an \(i < n\) such that \(Te_i = e_{i+1}\text{,}\) \(Te_{i+1} = e_i\text{,}\) and \(Te_k = e_k\) for all \(k\neq i,i+1\text{.}\)
    Note that, if \(T\) is of Type 1, then \(\det(T) = c\text{,}\) if \(T\) is of Type 2, then \(\det(T) = 1\text{,}\) and if \(T\) is of Type 3, then \(\det(T) = -1\text{.}\)
    🔗

    Lemma 1.3.3.

    Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map. Then
    \begin{equation*} m_n^*(T(E)) \leq |\det(T)| m_n^*(E) \end{equation*}
    for all \(E \subseteq \mathbb{R}^n\text{.}\)
    🔗
    🔗

    Proof.

    Let \(I = I_1 \times \ldots \times I_n\) be an interval in \(\mathbb{R}^n\text{.}\) If \(T\) is of Type 1 for some non-zero constant \(c\text{,}\) then \(T(I)\) is an interval and \(m_n(T(I)) = |c| m_n(I) = |\det(T)| m_n(I)\text{.}\) Similarly, if \(T\) is of Type 3, then \(T(I)\) is an interval and \(m_n(T(I)) = m_n(I) = |\det(T)| m_n(I)\text{.}\)
    🔗
    Suppose \(T\) is of Type 2 and \(n=2\text{.}\) As \(T\) is linear, we can assume that \(I = [0,a]\times[0,b]\) for some \(a,b\geq 0\text{.}\) Then
    \begin{equation*} T(I) = \{ (x,y) \colon 0\leq x \leq a,\ ax \leq y \leq ax + b\} \end{equation*}
    is a parallelogram in \(\bR^2\text{.}\) By Exercise 1.4.8, \(T(I)\) is measurable and
    \begin{equation*} m_2(T(I)) =ab = m_2(I). \end{equation*}
    Note that, if \(n > 2\) and \(T\) is of Type 2, then \(T\) only changes on the \(i\) and \(i+1\) coordinates of an \(x \in \bR^n\text{,}\) and leaves all other coordinates fixed. Thus, by the \(n = 2\) case, \(T(I)\) is measurable and \(m_n(T(I)) = m_n(I) = |\det(T)| m_n(I)\text{.}\)
    🔗
    Suppose now, that \(T\) is of Type 1, Type 2, or Type 3. Take any \(E \subseteq \bR^n\) with \(m_n^*(E)<\infty\text{.}\) Let \(\{I_k\}_{k=1}^\infty\) be a countable collection of intervals in \(\bR^n\) such that \(E \subseteq \bigcup_{k=1}^\infty I_k\text{.}\) Then
    \begin{align*} m_n^*(T(E)) \amp \leq \sum_{k=1}^\infty m_n(T(I_k)) \\ \amp = |\det(T)| \sum_{k=1}^\infty m_n(I_k). \end{align*}
    Taking an infimum over all such collections of intervals, we have that \(m_n^*(T(E)) \leq |\det(T)| m_n^*(E)\text{.}\)
    🔗
    Now suppose that \(T\) is an arbitrary invertible linear map on \(\bR^n\text{.}\) Then \(T = T_1\ldots T_k\) where each \(T_i\) is of Type 1, Type 2, or Type 3. Thus, for \(E \subseteq \bR^n\)
    \begin{align*} m_n^*(T(E)) \amp \leq |\det(T_1)| m_n^*(T_2T_3\ldots T_k (E)) \\ \amp \leq |\det(T_1)| \ldots|det(T_k)| m_n^*(E) \\ \amp = |\det(T)| m_n^*(E), \end{align*}
    completing the proof.
    🔗
    🔗

    Lemma 1.3.4.

    Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map. Then, if \(E \subseteq \bR^n\) is Lebesgue measurable, \(T(E)\) is Lebesgue measurable.
    🔗
    🔗

    Proof.

    Let \(E \subseteq \mathbb{R}\) be a bounded Lebesgue measurable set. By Corollary 1.2.9 there is an \(F_\sigma\)-set \(F \subseteq E\) such that \(m_n(E) = m_n(F) = 0\text{.}\) Let \(B = E\backslash F\text{.}\) Then \(m_n(B) = 0\text{.}\)
    🔗
    Note that, since \(T\) is linear, \(T^{-1}\) is linear, and therefore continuous. Hence \(T\) sends closed sets to closed sets, and hence \(F_\sigma\) sets to \(F_\sigma\) sets. Further, since \(T\) is continuous, there is an \(M \geq 0\text{,}\) such that \(\|T(x)\|_2 \leq M \|x\|_2\text{.}\) Hence, \(T\) sends bounded sets to bounded sets. Hence \(T(F)\) is a bounded \(F_\sigma\) set. In particular, \(T(F)\) is Lebesgue measurable.
    🔗
    By Lemma 1.3.3,
    \begin{equation*} m_n^*(T(B))\leq |\det(T)|m_n(B) = 0. \end{equation*}
    Hence, \(m_n^*(T(B))=0\text{,}\) and hence \(T(B)\) is Lebesgue measurable. Thus,
    \begin{equation*} T(E) = T(F) \cup T(B) \end{equation*}
    is Lebesgue measurable. As any measurable set is a countable union of bounded measurable sets, the result follows for all Lebesgue measurable sets.
    🔗
    🔗

    Proposition 1.3.5.

    Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map and let \(f\) be a Lebesgue measurable function on \(\mathbb{R}^n\text{,}\) then \(f \circ T\) is Lebesgue measurable.
    🔗
    🔗

    Proof.

    Take, \(U \subseteq \bR\) open. Since, \(f\) is measurable, \(f^{-1}(U)\) is a Lebesgue measurable set in \(\bR^n\text{.}\) By Lemma 1.3.4, \(T^{-1}(f^{-1}(U))\) is a Lebesgue measurable set in \(\bR^n\text{.}\) Thus, \((f\circ T)^{-1}\) is Lebesgue measurable.
    🔗
    🔗
    We state now the main theorem of this section. We delay the proof for now, to deal with some special cases.
    🔗

    Theorem 1.3.6.

    Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map and let \(f\) be a Lebesgue integrable function on \(\mathbb{R}^n\text{.}\) Then \(f\circ T\) is integrable and
    \begin{equation*} \int_{\mathbb{R}^n} f\circ T \, d\mu = \frac{1}{\left|\det(T)\right|} \int_{\mathbb{R}^n} f \, d\mu. \end{equation*}
    🔗
    🔗
    The proof of Theorem 1.3.6 will ultimately follow an inductive argument. It suffices to show that the result holds for linear maps of Type 1, Type 2, and Type 3. We will prove the \(n=1\) and \(n=2\) cases first, and then use these cases to prove the general case. We leave the \(n=1\) case, Lemma 1.3.7, as an exercise.
    🔗

    Lemma 1.3.7.

    Let \(f \colon \bR \rightarrow \bR\) be a Lebesgue integrable function. Then if \(\alpha \in \bR\) is non-zero,
    \begin{equation*} \int_\bR f\,dm = |\alpha| \int_\bR f(\alpha x) \, dm(x). \end{equation*}
    🔗
    🔗

    Lemma 1.3.8.

    Let \(f \colon \bR^2 \rightarrow \bR\) be a Lebesgue integrable function. Let \(c \in \bR\) be non-zero. Define functions \(g,h,k\) on \(\bR^2\) by
    \begin{equation*} g(x,y) = f(y,x),\ h(x,y) = f(x,x+y),\text{ and } k(x,y) = f(cx,y). \end{equation*}
    Then \(g,h,\) and \(k\) are integrable, and
    \begin{equation*} \int_{\bR^2} g \, dm_2 = \int_{\bR^2} h \, dm_2 = \int_{\bR^2} f \, dm_2, \end{equation*}
    and
    \begin{equation*} \int_{\bR^2} k \, dm_2 = \frac{1}{|c|} \int_{\bR^2} f \, dm_2. \end{equation*}
    🔗
    🔗

    Proof.

    Note that \(h,g,\) and \(k\) are measurable by Proposition 1.3.5. We will prove the result for \(h\text{.}\) We will assume that \(f\) is non-negative. Then general case follows from writing \(f\) as the difference of its positive and negative parts. The proofs for \(g\) and \(k\) are similar, with an application of Lemma 1.3.7 in the case of \(k\text{.}\) Note that, if \(l \colon \bR \rightarrow \bR\) is Lebesgue integrable and \(\beta \in \bR\) then
    \begin{equation} \int_\bR l(x+\beta)\,dm(x) = \int_\bR l\,dm.\tag{1.2} \end{equation}
    By Tonelli’s theorem, we have that
    \begin{align*} \int_{\bR^2}h\,dm_2 \amp = \int_\bR \int_\bR h(x,y)\, dm(y)dm(x) \\ \amp = \int_\bR \int_\bR f(x,x+y)\, dm(y)dm(x) \\ \amp = \int_\bR \int_\bR f(x,y)\, dm(y)dm(x). \end{align*}
    🔗
    🔗
    We can now prove Theorem 1.3.6.
    🔗

    Proof of Theorem 1.3.6.

    Lemma 1.3.7 and Lemma 1.3.8 show that the result holds for \(n=1,2\text{.}\) Assume now that \(n\geq 3\) and the result holds for \(\bR^k\) when \(k< n\text{.}\) Assume that \(T\) is of Type 1, Type 2, and Type 3. Then, either \(Te_1 = e_1\) and \(T \{(0,x_2,\ldots,x_n)\} \subseteq \{(0,x_2,\ldots,x_n)\}\text{,}\) or \(Te_n = e_n\) and \(T\{(x_1,x_2,\ldots,0)\} \subseteq \{(x_1,x_2,\ldots,0)\}\text{.}\) Assume the former, the latter case is done similarly. Let \(T'\) be the restriction to the \(n-1\)-dimensional space \(\{(0,x_2,\ldots,x_n)\} \text{.}\) Then, by Fubini’s Theorem (and Proposition 1.3.5)
    \begin{align*} \int_\bR^n f\circ T \,dm_n \amp= \int_\bR \left( \int_{\bR^{n-1}} f(x_1, T'(x_2,\ldots,x_n)))\,dm_{n-1}\right)dm\\ \amp = \frac{1}{|\det(T')|}\int_\bR \left( \int_{\bR^{n-1}} f(x_1, x_2,\ldots,x_n))\,dm_{n-1}\right)dm\\ \amp = \frac{1}{|\det(T')|}\int_\bR^n f \,dm_n. \end{align*}
    Noting that \(\det(T) = \det(T')\) completes the proof.
    🔗
    🔗

    Corollary 1.3.9.

    Let \(T \colon \bR^n\rightarrow \bR^n\) be an invertible linear map and let \(E \subseteq \bR^n\) be a Lebesgue measurable set with \(m_n(E) < \infty\text{.}\) Then, \(|\det(T)|m_n(T(E))=m_n(E).\)
    🔗
    🔗

    Proof.

    Let \(f = \chi_E\text{.}\) Note that
    \begin{equation*} \chi_E \circ T = \chi_{T(E)}. \end{equation*}
    Thus, by Theorem 1.3.6,
    \begin{equation*} m_n(T(E)) = \int_\bR^n \chi_{T(E)}\,dm_n = \frac{1}{|\det(T)|}\int_\bR^n \chi_E\,dm_n = \frac{1}{|\det(T)|}m_n(E), \end{equation*}
    and so \(|\det(T)|\det(T(E)) = m_n(E).\)
