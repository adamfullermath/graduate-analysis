    Section 6.2 Integration of measurable functions

    Recall that if \(f\) is a measurable function on a measure space \((X,\mathcal{F},\mu)\) then \(f = f_+ - f_-\) where \(f_+ = \max\{0,f\}\) is the positive part of \(f\) and \(f_- = \max\{0,-f\}\) is the negative part of \(f\text{.}\) Further, \(|f| = f_+ + f_-\text{.}\) As \(|f|\text{,}\) \(f_+\) and \(f_-\) are non-negative measurable functions we can use Theorem 6.1.20 to get
    🔗
    \begin{equation*} \int_X |f| \,d\mu = \int_X f_+ \,d\mu + \int_X f_-\,d\mu. \end{equation*}
    Thus \(|f|\) is integrable if and only if \(f_+\) and \(f_-\) are integrable.
    🔗

    Definition 6.2.1.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be a measurable function on \(X\) with positive part \(f_+\) and negative part \(f_-\text{.}\) The function \(f\) is integrable with respect to \(\mu\) if \(|f|\) is integrable. That is, \(f\) is integrable if \(\int_X |f|\,d\mu < \infty\text{.}\) If \(f\) is integrable we define the integral of \(f\) over \(X\) with respect to \(\mu\) by
    🔗
    \begin{equation*} \int_X f\,d\mu = \int_X f_+ \,d\mu - \int_X f_- \,d\mu. \end{equation*}
    If \(f\) is integrable and \(E \subseteq X\) is measurable, we define the integral of \(f\) over \(E\) with respect to \(\mu\) as
    🔗
    \begin{equation*} \int_E f \,d\mu = \int_X f\chi_E \,d\mu. \end{equation*}
    When \(X \subseteq \mathbb{R}\) is Lebesgue measurable, \(\mathcal{F}\) is the Lebesgue measurable subsets of \(X\text{,}\) and \(\mu = m\) is the Lebesgue measure, an integrable function \(f\) is called Lebesgue integrable and the integral \(\int_X f\,d\mu\) is called the Lebesgue integral.
    🔗
    🔗
    We first note the following important property relating the integral of \(f\) and the integral of \(|f|\text{.}\)
    🔗

    Proposition 6.2.2.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be an integrable function on \(X\text{.}\) Then
    🔗
    \begin{equation*} \left| \int_X f\,d\mu \right| \leq \int_X |f| \,d\mu. \end{equation*}
    🔗

    Proof.

    Let \(f = f_+ - f_-\) be the decomposition of \(f\) into its positive and negative parts. Then
    🔗
    \begin{equation*} \begin{aligned} \left| \int_X f\,d\mu \right| & = \left| \int_X f_+\,d\mu - \int_X f_-\,d\mu \right| \\ &\leq \int_X f_+\,d\mu + \int_X f_-\,d\mu \\ &= \int_X |f|\,d\mu. \end{aligned} \end{equation*}
    🔗

    Theorem 6.2.3. Linearity and Monotonicity of Integration.

    Let \((X,\mathcal{F},\mu)\) be a measure space, let \(f\) and \(g\) be integrable functions on \(X\text{,}\) and let \(\alpha, \beta \in\mathbb{R}\text{.}\) Then \(\alpha f + \beta g\) is integrable and
    🔗
    \begin{equation*} \int_X (\alpha f + \beta g)\,d\mu = \alpha \int_X f\,d\mu + \beta \int_X g\, d\mu. \end{equation*}
    Further, it \(f \leq g\) then
    🔗
    \begin{equation*} \int_X f\, d\mu \leq \int_X g\, d\mu. \end{equation*}
    🔗

    Proof.

    It is straightforward to show that \(\int_X \alpha f\,d\mu = \alpha \int_X f\,d\mu\text{.}\) We will show \(\int_X(f+g)\,d\mu = \int_X f\,d\mu + \int_X g\,d\mu\text{.}\) First note that by Theorem 6.1.13, \(f\) and \(g\) being integrable implies \(f\) and \(g\) are finite a.e. Thus \(f + g\) is defined. Note that \(|f + g| \leq |f| + |g|\text{.}\) By the monotonicity of integration of non-negative measurable functions, it follows that \(f+g\) is integrable.
    🔗
    As \(f+g = (f+g)_+ - (f+g)_-\text{,}\) \(f = f_+ - f_-\text{,}\) and \(g = g_+ -g_-\text{,}\) it follows that
    🔗
    \begin{equation*} (f+g)_+ - (f+g)_- = (f_+ - f_-) + (g_+ -g_-) \end{equation*}
    and so
    🔗
    \begin{equation*} (f+g)_+ +f_- + g_- =(f+g)_- +f_+ + g_+. \end{equation*}
    By Theorem 6.1.20, we have then that
    🔗
    \begin{equation*} \int_X(f+g)_+\,d\mu +\int_Xf_-\,d\mu + \int_Xg_-\,d\mu =\int_X(f+g)_-\,d\mu +\int_Xf_+\,d\mu + \int_Xg_+\,d\mu \end{equation*}
    Thus
    🔗
    \begin{equation*} \int_X(f+g)_+\,d\mu - \int_X(f+g)_-\,d\mu = \int_Xf_+\,d\mu -\int_Xf_-\,d\mu + \int_Xg_+\,d\mu - \int_Xg_-\,d\mu, \end{equation*}
    and hence
    🔗
    \begin{equation*} \int_X (f+ g)\,d\mu = \int_Xf\,d\mu + \int_Xg\,d\mu \end{equation*}
    If \(f\leq g\text{,}\) then \(g-f \geq 0\) and so
    🔗
    \begin{equation*} 0 \leq \int_X (g-f)\,d\mu = \int_X g\,d\mu - \int_X f\,d\mu. \end{equation*}
    Thus \(\int_X g\,d\mu \geq \int_X f\,d\mu\text{.}\)
    🔗
    🔗
    Several results can now be stated which follow readily from Theorem 6.2.3 and what we have already proved for integrals of non-negative linear functions.
    🔗

    Corollary 6.2.4.

    If \(A\) and \(B\) are disjoint measurable subsets in a measure space \((X,\mathcal{F},\mu)\) and \(f\) is integrable, then
    🔗
    \begin{equation*} \int_{A\cup B} f\,d\mu = \int_A f\,d\mu + \int_Bf\,d\mu. \end{equation*}
    🔗

    Proposition 6.2.5.

    If \((X,\mathcal{F},\nu)\) is a measure space, \((f_n)\) is a sequence of integrable functions on \(X\text{,}\) and \(f = \sum_{k=1}^\infty f_k\) is integrable, then
    🔗
    \begin{equation*} \int_X f \,d\mu = \sum_{k=1}^\infty \int_X f_k \,d\mu. \end{equation*}
    🔗

    Proof.

    Apply Corollary 6.1.18 and the linearity of integration.
    🔗
    🔗
    In Exercise 6.4.8 it is shown that \(\nu(E) = \int_E f\,d\mu\) defines a measure when \(f \geq 0\text{.}\) The following proposition is a straightforward application of Exercise 6.4.8, the linearity of integration, and the continuity of measure.
    🔗

    Proposition 6.2.6. Continuity of Integration.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be an integrable function on \(X\text{.}\)
    🔗
    1. If \(\{E_n\}_n\) is an ascending sequence of measurable sets in \(X\) with \(E = \bigcup_n E_n\) then
      \begin{equation*} \int_E f\,d\mu = \lim_n \int_{E_n} f\,d\mu. \end{equation*}
      🔗
      🔗
    2. If \(\{F_n\}_n\) is a descending sequence of measurable sets in \(X\) with \(F = \bigcap_n F_n\) then
      \begin{equation*} \int_F f\,d\mu = \lim_n \int_{F_n} f\,d\mu. \end{equation*}
      🔗
      🔗
    🔗
    The following theorem continues our theme of results which ask when a limit of integrals is the integral of the pointwise limit.
    🔗

    Theorem 6.2.7. Lebesgue Dominated Convergence Theorem.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \((f_n)\) be a sequence of measurable functions which converge pointwise a.e. to a measurable function \(f\text{.}\) If there is an integrable non-negative function \(g\) on \(X\) such that \(|f_n|\leq g\) a.e. for all \(n\text{,}\) then \(f\) is integrable over \(X\) and
    🔗
    \begin{equation*} \int_X f \,d\mu = \lim_n\int_X f_n\,d\mu. \end{equation*}
    🔗

    Proof.

    By removing a set of measure \(0\text{,}\) we will assume that the sequence \((f_n)_n\) converges pointwise to \(f\) and \(|f_n|\leq g\) everywhere. Note that each \(f_n\) is integrable since \(|f_n|\leq g\) and \(g\) is integrable. It follows that \(|f| \leq g\text{,}\) and so \(f\) is integrable as well.
    🔗
    The sequence \((f_n + g)_n\) is a sequence of non-negative measurable functions which converges pointwise to \(f +g\text{.}\) Thus, by Fatou’s Lemma,
    🔗
    \begin{equation*} \int_X(f+g)\,d\mu \leq \liminf_n \int_X (f_n + g)\,d\mu. \end{equation*}
    The linearity of integration thus gives that
    🔗
    \begin{equation} \int_X f \,d\mu \leq \liminf_n \int_X f_n\,d\mu.\tag{6.2} \end{equation}
    The sequence \((g - f_n)_n\) is also a sequence of non-negative measurable functions. This sequence converges pointwise to \(g -f\text{.}\) Applying Fatou’s Lemma and the linearity of integration we have
    🔗
    \begin{equation*} \begin{aligned} \int_X g\,d\mu - \int_X f\,d\mu = \int_X(g-f)\,d\mu &\leq \liminf_n \int_X (g - f_n)\,d\mu\\ &= \int_X g\,d\mu - \limsup_n \int_X f_n\,d\mu. \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation} \int_X f \,d\mu \geq \limsup_n \int_X f\,d\mu.\tag{6.3} \end{equation}
    By (6.2) and (6.3) we have that
    🔗
    \begin{equation*} \limsup_n \int_X f_n\,d\mu \leq \int_X f \,d\mu \leq \liminf_n \int_X f_n\,d\mu \end{equation*}
    It follows that \(\left(\int_X f_n\,d\mu\right)_n\) is a convergent sequence and \(\lim_n \int_X f_n\,d\mu = \int_X f\,d\mu\)
    🔗
    🔗

    Remark 6.2.8.

    Note that the Lebesgue Dominated Convergence Theorem is not a generalization of the Monotone Convergence Theorem. The Lebesgue Dominated Convergence Theorem only deals with integrable functions, whilst the Monotone Convergence Theorem allows for possibly infinite integrals.
    🔗
    However, if we restrict the Monotone convergence theorem to an increasing sequence \((f_n)_n\) of integrable non-negative functions which converge pointwise a.e. to an integrable function \(f\text{,}\) then this is a special case of the Lebesgue Decomposition Theorem.
