    Section 2.4 Linear functionals on \(C_0(X)\)

    Let \(X\) be a locally compact Hausdorff space. Our goal is to describe the dual space of \(C_0(X)\text{.}\) We begin with some examples.
    🔗

    Example 2.4.1.

    Let \(X\) be a locally compact Hausdorff space, and take any \(x \in X\text{.}\) The point-evaluation map
    🔗
    \begin{equation*} \phi_x(f) = f(x) \end{equation*}
    for all \(f \in C_0(X)\) is linear. Further, for \(f\in C_0(X)\text{,}\)
    🔗
    \begin{equation*} |\phi_x(f)|=|f(x)|\leq \|f\|_\infty. \end{equation*}
    Thus \(\varphi\) is a linear functional on \(C(X)\) with \(\|\varphi\|_\mathrm{op} \leq 1\text{.}\)
    🔗
    Let \(U\) be any open neighborhood of \(x\text{.}\) By Urysohn’s Lemma, there is a continuous function \(f \in C_c(X) \subseteq C_0(X)\) such that \(\|f\|_\infty =1\) and \(f(x) = 1\text{.}\) Therefore,
    🔗
    \begin{equation*} |\phi_x(f)| = f(x) = 1 = \|f\|_\infty, \end{equation*}
    and hence \(\|\phi_x\|=1\text{.}\)
    🔗
    🔗

    Example 2.4.2.

    Let \(X\) be a locally compact Hausdorff space with Borel sets \(\mathcal{B}(X)\text{,}\) and let \(\mu\) be a finite Radon measure on \((X,\mathcal{B}(X))\text{.}\) By Corollary II.5.1.5, every continuous real-valued function \(f\) on \(X\) is a measurable function on \((X,\mathcal{B}(X))\text{.}\) Since \(\mu\) is a finite measure, any bounded measurable function on \(X\) is integrable. Thus, if \(f \in C_0(X)\text{,}\) then \(f\) is integrable. Define a linear map from \(C_0(X)\) to \(\mathbb{R}\text{,}\) by
    🔗
    \begin{equation*} \phi_\mu(f) = \int_X f\,d\mu. \end{equation*}
    Note that, if \(f \in C_0(X)\text{,}\) then
    🔗
    \begin{equation*} |\phi_\mu(f)| = \left|\int_X f\,d\mu\right| \leq \|f\|_\infty \mu(X). \end{equation*}
    Hence, \(\phi_\mu\) is a linear functional on \(C_0(X)\) and \(\|\phi_\mu\|_\mathrm{op} \leq \mu(X)\text{.}\)
    🔗
    Since \(\mu\) is a Radon measure, there is an increasing sequence \(\{K_n\}_n\) of compact subsets of \(X\) such that
    🔗
    \begin{equation*} \mu(X) = \lim_{n\rightarrow\infty}\mu(K_n). \end{equation*}
    By Urysohn’s Lemma, for each \(n\) there is a compactly supported positive function \(f_n\) on \(X\) such that
    🔗
    \begin{equation*} f_n(x) = 1 \text{ for all }x\in K_n \text{ and }\|f_n\|_\infty = 1. \end{equation*}
    For each \(n\text{,}\)
    🔗
    \begin{equation*} |\phi_\mu(f_n)| =\int_X f_n\,d\mu \geq \int_{K_n} f_n\,d\mu = \mu(K_n). \end{equation*}
    Hence, for each \(n\text{,}\)
    🔗
    \begin{equation*} \|\phi_\mu\|_\mathrm{op} = \sup\{|\phi_\mu(f)|\colon f\in C_0(X),\ \|f\|_\infty = 1\} \geq \mu(K_n). \end{equation*}
    Thus
    🔗
    \begin{equation*} \|\phi_\mu\|_\mathrm{op} = \mu(X). \end{equation*}
    🔗

    Remark 2.4.3.

    Note that Example 2.4.1 is a special case of Example 2.4.2. Indeed, if \(x\in X\text{,}\) and \(\delta_x\) is the Dirac measure at \(x\text{,}\) then
    🔗
    \begin{equation*} \int_X f\,d\delta_x = f(x) \end{equation*}
    by Exercise II.6.4.3.
    🔗
    🔗
    Let \(\nu\) be a finite Radon signed measure on a locally compact space \(X\text{.}\) Let \(\nu= \nu_+ - \nu_-\) be the Jordan decomposition of \(\nu\text{.}\) Then both \(\nu_+\) and \(\nu_-\) are finite Radon measures. If \(f\) is a Borel measurable function on \(X\) which is integrable with respect to \(|\nu|\text{,}\) we can define the integral of \(f\) with respect to \(\nu\) by
    🔗
    \begin{equation*} \int_X f\,d\nu = \int_X f\,d\nu_+ - \int_X f\,d\nu_-. \end{equation*}
    We can use this to extend Example 2.4.2.
    🔗

    Example 2.4.4.

    Let \(\nu\) be a finite Radon signed measure on a locally compact Hausdorff space \(X\text{.}\) For each \(f \in C_0(X)\) define \(\phi_\nu\) by
    🔗
    \begin{equation*} \phi_\nu(f) = \int_X f\,d\nu. \end{equation*}
    Clearly \(\phi_\nu\) is linear, by the linearity of integration. Note that,
    🔗
    \begin{equation*} \begin{aligned} \left| \int_X f \,d\nu\right| &= \left|\int_X f\,d\nu_+ - \int_X f\,d\nu_-\right| \\ &\leq \int_X|f|\,d\nu_+ + \int_X|f|\,d\nu_- \\ &= \int_X|f|\,d|\nu|, \end{aligned} \end{equation*}
    where the last equality follows from Exercise II.7.5.1. Thus, for \(f \in C_0(X)\text{,}\)
    🔗
    \begin{equation*} \begin{aligned} |\phi_\nu(f)| \leq |\phi_{|\nu|}(f)| \leq \|f\|_\infty |\nu|(X), \end{aligned} \end{equation*}
    where the last inequality comes from Example 2.4.2. Hence \(\phi_\nu \in C_0(X)^*\text{,}\) with \(\|\phi_\nu\|_\mathrm{op}\leq |\nu|(X).\)
    🔗
    🔗
    In Example 2.4.4 we showed \(\|\phi_\nu\|_\mathrm{op} \leq |\nu|(X)\) for a signed measure \(\nu\text{.}\) We have left open the question of the exact value of \(\|\phi_\nu\|_\mathrm{op}\text{.}\) This is answered in Theorem 2.4.12, where it is shown that \(\|\phi_\nu\|_\mathrm{op} = |\nu|(X)\text{.}\)
    🔗

    Definition 2.4.5.

    Let \(X\) be a locally compact Hausdorff space. A linear functional \(\varphi\) in the dual space of \(C_0(X)\) is positive if \(\varphi(f)\geq 0\) for all non-negative functions \(f \in C_0(X)\text{.}\)
    🔗
    🔗

    Remark 2.4.6.

    Note that, if \(\varphi\in C_0(X)^*\) is a positive linear functional, then
    🔗
    \begin{equation*} \|\varphi\|_\mathrm{op} = \sup \{\varphi(f) \colon f\geq 0,\ \|f\|_\infty = 1\}. \end{equation*}
    That is, the norm of \(\varphi\) can be calculated using only non-negative functions, see Exercise 2.5.14
    🔗
    🔗

    Example 2.4.7.

    The linear functionals described in Example 2.4.2 (and Example 2.4.1) are positive linear functionals. The linear functionals in Example 2.4.4 are not necessarily positive.
    🔗
    🔗
    Let \(\nu\) be a finite Radon signed measure on a locally compact Hausdorff space \(X\text{.}\) With the notation from Example 2.4.2 and Example 2.4.4, we have that
    🔗
    \begin{equation*} \phi_\nu = \phi_{\nu_+} - \phi_{\nu_-}, \end{equation*}
    when \(\nu = \nu_+ - \nu_-\) is the Jordan decomposition of \(\nu\text{.}\) That is, \(\phi_\nu\) is the difference of two positive linear functionals. We will show that we can write any linear functional on \(C_0(X)\) as the difference of two positive linear functionals. We have seen two theorems called the Jordan Decomposition Theorem: first for signed measures (Theorem II.4.1.9), and then for functions of bounded variation on closed bounded intervals of \(\mathbb{R}\) (Theorem II.4.2.10). In Proposition II.4.2.15 we saw that restricting Theorem II.4.1.9 to Borel signed measures on closed bounded intervals recovers Theorem II.4.2.10. That is, the Jordan Decomposition of functions of bounded variation is a special case of the Jordan decomposition of signed measures. We will now state a third theorem called the Jordan Decomposition Theorem, this time for linear functionals. Whilst not yet clear, we will eventually see that this too can be viewed as a special case of the Jordan decomposition of signed measures.
    🔗

    Theorem 2.4.8. Jordan Decomposition Theorem.

    Let \(X\) be a locally compact Hausdorff space and let \(\varphi\) be a linear functional on \(C_0(X)\text{.}\) There are positive linear functionals \(\phi_+,\phi_- \in C_0(X)^*\) such that
    🔗
    1. \(\varphi = \phi_+ - \phi_-\text{;}\) and
      🔗
      🔗
    2. \(\displaystyle \|\varphi\|_\mathrm{op} = \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}.\)
      🔗
      🔗
    🔗

    Proof.

    We will first define \(\phi_+\) on non-negative functions. When \(f \in C_0(X)\) with \(f\geq 0\) let
    🔗
    \begin{equation*} \phi_+(f) = \sup\{\varphi(g) \colon 0\leq g \leq f\}. \end{equation*}
    As \(\varphi(0)=0\text{,}\) it follows that \(\phi_+(f)\geq 0\) for all \(f\geq 0\) in \(C_0(X)\text{.}\) Also, if \(0 \leq g \leq f\text{,}\) then
    🔗
    \begin{equation*} |\varphi(g)| \leq \|\varphi\|_\mathrm{op} \|g\|_\infty \leq \|\varphi\|_\mathrm{op} \|f\|_\infty, \end{equation*}
    thus \(0\leq \phi_+(f)\leq \|\varphi\|_\mathrm{op} \|f\|_\infty\text{.}\)
    🔗
    We want to show that \(\phi_+\) is linear on the non-negative functions. If \(0\leq f \in C_0(X)\) and \(c \geq 0\text{,}\) then \(\phi_+(cf)=c\phi_+(f)\text{.}\) If \(0\leq g_1 \leq f_1\) and \(0\leq g_2 \leq f_2\text{,}\) then
    🔗
    \begin{equation*} 0\leq g_1 + g_2 \leq f_1 + f_2. \end{equation*}
    Thus
    🔗
    \begin{equation*} \phi_+(f_1 + f_2) \geq \phi_+(f_1) + \phi_+(f_2). \end{equation*}
    Conversely, if \(0 \leq g \leq f_1 + f_2\text{,}\) let \(g_1 = \min\{g,f_1\}\) and \(g_2 = g-g_1\text{.}\) Then \(0\leq g_1 \leq f_1\) and \(0\leq g_2 \leq f_2\text{,}\) and so
    🔗
    \begin{equation*} \varphi(g) = \varphi(g_1) + \varphi(g_2) \leq \phi_+ (f_1)+ \phi_+(f_2). \end{equation*}
    Hence
    🔗
    \begin{equation*} \phi_+(f_1+f_2)\leq \phi_+(f_1) + \phi_+(f_2). \end{equation*}
    Therefore \(\phi_+(f_1 +f_2)= \phi_+(f_1)+ \phi_+(f_2)\text{.}\)
    🔗
    We will now extend \(\phi_+\) to all of \(C_0(X)\text{.}\) If \(f \in C_0(X)\text{,}\) then \(f = f_+ - f_-\text{,}\) where \(f_+\) is the positive part of \(f\text{,}\) and \(f_-\) is the negative part of \(f\text{.}\) Define
    🔗
    \begin{equation*} \phi_+(f) = \phi_+(f_+)-\phi_+(f_-). \end{equation*}
    Note that, if \(f = g-h\) where \(g,h\geq 0\text{,}\) then
    🔗
    \begin{equation*} f_+ + h = g + f_-, \end{equation*}
    and so, by the linearity of \(\phi_+\) on positive functions,
    🔗
    \begin{equation*} \phi_+(f)= \phi_+(g)-\phi_+(h). \end{equation*}
    Thus, if \(f,g\in C_0(X)\text{.}\) Then
    🔗
    \begin{equation*} f+g = (f_+ + g_+) - (f_- + g_-), \end{equation*}
    and so \(\phi_+(f+g) = \phi_+(f) + \phi_+(g)\text{.}\) It is straightforward to show that
    🔗
    \begin{equation*} \phi_+(cf) = c\phi_+(f) \end{equation*}
    for all scalars \(c \in \mathbb{R}\) and \(f \in C_0(X)\text{.}\) Thus \(\phi_+\) is linear. Further, for any \(f\in C_0(X)\text{,}\)
    🔗
    \begin{equation*} \begin{aligned} |\phi_+(f)| &\leq \max\{|\phi_+(f_+)|,\phi_+(f_-)\}\\ &\leq \|\varphi\|_\mathrm{op} \max\{\|f_+\|_\infty,\|f_-\|_\infty\}\\ &= \|\varphi\|_\mathrm{op}\|f\|_\infty. \end{aligned} \end{equation*}
    Thus \(\phi_+ \in C_0(X)^*\) and \(\|\phi_+\|_\mathrm{op} \leq \|\varphi\|_\mathrm{op}\text{.}\)
    🔗
    Let \(\phi_- = \phi_+ - \varphi\text{.}\) Then \(\phi_- \in C_0(X)^*\text{,}\) and if \(f\geq 0\) then
    🔗
    \begin{equation*} \phi_-(f) = \sup\{\varphi(g) \colon 0 \leq g \leq f\}- \varphi(f)\geq 0. \end{equation*}
    Thus, \(\phi_-\) is positive.
    🔗
    It remains to show that \(\|\varphi\|_\mathrm{op} = \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}\text{.}\) Since, \(\varphi = \phi_+ - \phi_-\text{,}\) it is immediate that
    🔗
    \begin{equation*} \|\varphi\|_\mathrm{op} \leq \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}. \end{equation*}
    Take \(\varepsilon >0\text{.}\) Take non-negative \(f_1,f_2 \in C_0(X)\) with \(\|f_1\|_\infty=\|f_2\|_\infty = 1\) such that
    🔗
    \begin{equation*} \|\phi_+\| = \phi_+(f_1)-\varepsilon/2 \text{ and }\|\phi_-\| = \phi_-(f_2)-\varepsilon/2. \end{equation*}
    Let \(f = \max\{f_1,f_2\}.\) Then, \(\|f\|_\infty =1\) and
    🔗
    \begin{equation*} \|\phi_+\| = \phi_+(f)-\varepsilon/2 \text{ and }\|\phi_-\| = \phi_-(f)-\varepsilon/2. \end{equation*}
    Take any \(g \in C_0(X)\) with \(0\leq g \leq f\text{.}\) Then
    🔗
    \begin{equation*} -f \leq 2g- f \leq f, \end{equation*}
    and so \(\|2g-f\|_\infty \leq \|f\|_\infty=1\text{.}\) Thus
    🔗
    \begin{equation} \|\varphi\|_\mathrm{op} \geq \varphi(2g-f) = \varphi(g) + (\varphi(g) - \varphi(f)).\tag{2.1} \end{equation}
    Taking the supremum over all \(g\) with \(0\leq g \leq f\) in (2.1) we get
    🔗
    \begin{equation*} \|\varphi\|_\mathrm{op} \geq \phi_+(f) + \phi_-(f) = \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op} - \varepsilon. \end{equation*}
    This holds for all \(\varepsilon >0\text{,}\) and hence \(\|\varphi\|_\mathrm{op} \geq \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}\text{.}\) It now follows that \(\|\varphi\|_\mathrm{op} = \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}\text{,}\) as desired.
    🔗
    🔗

    Definition 2.4.9.

    Let \(X\) be a locally compact Hausdorff space, and let \(\varphi\) be a linear functional on \(C_0(X)\text{.}\) If \(\phi_+\) and \(\phi_-\) are positive linear functionals in \(C_0(X)^*\) such that \(\varphi = \phi_+ - \phi_-\) and \(\|\varphi\|_\mathrm{op} = \|\phi_+\|_\mathrm{op} + \|\phi_-\|_\mathrm{op}\text{,}\) then we call \(\phi_+\) and \(\phi_-\) the Jordan decomposition of \(\varphi\).
    🔗

    Theorem 2.4.11.

    Let \(X\) be a locally compact Hausdorff space and let \(\varphi\) be a positive linear functional on \(C_c(X)\text{.}\) There is a unique finite Radon measure \(\mu\) on \(X\) such that
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu \end{equation*}
    for all \(f \in C_c(X)\text{.}\) Further, \(\|\varphi\|= \mu(X)\text{.}\)
    🔗
    🔗

    Proof.

    For the purposes of this proof, we introduce a piece of notation: if \(U \subseteq X\) is an open set, and \(f \in C_c(X)\) satisfies \(0\leq f \leq 1\) and \(f\) has compact support inside \(U\text{,}\) we will write \(f \prec U\text{.}\)
    🔗
    When \(U \subseteq X\) is a non-empty open set, define
    🔗
    \begin{equation*} \mu(U) = \sup\{\varphi(f) \colon f \prec U\}, \end{equation*}
    and define \(\mu(\emptyset) =0\text{.}\) For arbitrary \(E \subseteq X\text{,}\) define
    🔗
    \begin{equation*} \mu^*(E) = \inf\{\mu(U) \colon E \subseteq U,\ U \text{ open}\}. \end{equation*}
    Note that, if \(U\) and \(V\) are open with \(U \subseteq V\text{,}\) then \(\mu(U) \subseteq \mu(V)\text{.}\) Hence, if \(U \subseteq X\) is open, \(\mu^*(U) = \mu(U)\text{.}\) We will show that \(\mu^*\) is an outer measure whose measurable sets include the Borel sets. We will then show that \(\mu^*\) restricts to a finite Radon measure \(\mu\) on the Borel sets satisfying
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu \end{equation*}
    for all \(f \in C_c(X)\text{.}\)
    🔗
    Let \(\{U_k\}_{k=1}^\infty\) be a collection of open sets, and let \(U\) be an open set satisfying
    🔗
    \begin{equation*} U \subseteq \bigcup_{k=1}^\infty U_k. \end{equation*}
    Take \(f \in C_c(X)\) with \(f \prec U\text{,}\) and let \(K \subseteq U\) be the compact support of \(U\text{.}\) Since \(K\) is compact, there is an \(n\text{,}\) such that
    🔗
    \begin{equation*} K \subseteq \bigcup_{k=1}^n U_k. \end{equation*}
    By Proposition~{prop: decomp of unity} there are functions \(h_1,\ldots,h_n \in C_c(X)\text{,}\) such that \(h_k \prec U_k\) for each \(1\leq k \leq n\text{,}\) and
    🔗
    \begin{equation*} \sum_{k=1}^n h_k(x) = 1 \end{equation*}
    for all \(x \in K\text{.}\) Hence \(f = \sum_{k=1}^n fh_k\text{,}\) and \(fh_k \prec U_k\) for \(1\leq k \leq n\text{.}\) Hence
    🔗
    \begin{equation*} \begin{aligned} \varphi(f) &= \sum_{k=1}^n \varphi(fh_k) \leq \sum_{k=1}^n \mu(U_k)\leq \sum_{k=1}^\infty \mu(U_k). \end{aligned} \end{equation*}
    Therefore,
    🔗
    \begin{equation*} \mu(U) = \sup\{ \varphi(f) \colon f\prec U\} \leq \sum_{k=1}^\infty \mu(U_k). \end{equation*}
    It follows that if \(E \subseteq X\text{,}\) then
    🔗
    \begin{equation*} \mu^*(E) = \inf\left\{\sum_{k=1}^\infty \colon \{U_k\}_{k=1}^\infty \text{ open},\ E \subset \bigcup_{k=1}^\infty U_k\right\}. \end{equation*}
    Hence, by Theorem II.2.3.2, \(\mu^*\) is an outer measure.
    🔗
    To show that Borel sets a \(\mu^*\)-measurable, it suffices to show that every open \(U \subseteq X\) is \(\mu^*\)-open. Fix open \(U \subseteq X\text{.}\) Take an open set \(E \subseteq U\text{,}\) and \(\mu(E) < \infty\text{.}\) Fix \(\varepsilon >0\text{.}\) Take \(f \prec E \cap U\text{,}\) such that \(\varphi(f)>\mu(E\cap U)- \varepsilon/2.\) Let \(K\) be the compact support of \(f\text{.}\) Then \(K\) is closed, and so \(E \cap K^c\) is open. Take \(g \prec E \cap K^c\text{,}\) such that \(\varphi(g)>\mu(E\cap K^c)- \varepsilon/2.\) Then \(f + g \prec E\text{,}\) and
    🔗
    \begin{equation*} \begin{aligned} \mu(E) &\geq \varphi(f+g) = \varphi(f) + \varphi(g)\\ &> \mu(E \cap U) + \mu(E \cap K^c) - \varepsilon\\ &\geq \mu(E \cap U) + \mu(E \cap U^c) - \varepsilon. \end{aligned} \end{equation*}
    Hence \(\mu(E) \geq \mu(E\cap U) + \mu(E \cap U^c)\text{.}\) As \(\mu^*\) is subadditive, it follows that \(\mu(E) = \mu(E \cap U) + \mu(E \cap U^c)\text{.}\) Now, suppose \(E \subseteq X\) is an arbitrary set with \(\mu^*(E)<\infty\text{.}\) Fix \(\varepsilon >0\text{.}\) There is an open set \(V\text{,}\) with \(E \subseteq V\text{,}\) such that
    🔗
    \begin{equation*} \mu(V) < \mu^*(E) + \varepsilon. \end{equation*}
    Using that \(V\) is open,
    🔗
    \begin{equation*} \begin{aligned} \mu^*(E) + \varepsilon &> \mu(V) = \mu(V\cap U) + \mu(V \cap U^c)\\ &\geq \mu^*(E\cap U) + \mu^*(E \cap U^c). \end{aligned} \end{equation*}
    As this holds for all \(\varepsilon>0\text{,}\) it folloows that \(\mu^*(E)\geq \mu^*(E\cap U) + \mu^*(E \cap U^c)\text{,}\) and hence \(\mu^*(E)= \mu^*(E\cap U) + \mu^*(E \cap U^c)\text{.}\) We have thus shown that all open subsets of \(X\) are \(\mu^*\)-measurable. Therefore all Borel sets are \(\mu^*\)-measurable.
    🔗
    Let \(\mu\) be the restriction of \(\mu^*\) to the Borel sets. We show that \(\mu\) is a finite Radon measure. That \(\mu\) is outer regular follows immediately from the definition of \(\mu\text{.}\) It remains to show that \(\mu(K)<\infty\) for all compact \(K \subseteq X\text{,}\) and that \(\mu\) is inner regular on open sets. Fix a compact set \(K \subseteq X\text{,}\) and fix non-negative \(f \in C_c(X)\) such that \(f(x)\geq 1\) for all \(x \in K\) (note that such an \(f\) exists by Urysohn’s Lemma). For \(\varepsilon>0\) let
    🔗
    \begin{equation*} U_\varepsilon= \{ x \in X \colon f(x) > 1-\varepsilon\}. \end{equation*}
    Then \(U_\varepsilon >0\) and \(K \subseteq U\text{.}\) If \(g \prec U_\varepsilon\text{,}\) then
    🔗
    \begin{equation*} \frac{1}{1-\varepsilon}f - g \geq 0. \end{equation*}
    Since \(\varphi\) is a positive linear functional, it follows that
    🔗
    \begin{equation*} \frac{\varphi(f)}{1-\varepsilon}\geq \varphi(g). \end{equation*}
    Taking the supremum over all \(g \prec U_\varepsilon\text{,}\) we have
    🔗
    \begin{equation*} \mu(K) \leq \mu(U_\varepsilon) \leq \frac{\varphi(f)}{1-\varepsilon}. \end{equation*}
    As this holds for all \(\varepsilon >0\text{,}\) it follows that
    🔗
    \begin{equation*} \mu(K) \leq \varphi(f). \end{equation*}
    Now take any open set \(U\) with \(K\subseteq U\text{.}\) By Urysohn’s Lemma, there is a \(f \prec U\) such that \(f(x) = 1\) for all \(x \in K\text{.}\) Thus
    🔗
    \begin{equation*} \mu(U)\geq \varphi(f)\geq \mu(K). \end{equation*}
    Since \(\mu\) is outer regular on \(K\text{,}\) it follows that
    🔗
    \begin{equation} \mu(K) = \inf\{\varphi(f) \colon f(x)\geq 1\text{ for }x\in K\}.\tag{2.2} \end{equation}
    Hence \(\mu(K)< \infty\text{.}\) Take now any open set \(U \subseteq X\text{.}\) Take any \(\alpha < \mu(U)\text{,}\) and choose \(f\prec U\) such that \(\varphi(f)>\alpha\text{.}\) Let \(K\) be the compact support of \(f\text{.}\) Note, if \(g \in C_c(X)\) and \(g(x) \geq 1\) for all \(x \in K\text{,}\) then \(g-f \geq 0\text{,}\) and therefore \(\varphi(g)\geq \varphi(f)>\alpha\text{.}\) Thus, by (2.2), \(\mu(K) > \alpha\text{.}\) We have shown that if \(\alpha <\mu(U)\text{,}\) then there is a compact set \(K \subseteq U\text{,}\) such that \(\alpha < \mu(K) \leq U\text{.}\) It follows that
    🔗
    \begin{equation*} \mu(U) = \sup\{\mu(K) \colon K \subseteq U\ \text{ compact}\}. \end{equation*}
    That is, \(\mu\) is inner regular on \(U\text{.}\) Thus \(U\) is a Radon measure. Finally,
    🔗
    \begin{equation*} \mu(X) = \sup\{\varphi(f) \colon f\prec X\} = \sup\{\varphi(f)\colon 0\leq f \leq 1\}= \|\varphi\|_\mathrm{op}<\infty. \end{equation*}
    Hence \(\mu\) is a finite Radon measure, and \(\|\varphi\|_\mathrm{op} = \mu(X)\text{.}\)
    🔗
    We will now show that
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu \end{equation*}
    for all \(f \in C_c(X)\text{.}\) Fix \(f \in C_c(X)\) with \(0\leq f \leq 1\) and fix \(N \in \mathbb{N}\text{.}\) For \(1\leq j \leq N\text{,}\) let
    🔗
    \begin{equation*} K_j = \left\{x\in X \colon f(x) \geq \frac{j}{N}\right\}. \end{equation*}
    Let \(K_0\) be the compact support of \(f\text{.}\) For \(1\leq j \leq N\text{,}\) let
    🔗
    \begin{equation*} f_j(x) = \min\left\{ \max\left\{ f(x)-\frac{j-1}{N},0 \right\},\frac{1}{N}\right\}. \end{equation*}
    Equivalently,
    🔗
    \begin{equation*} f_j(x) = \begin{cases}0 & \text{if }f(x) \notin K_{j-1}\\ f(x) - \frac{j-1}{N}& \text{if }f(x) \in K_{j-1} \backslash K_j\\ \frac{1}{N} & \text{if }f(x) \in K_j. \end{cases} \end{equation*}
    Then \(f_j \in C_c(X)\) for \(1\leq j \leq N\text{,}\) and \(f = \sum_{j=1}^N f_j\text{.}\)
    🔗
    For each \(1\leq j \leq N\text{,}\)
    🔗
    \begin{equation*} \frac{\mu(K_j)}{N} \leq \int_X f_j \,d\mu \leq \frac{1}{N}\mu(K_{j-1}). \end{equation*}
    Note, if \(K_{j-1}\subseteq U\) for an open set \(U\text{,}\) then \(Nf_j \prec U\text{.}\) Thus, \(\varphi(f_j) \leq \mu(U)/N\text{.}\) Hence, since \(\mu\) is outer regular and by (2.2),
    🔗
    \begin{equation*} \frac{\mu(K_j)}{N} \leq \varphi(f_j) \leq \frac{1}{N}\mu(K_{j-1}). \end{equation*}
    Since \(f = \sum_{j=1}^N f_j\text{,}\) we have that
    🔗
    \begin{equation*} \begin{aligned} \frac{1}{N} \sum_{j=1}^N \mu(K_j)&\leq \int_X f\,d\mu \leq \frac{1}{N} \sum_{k=0}^{N-1}\mu(K_j), \text{ and}\\ \frac{1}{N} \sum_{j=1}^N \mu(K_j)&\leq \varphi(f) \leq \frac{1}{N} \sum_{k=0}^{N-1}\mu(K_j). \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation*} \left| \varphi(f) - \int_X f\,d\mu \right| \leq \frac{\mu(K_0)-\mu(K_N)}{N}\leq \frac{\mu(K_0)}{N}<\infty. \end{equation*}
    Taking the limit as \(N \rightarrow \infty\text{,}\) we see that
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu. \end{equation*}
    As every element of \(C_c(X)\) is a linear combination of norm one positive functions, we have \(\varphi(f) = \int_X f\,d\mu\) for all \(f \in C_c(X)\text{,}\) as desired.
    🔗
    It remains to show that \(\mu\) is unique. Suppose \(\nu\) is a finite Radon measure on \(X\) that also satisfies \(\varphi(f) = \int_X f\,d\nu\) for all \(f \in C_c(X)\text{.}\) Take an open set \(U \subseteq X\text{.}\) If \(f \prec U\text{,}\) then
    🔗
    \begin{equation*} \varphi(f) = \int_X \,d\nu \leq \nu(U). \end{equation*}
    Also, if \(K \subseteq U\) is compact, then by Urysohn’s lemma there is an \(f \prec U\) such that \(f(x) = 1\) for all \(x \in K\text{.}\) Thus,
    🔗
    \begin{equation*} \nu(K) \leq \int_X f\,d\nu = \varphi(f). \end{equation*}
    Since \(\varphi\) is inner regular on \(U\text{,}\) it follows that
    🔗
    \begin{equation*} \nu(U) = \sup\{\varphi(f) \colon f \prec U\}. \end{equation*}
    Hence \(\nu = \mu\) on all open subsets of \(X\text{.}\) As \(\nu\) and \(\mu\) are outer regular on all Borel sets, it follows that \(\nu = \mu\text{.}\)
    🔗

    We have shown that every positive linear functional on \(C_c(X)\) is given by integration with respect to a finite Radon measure; and we have shown that every linear functional on \(C_0(X)\) can be written as the difference of two positive linear functionals. We will piece these two things together to give a complete description of the dual space \(C_0(X)\text{.}\)
    🔗
    We first introduce some notation. When \(X\) is a locally compact Hausdorff space, we denote by \(\mathcal{M}(X)\) all finite Radon measure on \(X\text{.}\) In Exercise 2.5.15 it is shown that \(\mathcal{M}(X)\) is a vector space. We put a norm on \(\mathcal{M}(X)\) by setting, for \(\nu \in \mathcal{M}(X)\text{,}\)
    🔗
    \begin{equation*} \|\nu\|_\mathrm{var} = |\nu|(X), \end{equation*}
    where \(|\nu|(X)\) is the total variation of \(\nu\text{.}\) That \(\|\cdot\|_\mathrm{var}\) defines is also left to Exercise 2.5.15.

    Theorem 2.4.12. Riesz-Markov-Kakutani Theorem.

    Let \(X\) be a locally compact Hausdorff space and let \(\mathcal{M}(X)\) be the space of finite Radon measures on \(X\text{.}\) If \(\varphi \in C_0(X)^*\text{,}\) then there is a \(\nu \in \mathcal{M}(X)\) satisfying
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\nu \end{equation*}
    for all \(f \in C_0(X)\) and \(\|\nu\|_\mathrm{var} = \|\varphi\|_\mathrm{op}.\)
    🔗
    🔗

    Proof.

    Take \(\varphi \in C_0(X)^*\text{.}\) By Theorem 2.4.8, there are positive linear functionals \(\phi_+\) and \(\phi_-\) such that \(\varphi = \phi_+ - \phi_-\) and \(\|\varphi\|_\mathrm{op} = \|\phi_+\|_\mathrm{op}+ \|\phi_-\|_\mathrm{op}\text{.}\) Both \(\phi_+\) and \(\phi_-\) restrict to positive linear functionals on \(C_c(X)\text{.}\) Thus, by Theorem 2.4.11, there are finite Radon measures \(\mu_1\) and \(\mu_2\) on \(X\) such that, for \(f \in C_c(X)\text{,}\)
    🔗
    \begin{equation*} \phi_+(f) = \int_X f\,d\mu_1 \text{ and }\phi_-(f) = \int_X f\,d\mu_2. \end{equation*}
    Further, \(\|\phi_+\|_\mathrm{op} = \|\mu_1\|_\mathrm{var}\) and \(\|\phi_-\|_\mathrm{op} = \|\mu_2\|_\mathrm{var}\text{.}\) As \(C_c(X)\) is dense in \(C_0(X)\text{,}\) we have
    🔗
    \begin{equation*} \phi_+(f) = \int_X f\,d\mu_1 \text{ and }\phi_-(f) = \int_X f\,d\mu_2 \end{equation*}
    for all \(f \in C_0(X)\text{.}\)
    🔗
    Let \(\mu = \mu_1 - \mu_2\text{.}\) Then \(\mu \in \mathcal{M}(X)\text{.}\) Let \(\mu = \mu_+ - \mu_-\) be the Jordan decomposition of \(\mu\text{.}\) Then, \(\mu_+ + \mu_2 = \mu_- + \mu_1\text{,}\) and so, by Exercise II.7.5.1,
    🔗
    \begin{equation*} \int_X f \,d\mu_+ + \int_X f \,d\mu_2 = \int_X f \,d\mu_- + \int_X f \,d\mu_1, \end{equation*}
    for all \(f \in C_0(X)\text{.}\) Rearranging the above equality, we get
    🔗
    \begin{equation*} \int_X f\,d\mu = \int_X f\,d\mu_1 - \int_X f\,d\mu_2 = \varphi(f), \end{equation*}
    for all \(f \in C_0(X)\text{.}\) By Example 2.4.4,
    🔗
    \begin{equation*} \mu_1(X) + \mu_2(X) = \|\varphi\|_\mathrm{op} \leq |\mu|(X). \end{equation*}
    Since \(\mu = \mu_1 - \mu_2\text{,}\) by Proposition II.4.1.14 we also have \(|\mu|(X) \leq \mu_1(X)+ \mu_2(X)\text{,}\) and so \(|\mu|(X) = \mu_1(X)+ \mu_2(X)\text{.}\) Therefore \(\|\varphi\|_\mathrm{op} = |\mu|(X)\text{.}\)
    🔗
    We will now prove uniqueness. Suppose \(\nu \in \mathcal{M}(X)\) also satisfies \(\varphi(x) = \int_X f\,d\nu\) for all \(f\in C_0(X)\text{.}\) Let \(\nu = \nu_+ - \nu_-\) be the Jordan decomposition of \(\nu\text{.}\) Then, for \(f \in C_0(X)\text{,}\) two applications of Exercise II.7.5.1 give
    🔗
    \begin{equation*} \begin{aligned} \int_X f\,d(\nu_+ + \mu_-) &= \int_X f\,d\nu_+ + \int_X f\,d\mu_-\\ &=\int_X f\,d\nu_- + \int_X f\,d\mu_+\\ &=\int_X f\,d(\nu_- + \mu_+). \end{aligned} \end{equation*}
    By the uniqueness in Theorem 2.4.11, \(\nu_+ + \mu_- = \nu_- + \mu_+\text{,}\) and hence \(\nu = \mu\text{.}\)

    Corollary 2.4.13.

    Let \(X\) be a locally compact Hausdorff space, and let \(\mathcal{M}(X)\) be the finite Radon signed measures on \(X\text{.}\) Let \(\Phi\) be the map
    🔗
    \begin{equation*} \Phi \colon \mathcal{M}(X) \rightarrow C_0(X)^*, \end{equation*}
    defined by
    🔗
    \begin{equation*} \Phi(\nu)(f) = \int_X f\,d\nu \end{equation*}
    for all \(f \in C_0(X)\text{.}\) Then \(\Phi\) is an isometric isomorphism.

    We can now prove that the Jordan decomposition of a linear functional on \(C_0(X)\) is unique, using the uniqueness of the Jordan decomposition for signed measures.

    Corollary 2.4.14.

    Let \(X\) be a locally compact Hausdorff space, and let \(\varphi\) be a linear functional on \(C_0(X)\text{.}\) If \(\phi_1,\phi_2\) are positive linear functionals on \(C_0(X)\) such that \(\varphi = \phi_1 - \phi_2\) and \(\|\varphi\|_\mathrm{op} = \|\phi_1\|_\mathrm{op} + \|\phi_2\|_\mathrm{op}\text{,}\) then \(\phi_1 = \phi_+\) and \(\phi_2 = \phi_-\) where \(\varphi = \phi_+ - \phi_-\) is the Jordan decomposition of \(\varphi\text{.}\)
    🔗
    🔗

    Proof.

    Let \(\phi_1,\phi_2\) be positive linear functionals on \(C_0(X)\) such that \(\varphi = \phi_1 - \phi_2\) and \(\|\varphi\|_\mathrm{op} = \|\phi_1\|_\mathrm{op} + \|\phi_2\|_\mathrm{op}\text{.}\) By Theorem 2.4.12, there is a finite Radon signed measure \(\mu\text{,}\) and finite Radon measures \(\mu_1\) and \(\mu_2\) such that \(|\mu|(X) = \|\varphi\|_\mathrm{op}\text{,}\) \(\mu_1(X) = \|\phi_1\|_\mathrm{op}\text{,}\) and \(\mu_2(X) = \|\phi_2\|_\mathrm{op}\text{;}\) and for all \(f \in C_0(X)\)
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu,\ \phi_1(f) = \int_X f\,d\mu_1,\ \text{ and }\phi_2(f) = \int_X f\,d\mu_2. \end{equation*}
    Thus, for all \(f \in C_0(X)\text{,}\)
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu = \int_X f\,d\mu_1 - \int_X f\,d\mu_2 = \int_X f\,d(\mu_1-\mu_2). \end{equation*}
    Hence, by the uniqueness of \(\mu\) from Theorem 2.4.12, \(\mu = \mu_1 - \mu_2\text{.}\) We also have that
    🔗
    \begin{equation*} |\mu|(X) = \|\phi_1\|_\mathrm{op} + \|\phi_2\|_\mathrm{op} = \mu_1(X) + \mu_2(X). \end{equation*}
    Therefore, by Proposition II.4.1.14, \(\mu_1 - \mu_2\) is the Jordan decomposition of \(\mu\text{.}\) That is \(\mu_1 = \mu_+\) and \(\mu_2 = \mu_-\text{.}\) As the Jordan decomposition of the signed measure \(\mu\) is unique, by Theorem II.4.1.9, it follows that \(\phi_1 = \phi_+\) and \(\phi_2 = \phi_-\) where \(\varphi = \phi_+ - \phi_-\) is the Jordan decomposition of \(\varphi\text{.}\)

    Example 2.4.15.

    The set of natural numbers \(\mathbb{N}\) with the discrete topology is a locally compact Hausdorff space. If \(c_0\) is the collection of all sequences which converge to \(0\text{,}\) then \(C_0(\mathbb{N}) = c_0\text{.}\) By Exercise 2.5.6, the dual of \(c_0\) is \(\ell^1\text{;}\) and by Corollary 2.4.13 the dual of \(c_0 = C_0(\mathbb{N})\) is \(\mathcal{M}(\mathbb{N})\text{,}\) the space of all finite Radon signed measures on \(\mathbb{N}\text{.}\) Thus \(\ell^1\) is isometrically isomorphic \(\mathcal{M}(\mathbb{N})\text{.}\)
    🔗
    Explicitly, the maps between \(\ell^1\) and \(\mathcal{M}(\mathbb{N})\) are given as follows: for \(a = (a_n)_n \in \ell^1\text{,}\) the function \(\nu\) on subsets of \(\mathbb{N}\) defined by
    🔗
    \begin{equation*} \nu_a(E) = \sum_{n \in E}a_n \end{equation*}
    defines a finite Radon signed measure on \(\mathbb{N}\text{.}\) For \(\nu \in \mathcal{M}(\mathbb{N})\text{,}\) we can define a sequence \(a_\nu = (\nu\{n\})_n\text{.}\) Then \(a_\nu \in \ell^1\text{.}\) These two operations are clearly inverses of eachother.

    Example 2.4.16.

    Again, consider the set of natural numbers \(\mathbb{N}\) with the discrete topology. There is a compact Hausdorff space \(\beta\mathbb{N}\) containing \(\mathbb{N}\) as a dense subset such that
    🔗
    \begin{equation*} C_b(\mathbb{N}) = C(\beta\mathbb{N}). \end{equation*}
    The space \(\beta \mathbb{N}\) is called the Stone-\u{C}ech compactification of \(\mathbb{N}\text{.}\) Note that \(\ell^\infty = C_b(\mathbb{N})\text{.}\) Thus, by Corollary 2.4.13,
    🔗
    \begin{equation*} (\ell^\infty)^* \simeq \mathcal{M}(\beta \mathbb{N}). \end{equation*}
    That is, the dual of \(\ell^\infty\) is (isometrically isomorphic to) the Radon signed measures on the Stone-\u{C}ech compactification of \(\mathbb{N}\text{.}\)

    Remark 2.4.17.

    Let \(X\) be a locally compact Hausdorff space. Take \(\varphi \in C_0(X)^*\text{,}\) and let \(\varphi = \phi_+ - \phi_-\) be the Jordan decomposition of \(\varphi\text{.}\) Let \(\mu\) be the finite Radon signed measure such that
    🔗
    \begin{equation*} \varphi(f) = \int_X f\,d\mu, \end{equation*}
    from Theorem 2.4.12. Let \(\mu = \mu_+ - \mu_-\) be the Jordan decomposition of \(\mu\text{.}\) In Corollary 2.4.14 it is shown that \(\mu_+\) and \(\mu_-\) are the unique measures such that
    🔗
    \begin{equation*} \phi_+(f) = \int_X f \,d\mu_+\text{ and }\phi_-(f) = \int_X f \,d\mu_-. \end{equation*}
    In other words, the Jordan decomposition for linear functionals on \(C_0(X)\) gives the Jordan decomposition for finite Radon signed measures on \(X\) (via the Riesz-Markov-Kakutani Theorem). In this way, the Jordan decomposition for linear functionals can be seen as a special case of the Jordan decomposition for signed measures.

    Remark 2.4.18.

    Let \([a,b]\subseteq\mathbb{R}\) be a closed bounded interval on \(\mathbb{R}\text{,}\) and let \(g\) be a right-continuous function of bounded variation on \([a,b]\text{.}\) By Theorem II.4.2.14 there is a unique Borel signed measure \(\nu\) on \([a,b]\) such that \(g(x) = \nu([a,x])\text{.}\) Let \(\varphi\) be the linear functional on \(C([a,b])^*\) defined by \(\varphi(f) = \int_X f\,d\nu\text{.}\)
    🔗
    Now let \(g = g_+ - g_-\text{,}\) \(\nu = \nu_+ - \nu_-\text{,}\) and \(\varphi = \phi_+ - \phi_-\) be the Jordan decompositions of \(g\text{,}\) \(\nu\text{,}\) and \(\varphi\) respectively. Then, by Proposition II.4.2.15,
    🔗
    \begin{equation*} g_+(x) = \nu_+([a,x]) \text{ and }g_-(x) = \nu_-([a,x]); \end{equation*}
    and, as discussed in Remark 2.4.17,
    🔗
    \begin{equation*} \phi_+(f) = \int_X f\,d\nu_+ \text{ and }\phi_-(f) = \int_X f\,d\nu_-. \end{equation*}
    Thus, the Jordan decomposition for linear functionals on \(C([a,b])\) recovers the Jordan decomposition for right-continuous functions of bounded variation on \([a,b]\text{.}\)

    Subsection Riesz’s result and Riemann-Stieltjes integration

    Riesz gave a description of the dual of \(C([a,b])\text{,}\) for a closed bounded interval \([a,b]\subseteq \mathbb{R}\text{.}\) Riesz described the linear functionals on \([a,b]\) not in terms of integration with respect to Borel measures on \([a,b]\text{,}\) but in terms of Reimann-Stieltjes integrals.
    🔗
    Let \(g\) be an increasing on the closed interval \([a,b]\text{,}\) and let \(f\) be a continuous function on \([a,b]\text{,}\) and let \(f\) be a continuous function on \([a,b]\text{.}\) Let \(P = \{x_0,\ldots,x_n\}\) be a partition of \([a,b]\) with
    🔗
    \begin{equation*} a = x_0 <x_1 < \ldots< x_n =b. \end{equation*}
    We define the lower Darboux-Stieltjes sum by
    🔗
    \begin{equation*} L(f,g,P) = \sum_{k=1}^n \inf_{x\in[x_{k-1},x_k]}f(x)(g(x_k)-g(x_{k-1})), \end{equation*}
    and the upper Darboux-Stieltjes sum by
    🔗
    \begin{equation*} U(f,g,P) = \sum_{k=1}^n \sup_{x\in[x_{k-1},x_k]}f(x)(g(x_k)-g(x_{k-1})). \end{equation*}
    The Riemann-Stieltjes integral of \(f\) with respect to \(g\), \(\int_a^b f\,dg\text{,}\) is defined as
    🔗
    \begin{equation*} \int_a^b f\,dg = \sup_P L(f,g,P) = \inf_P U(f,g,P). \end{equation*}
    Note that, if \(g(x)= x\text{,}\) then the Riemann-Stieltjes integral is the same as the Riemann integral. For arbitrary increasing \(g\text{,}\) the theory of Riemann-Stieltjes integrals aligns closely with that of Riemann integrals.
    🔗
    More generally, if \(g\) is a function of bounded variation, we can define the Riemann-Stieltjes integral \(\int_a^b f\,dg\text{,}\) either by considering sums of the form
    🔗
    \begin{equation*} \sum_{k=1}^n f(x_k^*)(g(x_k)-g(x_{k-1})), \end{equation*}
    for partitions \(\{x_0,\ldots,x_n\}\) of \([a,b]\text{,}\) and \(x_k^* \in [x_{k-1},x_k]\text{;}\) or by defining
    🔗
    \begin{equation*} \int_a^b f\,dg = \int_a^b f\,dg_+ - \int_a^b f\,dg_-, \end{equation*}
    where \(g= g_+ - g_-\) is the Jordan decomposition of \(g\text{.}\)
    🔗
    Let \(\varphi\) be a linear functional on \(C([a,b])\text{.}\) Riesz constructs a function of bounded variation \(g\) using \(\varphi\text{.}\) One way to do this, as done in [8], is to extend the domain of \(\varphi\) to contain bounded point-wise limits of increasing continuous functions. The function \(g\) is then defined by
    🔗
    \begin{equation*} g(x) = \varphi(\chi_{[0,x]}). \end{equation*}
    It is then shown that that
    🔗
    \begin{equation*} \varphi(f) = \int_a^b f\,dg, \end{equation*}
    for all \(f \in C([a,b])\text{.}\)
    🔗
    One can show that the function \(g\) is right-continuous. Thus, by Theorem II.4.2.14, there is a Borel signed measure \(\nu\) on \([a,b]\) such that \(\nu([a,x]) = f(x)\) for all \(x \in [a,b]\text{.}\) It can be shown that, if \(f\) is a continuous function on \([a,b]\) then
    🔗
    \begin{equation*} \int_a^bf\,dg = \int_{[a,b]} f\,d\nu. \end{equation*}
    Thus
    🔗
    \begin{equation*} \varphi(f) = \int_{[a,b]} f\,d\nu, \end{equation*}
    and we recover the measure theoretic description of the dual of \(C([a,b])\text{.}\)
