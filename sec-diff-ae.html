    Section A.2 Differentiable almost everywhere functions

    The examples presented in this section illustrate further the stark difference between continuity and differentiability. Further, they show the strength of the Lebesgue integral over the Riemann integral. The examples below are all continuous functions \(f \colon [0,1] \longrightarrow \mathbb{R}\) with \(f'\) defined (almost) everywhere, but \(f'\) is not Riemann integrable. However, in these examples \(f'\) will be Lebesgue integrable.
    🔗

    Subsection Schwarz’s example

    Schwarz published the following example in 1873 [13].
    🔗
    Schwarz presented this function in order to show that it is possible for a continuous function \(f\) to be differentiable on some points, but for \(f'\) to be undefined on a dense set. Thus, differentiability is not a local property. It was perhaps the first such example. This example served to dispute earlier conjectures on the nature of differentiability. Explicitly, Schwarz gives exhibits a continuous increasing function on \([0,1]\) which is not differentiable on any dyadic rational, i.e. \(f\) is not differentiable on any rational of the form \(m/2^k\text{,}\) where \(m,k\geq 0\) are integers.
    🔗
    Before we give the example, we first introduce a continuous function \(\varphi\) on \(\mathbb{R}\text{.}\) Define \(\varphi\) by
    🔗
    \begin{equation*} \varphi(x) = \lfloor x \rfloor + \sqrt{x - \lfloor x \rfloor} \end{equation*}
    where \(\lfloor x \rfloor\) is the floor function (i.e. \(x\) rounded down to the nearest integer).
    🔗
    The function \(\varphi\) is continuous on all of \(\mathbb{R}\text{,}\) and is differentiable on all non-integer values. Indeed, it is not hard to show for a \(x \notin \mathbb{Z}\)
    🔗
    \begin{equation*} \varphi'(x) = \frac{1}{2\sqrt{x-\lfloor x\rfloor}}. \end{equation*}
    If \(x\in \mathbb{Z}\) then \(\varphi(x) = x\text{,}\) and so, if \(0<h<1\) then
    🔗
    \begin{equation*} \frac{\varphi(x+h)-\varphi(x)}{h}=\frac{\sqrt{h}}{h}=\frac{1}{\sqrt{h}}. \end{equation*}
    Thus, for \(x \in \mathbb{Z}\text{,}\)
    🔗
    \begin{equation*} \lim_{h\rightarrow0^+}\frac{\varphi(x+h)-\varphi(x)}{h}=\infty. \end{equation*}
    On the other hand, for \(x\in\mathbb{Z}\) and \(0<h<1\)
    🔗
    \begin{equation*} \frac{\varphi(x-h)-\varphi(x)}{-h}=\frac{-1 +\sqrt{1-h}}{-h} \end{equation*}
    and so, by an application of L’H\^{o}pital’s rule,
    🔗
    \begin{equation*} \lim_{h\rightarrow 0^+}\frac{\varphi(x-h) - \varphi(x)}{-h}=\frac{1}{2}. \end{equation*}
    Thus \(\varphi\) is not differentiable on \(\mathbb{Z}\text{:}\) the right-sided derivative is \(\infty\) and the left-sided derivative is \(\frac{1}{2}\) on \(\mathbb{Z}\text{.}\)
    🔗
    We now define the function \(f\colon [0,1]\rightarrow\mathbb{R}\) by
    🔗
    \begin{equation*} f(x) = \sum_{n=0}^\infty \frac{\varphi(2^nx)}{4^n}. \end{equation*}
    Note that \(f\) is uniformly convergent on all compact subsets of \(\mathbb{R}\text{,}\) and so \(f\) is continuous. This can be shown, e.g., using the inequality \(\varphi(x) \leq x + 1\)
     1 
    A better, indeed the best, linear upper bound for \(\varphi\) is \(\varphi(x) \leq x + \frac{1}{4}\text{,}\) but there is no need for precision here.
    together with the Weierstrass \(M\)-test.
    🔗
    We will now study \(f'\text{.}\) The basic idea is that, since \(\varphi\) is not differentiable on the integers, \(f\) is not differentiable on dyadic rationals.
    🔗
    Suppose \(x_0 = m/2^k\) for some non-negative integers \(m\) and \(k\text{.}\) If \(0<h<1/2^k\text{,}\) then
    🔗
    \begin{equation*} \begin{aligned} \frac{f(x_0+h) - f(x_0)}{h} &= \sum_{n=1}^\infty\frac{\varphi(2^nx_0+2^nh)-\varphi(2^nx_0)}{4^nh}\\ &> \frac{\varphi(2^kx_0+2^kh)-\varphi(2^kx_0)}{4^kh}\\ &=\frac{\sqrt{2^kh}}{4^kh}\\ &= \frac{1}{(2\sqrt{2})^{k} \sqrt{h}} \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation*} \lim_{h\rightarrow 0^+} \frac{f(x_0+h) - f(x_0)}{h} = \infty. \end{equation*}
    Taking \(x_0 = m/2^k\) still, we now consider the left sided derivative. Consider \((f(x_0-h)-f(x_0))/(-h)\text{.}\) For \(n\geq k\text{,}\) \(2^nx_0\) is an integer. Note, for any integer \(y\) and \(h>0\text{,}\) if \(\lfloor y-h\rfloor = y -p\text{,}\) then \(0\leq p-h < 1\) and so
    🔗
    \begin{equation*} \frac{\varphi(y-h) - \varphi(y)}{-h}=\frac{p -\sqrt{p-h}}{h}\leq 1. \end{equation*}
    Thus, for \(0<h<1\)
    🔗
    \begin{equation*} \begin{aligned} \frac{f(x_0-h) - f(x_0)}{-h} &= \sum_{n=0}^\infty\frac{\varphi(2^nx_0 - 2^nh)-\varphi(2^nx_0)}{2^n(-2^nh)}\\ &\leq \sum_{n=0}^{k-1}\frac{\varphi(2^nx_0 - 2^nh)-\varphi(2^nx_0)}{2^n(-2^nh)} + \sum_{n=k}^\infty\frac{1}{2^n}. \end{aligned} \end{equation*}
    By the Weierstrass \(M\)-test we see that
    🔗
    \begin{equation*} F(h) = \frac{f(x_0-h) - f(x_0)}{-h} \end{equation*}
    is uniformly convergent for \(h \in (0,1).\) Take the limit as \(h \rightarrow 0^+\) we get a left-sided derivative as
    🔗
    \begin{equation*} \lim_{h\rightarrow 0^+} \frac{f(x_0-h) - f(x_0)}{-h}= \frac{1}{2^k} + \sum_{n=0}^{k-1}\frac{1}{2^n}\varphi'(2^nx_0). \end{equation*}
    The above set shows that there is a dense set on which \(f\) is not differentiable. Since \(f\) is increasing and continuous, \(f'\) is defined almost everywhere, by Corollary II.7.3.12. Alternatively, a similar analysis to the above can explicitly show that \(f'(x)\) exists for \(x \in (0,1)\) when \(x\) is not a dyadic rational.
    🔗
    🔗

    Subsection A function that is not differentiable on a prescribed set

    Corollary II.7.3.12 tells us that if \(f\) is continuous of bounded variation then \(f'\) is defined almost everywhere. We will now show that given \(E \subseteq \mathbb{R}\) with \(m(E) = 0\text{,}\) there is a continuous function \(f\) of bounded variation which is not differentiable at any point in \(E\text{,}\) but differentiable at all other points. I believe this construction is due to Riesz and Sz.-Nagy; it can be found in their book [8].
    🔗
    Let \([a,b]\subseteq \mathbb{R}\) be a closed interval, and let \(E \subseteq [a,b]\) be a set with Lebesgue measure \(0\text{.}\) By EXERCISE, there is a countable family of open intervals \(\{(c_k,d_k)\}_{k=1}^\infty\) in \(\mathbb{R}\) such that, for each \(x \in E\) there is an infinite subsequence \((k_n)\) of natural numbers such that \(x \in (c_{k_n},d_{k_n})\) for each \(k_n\) and
    🔗
    \begin{equation*} \sum_{k_n} (d_{k_n} - c_{k_n}) <\infty. \end{equation*}
    Fix such a collection \(\{(c_k,d_k)\}_k\) and define a function \(f\) on \([a,b]\) by
    🔗
    \begin{equation*} f(x) = \sum_{k=1}^\infty \ell((c_k,d_k) \cap (-\infty, x)), \end{equation*}
    where \(\ell\) is the length function on intervals in \(\mathbb{R}\text{.}\)
