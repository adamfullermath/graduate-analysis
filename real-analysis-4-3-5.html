    Section 2.2 Compactness

    Definition 2.2.1.

    Let \((X,\tau)\) be a topological space and let \(A \subseteq X\text{.}\) An open cover of \(A\) is a collection of open sets \(\mathcal{F} \subseteq \tau\) such that
    🔗
    \begin{equation*} A \subseteq \bigcup_{F\in\mathcal{F}} F. \end{equation*}
    An open cover \(\mathcal{F}\) of \(A\) has a finite subcover of \(A\) if there is a finite subset \(\{F_1,\ F_2,\ldots,F_n\} \subseteq \mathcal{F}\) such that
    🔗
    \begin{equation*} A \subseteq F_1 \cup F_2 \cup \ldots \cup F_n. \end{equation*}
    A set \(A \subseteq X\) is called compact given any open cover \(\mathcal{F}\) of \(A\text{,}\) \(\mathcal{F}\) has a finite subcover of \(A\text{.}\)
    🔗
    🔗
    The following theorem gives an easy characterization of compact sets in \(\mathbb{R}\text{.}\)
    🔗

    Theorem 2.2.2. Heine-Borel Theorem.

    A subset \(A \subseteq \mathbb{R}\) is compact if and only if \(A\) is closed and bounded.
    🔗
    🔗

    Proof.

    We will first show that a closed bounded interval \([a,b]\) is compact. If \(a=b\text{,}\) then \([a,b]\) is finite and clearly compact. Assume \(a<b\text{.}\) Let \(\mathcal{F}\) be an arbitrary open cover of \([a,b]\text{.}\) Note that, for any \(a\leq x\leq b\text{,}\) \(\mathcal{F}\) is also an open cover of \([a,x]\text{.}\) Let
    🔗
    \begin{equation*} E = \{x\in [a,b] \colon \mathcal{F} \text{ has a finite subcover of }[a,x]\}. \end{equation*}
    Note that \([a,a]= \{a\}\text{.}\) As \(\mathcal{F}\) is an open cover of \([a,b]\) there is an open set \(F \in \mathcal{F}\) such that \(a\in F\text{.}\) Hence \(a\in E\text{.}\)
    🔗
    As \(E\) is a non-empty, bounded subset of \(\mathbb{R}\) we can take its supremum. Let
    🔗
    \begin{equation*} c = \sup E. \end{equation*}
    Clearly, \(a< c\leq b\text{.}\) If we can show that \(c=b\) then we are done.
    🔗
    Now, since \(c\in [a,b]\text{,}\) there is an open set \(G \in \mathcal{F}\) such that \(c\in G\text{.}\) As \(G\) is open, there exists \(r>0\) such that \((c-r,c+r)\subseteq G\text{.}\) As \(c-r\) is not an upper bounded for \(E\text{,}\) there exists \(x\in E\) such that \(c-r<x\leq c\text{.}\) By definition of \(E\text{,}\) there are open sets \(F_1, \ldots, F_n \in \mathcal{F}\) such that
    🔗
    \begin{equation*} [a,x] \subseteq F_1 \cup F_2 \cup \ldots \cup F_n. \end{equation*}
    Thus
    🔗
    \begin{equation*} [a, c+r) \subseteq F_1 \cup F_2 \cup \ldots \cup G. \end{equation*}
    It follows that \(c=b\text{,}\) otherwise we could find \(x\in [a,b]\) with
    🔗
    \begin{equation*} a\leq c< x < c+r \end{equation*}
    and have \([a,x]\) finitely covered by \(F_1,\ldots,F_n,G\text{;}\) contradicting the definition of \(c\) as the supremum of \(E\text{.}\) We have now shown that \([a,b]\) is compact.
    🔗
    Let \(A\) be an arbitrary closed bounded subset of \(\mathbb{R}\) and let \(\mathcal{F}\) be any open cover of \(A\text{.}\) As \(A\) is bounded there is a closed interval \([a,b]\) such that \(A \subseteq [a,b]\text{.}\)
    🔗
    Let \(A^c = \mathbb{R}\backslash A\) be the complement of \(A\text{.}\) Since \(A\) is closed, \(A^c\) is open. Let \(\mathcal{F}' = \mathcal{F} \cup \{A^c\}\text{.}\) Then \(\mathcal{F}'\) is an open cover of \([a,b]\text{.}\) Since \([a,b]\) is compact, \(\mathcal{F}'\) has a finite subcover of \([a,b]\text{.}\) That is
    🔗
    \begin{equation*} A \subseteq [a,b] \subseteq F_1 \cup \ldots \cup F_n, \end{equation*}
    for some \(F_1,\ldots,F_n \in \mathcal{F}'\text{.}\) If \(F_1,\ldots, F_n \in \mathcal{F}\text{,}\) then we have a finite subcover of \(A\) from open sets in \(\mathcal{F}\text{,}\) and we are done. Otherwise one of \(F_1,\ldots,F_n\) is equal to \(A^c\text{.}\) Without loss of generality, say \(F_1 = A^c\text{.}\) Then we have
    🔗
    \begin{equation*} A \subseteq F_2 \cup \ldots \cup F_n \end{equation*}
    and so \(F_2, \ldots, F_n\) form a finite open subcover of \(A\) from \(\mathcal{F}\text{.}\) Thus \(A\) is compact.
    🔗
    To prove the converse statement, that a set \(A\) is closed and bounded if it is compact, it suffices to prove the following two results, which we leave as an exercise:
    🔗
    1. if \(A\) is unbounded then \(A\) is not compact;
      🔗
      🔗
    2. if \(A\) is not closed, then \(A\) is not compact.
      🔗
      🔗
    🔗

    Definition 2.2.3.

    A set \(A \subseteq \mathbb{R}\) is called sequentially compact if every infinite sequence of elements of \(A\) has a subsequence which converges to an element of \(A\text{.}\)
    🔗
    🔗
    The Bolzano-Weierstrass theorem, together with Proposition Proposition 2.1.6, tells us that a set \(A\subseteq \mathbb{R}\) is sequentially compact if and only if it is closed and bounded. Combining the Bolzano-Weierstrass Theorem and the Heine-Borel Theorem we get the following result.
    🔗

    Corollary 2.2.4.

    Let \(A\subseteq \mathbb{R}\text{.}\) Then the following are equivalent:
    🔗
    1. \(A\) is compact;
      🔗
      🔗
    2. \(A\) is sequentially compact;
      🔗
      🔗
    3. \(A\) is closed and bounded.
      🔗
      🔗
    🔗
    Many statements which are true for continous real-valued functions on closed bounded intervals, are in fact true for real-valued functions on closed bounded sets, using similar proofs (which rely on compactness/sequential compactness). We record some of these facts below:
    🔗

    Proposition 2.2.5.

    Let \(E\subseteq \mathbb{R}\) be a closed bounded set. Let \(f \colon E \rightarrow \mathbb{R}\) be a continuous function. Then
    🔗
    1. \(f\) is uniformly continuous;
      🔗
      🔗
    2. \(f\) attains its maximum and minimum value.
      🔗
      🔗
    🔗

    Remark 2.2.6.

    The Heine-Borel Theorem and Corollary Corollary 2.2.4 actually hold in \(\mathbb{R}^n\text{:}\) a set \(A\subseteq \mathbb{R}^n\) is compact if and only if it is sequentially compact if and only if it is closed and bounded.
    🔗
    However, in general topology the three conditions need not be the same. A metric space \(X\) is compact if and only if it is sequentially compact. However, there are topological spaces which are not metric spaces, which are sequentially compact but not compact.
    🔗
    If \(X\) is a normed vector space which is complete (i.e. Cauchy sequences are convergent) then the statement: ``\(A \subseteq X\) is compact if and only if \(A\) is closed and bounded" is true if and only if \(X\) is finite dimensional. That is, it is possible to find closed and bounded sets which are not compact.
    🔗
    In this course we will study a large class of complete, normed vector spaces which are infinite dimensional: \(L^p\)-spaces.
