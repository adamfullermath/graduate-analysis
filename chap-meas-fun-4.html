    Section 5.2 Approximations of measurable functions

    Equivalently, there are several ways we can say that a sequence \((f_n)_n\) of functions converges to a function \(f\text{.}\) You will already be familiar with the concept of pointwise convergence of functions and, at least for functions from \(\mathbb{R}\) to \(\mathbb{R}\text{,}\) uniform convergence. We recall these definitions below, together with an unfamiliar form of convergence: convergence pointwise almost everywhere.
    🔗

    Definition 5.2.1.

    Let \((X,\mathcal{F},\mu)\) be a measure space, let \((f_n)_n\) be a sequence of extended real-valued functions on \(X\text{,}\) and let \(f\) be an extended real-valued function on \(x\text{.}\)
    🔗
    The sequence \((f_n)\) converges to \(f\) pointwise almost everywhere
     1 
    often abbreviated to pointwise a.e.
    if there is an \(E \subseteq X\) with \(\mu(E) = 0\text{,}\) such that
    🔗
    \begin{equation*} \lim_{n \rightarrow \infty} f_n(x) = f(x) \end{equation*}
    for all \(x \in X \backslash E\text{.}\)
    🔗
    If
    🔗
    \begin{equation*} \lim_{n \rightarrow \infty} f_n(x) = f(x) \end{equation*}
    for all \(x \in X\) then we say that the sequence \((f_n)_n\) converges pointwise to \(f\text{.}\)
    🔗
    The sequence \((f_n)_n\) converges uniformly if for all \(\varepsilon >0\) there exists an \(N \geq 0\) such that
    🔗
    \begin{equation*} |f_n(x) - f(x)| < \varepsilon \end{equation*}
    for all \(x \in X\) and \(n \geq N\text{.}\)
    🔗
    🔗

    Remark 5.2.2.

    The relationship between these three forms of convergence is:
    🔗
    \begin{equation*} \text{uniform}\implies \text{pointwise} \implies \text{pointwise a.e.}. \end{equation*}
    Neither of the implications go the opposite way, however. Recall that the sequence \((x^n)_n\) on \([0,1]\) converges pointwise to the function
    🔗
    \begin{equation*} f(x) = \begin{cases} 0 & x \in [0,1)\\ 1 & x = 1, \end{cases} \end{equation*}
    but \((x^n)_n\) does not converge uniformly to \(f\text{.}\)
    🔗
    Suppose \(A \subseteq \mathbb{R}\) is a non-empty Lebesgue measurable set with \(m(A) = 0\text{.}\) Define functions
    🔗
    \begin{equation*} f_n(x) = \begin{cases} 0& x \in \mathbb{R} \backslash A, \\ (-1)^n & x \in A.\end{cases} \end{equation*}
    Then \(f_n\) converges pointwise a.e. to the zero function \(0\text{,}\) whilst \(f_n\) does not converge pointwise to any function.
    🔗
    🔗
    Our main goal in this section is two prove two theorems: The Simple Approximation Theorem, Theorem 5.2.4, which says that any measurable function is a pointwise limit of simple functions; and Ergoroff’s Theorem, Theorem 5.2.5, which says that if \((f_n)_n\) converge pointwise to \(f\text{,}\) then there is a large set on which the convergence is uniform.
    🔗
    We first establish that a pointwise limit of measurable functions is measurable.
    🔗

    Proposition 5.2.3.

    Let \((X,\mathcal{F},\mu)\) be a measure space. Suppose \((f_n)_n\) is a sequence of measurable functions on \(X\) that converge pointwise almost everywhere to a function \(f\text{.}\) Then, if either
    🔗
    1. \((X,\mathcal{F},\mu)\) is a complete measure space; or
      🔗
      🔗
    2. the sequence \((f_n)_n\) converges pointwise to \(f\text{,}\)
      🔗
      🔗
    then \(f\) is a measurable function.
    🔗
    🔗

    Proof.

    Let \(E\) be a measurable set with \(\mu(E) = 0\) such that \((f_n(x))_n\) converges to \(f(x)\) for all \(x \in X \backslash E\text{.}\) If \((X,\mathcal{F},\mu)\) is complete, then \(f\) is measurable if and only if \(f|_{X \backslash E}\) is measurable, by Proposition 5.1.6. Thus, by excising \(E\) if necessary, we can assume that \((f_n)_n\) converges pointwise to \(f\text{.}\)
    🔗
    Fix \(c \in \mathbb{R}\) and let
    🔗
    \begin{equation*} A = \{x \in X \colon f(x) > c\}. \end{equation*}
    Our goal is to show that \(A\) is a measurable set. Note that, if \(x_0 \in A\text{,}\) then there exists an \(n, N >0\) such that
    🔗
    \begin{equation*} x_0 \in \bigcap_{k\geq K} \left\{x \in X \colon f_k(x) > c+\frac{1}{n} \right\}. \end{equation*}
    Conversely, suppose \(x_0 \in X\) satisfies
    🔗
    \begin{equation*} x_0 \in \bigcup_{(n,K) \in \mathbb{N}^2}\bigcap_{k\geq K} \left\{x \in X \colon f_k(x) > c+\frac{1}{n} \right\}. \end{equation*}
    Then \(f(x_0) \geq c + \frac{1}{n} > c\text{,}\) and hence \(x_0 \in A\text{.}\) Thus
    🔗
    \begin{equation*} A = \bigcup_{(n,K) \in \mathbb{N}^2}\bigcap_{k\geq K} \left\{x \in X \colon f_k(x) > c+\frac{1}{n} \right\}. \end{equation*}
    Thus, since each \(f_k\) is a measurable function, \(A\) is a countable union of a countable intersection of measurable sets. Thus \(A\) is measurable. As \(c \in \mathbb{R}\) was chosen arbitrarily, \(f\) is a measurable function.
    🔗
    🔗

    Theorem 5.2.4. The Simple Approximation Theorem.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(f\) be an extended real-valued function on \(X\text{.}\) Then \(f\) is measurable if and only \(f\) is the pointwise limit of a sequence of simple functions \((\phi_n)_n\) satisfying \(|\phi_n|\leq |f|\) for all \(n\text{.}\)
    🔗
    If \(f\) is a non-negative function, the sequence \((\phi_n)_n\) can be chosen to be increasing.
    🔗
    If \((X,\mathcal{F},\mu)\) is \(\sigma\)-finite, then the simple functions \((\phi_n)_n\) can each be chosen to be of finite support.
    🔗
    🔗

    Proof.

    If \(f\) is a pointwise limit of simple functions, then \(f\) is measurable by Proposition 5.2.3.
    🔗
    Suppose now that \(f\) is measurable. Assume first that \(f\geq 0\text{.}\) For each \(n\text{,}\) and \(k = 1,\ldots, n^2\text{,}\) let
    🔗
    \begin{equation*} E^n_k = f^{-1}\left(\left[\frac{k-1}{n},\frac{k}{n}\right)\right) \end{equation*}
    For each \(n\text{,}\) the sets \(\{E^n_k\}_{k=1}^{n^2}\) are disjoint measurable sets. Define scalars \((a^n_k)_{n,k}\) by
    🔗
    \begin{equation*} a^n_k = \frac{k-1}{n} \end{equation*}
    For each \(n\) define a simple function \(\phi_n\) by
    🔗
    \begin{equation*} \phi_n(x) = \begin{cases} a_n^k & x \in E_n^k\\ n& f(x) \geq n. \end{cases} \end{equation*}
    Note \(\phi_n\leq f\text{.}\)
    🔗
    We claim that \((\phi_n)_n\) converge pointwise to \(f\text{.}\) First, suppose \(f(x) < \infty\text{.}\) Fix \(\varepsilon >0\) Choose an integer \(N\) such that \(f(x) < N\) and \(\frac{1}{N}< \varepsilon\text{.}\) Then, if \(n \geq N\text{,}\) there is a \(k\) such that \(x \in E^n_k\text{.}\) Thus,
    🔗
    \begin{equation*} |f(x) - \phi_n(x)| = |f(x) - a^n_k| < \frac{1}{n} < \varepsilon. \end{equation*}
    Thus \(\lim_n \phi_n(x) = f(x)\text{.}\)
    🔗
    If \(f(x) = \infty\text{,}\) then \(\phi_n(x) = n\) for all \(n\text{.}\) Thus
    🔗
    \begin{equation*} \lim_n \phi_n(x) = \lim_n n = \infty = f(x). \end{equation*}
    Hence \((\phi_n)_n\) converges pointwise to \(f\text{.}\)
    🔗
    By replacing \(\phi_n\) with \(\max\{\phi_1,\ldots,\phi_n\}\) we can make sure the sequence \((\phi_n)_n\) is increasing.
    🔗
    For an arbitrary measurable function \(f\) we can find a sequences \((\phi_n)_n\) and \((\psi_n)_n\) of simple functions which converge pointwise to \(f_+\) and \(f_-\) respectively, with \(\phi_n \leq f_+\) and \(\psi_n \leq f_-\text{.}\) Then \((\phi_n-\psi_n)_n\) is a sequence of simple functions which converge pointwise to \(f\text{.}\) Further
    🔗
    \begin{equation*} |\phi_n - \psi_n| \leq \phi_n + \psi_n \leq f_+ + f_- = |f|. \end{equation*}
    Finally, suppose that \((X,\mathcal{F},\mu)\) is a \(\sigma\)-finite measure space and \((\phi_n)_n\) is a sequence of simple functions that converge pointwise to a measurable function \(f\text{.}\) Let \(\{E_n\}_{n=1}^\infty\) be an increasing sequence of measurable sets such that \(\mu(E_n) < \infty\) for all \(n\geq 1\text{,}\) and \(X = \bigcup_{n\in\mathbb{N}}E_n\text{.}\) For each \(n\) define the function
    🔗
    \begin{equation*} \psi_n = \phi_n \chi_{E_n}. \end{equation*}
    Then \((\psi_n)_n\) is a sequence of finitely supported simple functions which converge pointwise to \(f\text{.}\)
    🔗
    🔗
    As noted in Remark 5.2.2, pointwise a.e. convergence is weaker than pointwise convergence which is weaker than uniform convergence. Our goal is to prove Ergoroff’s Theorem, which states that, in certain cases, pointwise a.e. convergence implies uniform convergence on a large set.
    🔗

    Theorem 5.2.5. Ergoroff’s Theorem.

    Let \((X,\mathcal{F},\mu)\) be a finite measure space and let \((f_n)_n\) be a sequence of measurable functions on \(X\) which converge pointwise a.e. to a finite a.e. function \(f\text{.}\) Then for all \(\varepsilon >0\) there is a measurable set \(X_\varepsilon \subseteq X\) such that
    🔗
    1. \((f_n)_n\) converges uniformly to \(f\) on \(X_\varepsilon\text{;}\)
      🔗
      🔗
    2. \(\mu(X \backslash X_\varepsilon) < \varepsilon\text{.}\)
      🔗
      🔗
    🔗

    Proof.

    By removing a set of measure \(0\) if necessary, we can assume that \((f_n)_n\) converges pointwise to \(f\) on \(E\text{.}\) By removing another measure \(0\) set, we can assume that \(f\) is real-valued.
    🔗
    Fix \(\eta >0\text{.}\) For each \(k\text{,}\) the function \(|f-f_k|\) is measurable, and hence the set
    🔗
    \begin{equation*} \{x \in X \colon |f(x)-f_k(x)| < \eta\} \end{equation*}
    \ is a measurable set. For each \(n\) let \(E_n\) be the measurable set
    🔗
    \begin{equation*} \begin{aligned} E_n &= \{x \in E \colon |f(x) - f_k(x)|< \eta,\ k\geq n\} \\ &= \bigcap_{k \geq n}\{x \in X \colon |f(x)-f_k(x)| < \eta\}. \end{aligned} \end{equation*}
    Then \(\{E_n\}_n\) is an increasing sequence of measurable sets and \(X = \bigcup_n E_n\text{,}\) since \((f_n)_n\) converges pointwise to \(f\) on \(X\text{.}\) By the continuity of measure,
    🔗
    \begin{equation*} \lim_{n\rightarrow \infty} \mu(E_n) = \mu(X). \end{equation*}
    Thus, for any \(\delta >0\) there is an \(K\) such that
    🔗
    \begin{equation*} \mu(X \backslash E_N) =<fn>Note that is being used here to apply the excision property</fn>\mu(X) - \mu(E_K) < \delta. \end{equation*}
    Note that \(|f_k - f|< \eta\) on \(E_K\) for \(k \geq K\text{.}\)
    🔗
    Fix \(\varepsilon > 0\text{.}\) For each \(n\text{,}\) apply the above argument with \(\eta = \frac{1}{n}\) and \(\delta = \frac{\varepsilon}{2^{n}}\) to find a measurable set \(A_n\) and an index \(K_n\) such that
    🔗
    \begin{equation*} \mu(X \backslash A_n) < \frac{\varepsilon}{2^{n}} \end{equation*}
    and
    🔗
    \begin{equation*} |f_k(x) - f(x)| < \frac{1}{n} \end{equation*}
    for all \(x \in A_n\) and \(k \geq K_n\text{.}\)
    🔗
    Let \(A = \bigcap_n A_n\text{.}\) We will show that \(\mu(X \backslash A) < \varepsilon\) and \((f_n)_n\) converges uniformly to \(f\) on \(A\text{.}\) First note that
    🔗
    \begin{equation*} \begin{aligned} \mu(X\backslash A) &= \mu\left(X \backslash \bigcap_n A_n\right) \\ &= \mu\left(\bigcup_n X \backslash A_n \right) \\ &\leq \sum_{n=1}^\infty \mu(X \backslash A_n)\\ &< \sum_{k=1}^\infty \frac{\varepsilon}{2^{n}}=\varepsilon. \end{aligned} \end{equation*}
    Take any \(\zeta > 0\text{.}\) Choose integer \(n\) such that \(\frac{1}{n}<\zeta\text{.}\) Then, if \(k \geq K_n\text{,}\)
    🔗
    \begin{equation*} |f_k(x) - f(x)|< \frac{1}{n}< \zeta \end{equation*}
    for all \(x \in A_n\text{.}\) As \(A \subseteq A_n\text{,}\) it follows that
    🔗
    \begin{equation*} |f_k(x) - f(x)|< \frac{1}{n}< \zeta \end{equation*}
    for all \(x \in A\) and \(k \geq K_n\text{.}\) Thus \((f_n)_n\) converges uniformly to \(f\) on \(A\) and \(\mu(X\backslash A)< \varepsilon.\)
    🔗
    🔗

    Corollary 5.2.6. Ergoroff’s Theorem on \(\mathbb{R}\).

    Let \(E \subseteq \mathbb{R}\) be a Lebesegue measurable set with \(m(E)< \infty\text{.}\) Let \((f_n)_n\) be a sequence of measurable functions on \(E\) which converge a.e. to a finite a.e. function \(f\text{.}\)
    🔗
    For each \(\varepsilon > 0\) there is a closed set \(F \subseteq E\) such that \(m(E \backslash F)< \varepsilon\) and \((f_n)_n\) converges uniformly to \(f\) on \(F\text{.}\)
    🔗
    🔗

    Proof.

    Fix \(\varepsilon >0\text{.}\) By Theorem 5.2.5 there is a measurable set \(A \subseteq E\) such that \(\mu(E \backslash A)< \varepsilon/2\) and \((f_n)_n\) converges uniformly to \(f\) on \(A\text{.}\) By Theorem 3.1.7, there is a closed set \(F \subseteq A\) such that \(m(A\backslash F)< \varepsilon /2\text{.}\)
    🔗
    As \(F \subseteq A\text{,}\) \((f_n)_n\) converges uniformly to \(f\) on \(F\text{.}\) Also
    🔗
    \begin{equation*} m(E\backslash F) = m(A\backslash F) + m(E\backslash A) < \varepsilon, \end{equation*}
    completing the proof.
