    Section 7.1 The Radon-Nikodym Theorem

    In Exercise 6.4.8 we saw how integration against a non-negative measurable function with respect to a measure \(\mu\) gave gives rise to a new measure \(\nu\text{.}\) In this section we will study measures of that type in detail, cumulating in two famed theorems: the Radon-Nikodym Theorem and the Lebesgue Decomposition Theorem.
    🔗
    We begin by recalling the definition of mutually singular measures, and extending this definition to signed-measures.
    🔗

    Definition 7.1.1.

    Let \((X,\mathcal{F})\) be a measurable space, and let \(\mu\) and \(\nu\) be two signed-measures on \((X,\mathcal{F})\text{.}\) The measures \(\mu\) and \(\nu\) are mutually singular, denoted \(\mu \perp \nu\text{,}\) if there are disjoint measurable sets \(A,B \subseteq X\) with \(A \cup B = X\) such that \(A\) is a null set for \(\mu\text{,}\) and \(B\) is a null set for \(\nu\text{.}\)
    🔗
    🔗

    Example 7.1.2.

    Let \(\varphi\) be the Cantor-Lebesgue function on \([0,1]\text{,}\) let \(C\) be the Cantor set in \([0,1]\) and let \(U = [0,1]\backslash U\text{.}\) Let \(\nu\) be the Borel measure on \([0,1]\) which is the Carathéodory extension of the premeasure
    🔗
    \begin{equation*} \nu((a,b]) = \varphi(b) - \varphi(a), \ \nu(\{0\}) = 0. \end{equation*}
    By Example 3.2.3, \(m(C) = 0\) and \(m(U) =1\text{,}\) and by Exercise 3.4.10, \(\nu(C) = 1\) and \(\nu(U) = 1\text{.}\) Thus \(\nu\) and \(m\) are mutually singular.
    🔗
    🔗
    Mutually singular measures are, in some sense, very different from eachother. We now introduce a comparison between measures.
    🔗

    Definition 7.1.3.

    Let \((X,\mathcal{F},\mu)\) be a measure space and let \(\nu\) be a measure on \((X,\mathcal{F})\text{.}\) Then \(\nu\) is absolutely continuous with respect to \(\mu\), written \(\nu \ll \mu\text{,}\) if for \(E \in \mathcal{F}\text{,}\) \(\nu(E) = 0\) if \(\mu(E) = 0\text{.}\)
    🔗
    If \(\lambda\) is a signed-measure on \((X,\mathcal{F})\text{,}\) then \(\lambda\) is absolutely continuous with respect to \(\mu\), written \(\lambda \ll \mu\text{,}\) if \(|\lambda| \ll \mu\text{.}\)
    🔗
    🔗
    Let \(\mu\) and \(\nu\) be two non-zero measures on the same measure space. If \(\mu \perp \nu\) then \(\mu \not\ll \nu\) and \(\mu \not\ll \nu\text{.}\) Conversely, if \(\mu \ll \nu\) or \(\nu \ll \mu\text{,}\) then \(\mu \not \perp \nu\text{.}\) Mutually orthogonality and absolute continuity are mutually exclusive. We will see, that given measures \(\mu\) and \(\nu\text{,}\) we can always write \(\nu= \nu_s + \nu_a\) where \(\nu_s \perp \mu\) and \(\nu_a \ll \mu\text{,}\) (The Lebesgue Decomposition Theorem, Theorem 7.1.10). Thus, mutual singularity and absolute continuity are two extremes. We first explore absolute continuity further.
    🔗
    Given a measure \(\mu\text{,}\) the next example shows that it is easy to create measures which are absolutely continuous with respect to \(\mu\text{.}\) We will see, with some reasonable conditions on \((X,\mathcal{F},\mu)\text{,}\) all measures which are absolutely continuous with respect to \(\mu\) will be of this form.
    🔗

    Example 7.1.4.

    Let \((X,\mathcal{F},\mu)\) be a measure space. Let \(f\) be a non-negative measurable function on \(X\text{,}\) and let \(g\) be an integrable function on \(X\text{.}\) Then
    🔗
    \begin{equation*} \nu_1(E) = \int_E f\,d\mu \end{equation*}
    defines a measure on \((X,\mathcal{F})\text{,}\) and
    🔗
    \begin{equation*} \nu_2(E) = \int_E g\,d\mu \end{equation*}
    defines a signed measure on \((X,\mathcal{F})\) (Exercise 6.4.8). Then \(\nu_1 \ll \mu\) and \(\nu_2 \ll \mu\text{.}\)
    🔗
    🔗
    We can also get examples of absolutely continuous measures (with respect to the Lebesgue measure) using what we have learnt about Borel measures on \(\mathbb{R}\text{.}\) Below is one such example.
    🔗

    Example 7.1.5.

    Let \(f(x) = x^2\) on \([0,1]\text{.}\) Let \(\nu\) be the Borel measure on \([0,1]\) induced by \(f\text{.}\) That is, \(\nu\) is the Borel measure which is the Carathéodory extension of the premeasure
    🔗
    \begin{equation*} \nu((a,b]) = b^2 - a^2,\ \nu(\{0\}) = 0 \end{equation*}
    as described in Corollary 3.3.10. We will show that \(\nu \ll m\text{.}\)
    🔗
    Take Borel set \(E \subseteq [0,1]\) such that \(m(E) = 0\text{.}\) We want to show that \(\nu(E) = 0\text{.}\) Fix \(\varepsilon >0\text{.}\) We will assume \(0 \notin E\) (removing \(0\) does not change \(m(E)\) or \(\nu(E)\)). By Proposition 3.1.1, there is a countable collection \(\{(a_i,b_i]\}_{i=1}^\infty\) of subsets of \([0,1]\) such that
    🔗
    \begin{equation*} E \subseteq \bigcup_{i=1}^\infty (a_i,b_i] \end{equation*}
    and \(\sum_{i=1}^\infty m((a_i,b_i])<\varepsilon/2.\) Now
    🔗
    \begin{equation*} \begin{aligned} \nu(E) \leq \nu\left(\bigcup_{i=1}^\infty (a_i,b_i]\right) &\leq \sum_{i=1}^\infty \nu((a_i,b_i]) \\ &= \sum_{i=1}^\infty(b_i^2-a_i^2) \\ &= \sum_{i=1}^\infty(b_i-a_i)(b_i + a_i) \\ &\leq 2\sum_{i=1}^\infty(b_i-a_i) \\ &< \varepsilon. \end{aligned} \end{equation*}
    As \(\nu(E) < \varepsilon\) for all \(\varepsilon >0\text{,}\) it follows that \(\nu(E) =0\text{.}\) Thus \(\nu\) is absolutely continuous with respect to \(m\text{.}\)
    🔗
    🔗

    Theorem 7.1.6. Radon-Nikodym Theorem.

    Let \((X,\mathcal{F},\mu)\) be a \(\sigma\)-finite measure space, and let \(\nu\) be a \(\sigma\)-finite measure on \((X,\mathcal{F})\) such that \(\nu \ll \mu\text{.}\) Then there is a non-negative measurable function \(f\) on \(X\) such that
    🔗
    \begin{equation} \nu(E) = \int_E f\,d\mu\tag{7.1} \end{equation}
    for all \(E \in \mathcal{F}\text{.}\) Further, if \(f\) and \(g\) are both non-negative functions satisfying (7.1), then \(f=g\) a.e.
    🔗
    🔗

    Proof.

    First assume that both \(\mu\) and \(\nu\) are finite measures. If \(\nu(E) = 0\) for all \(E \in \mathcal{F}\text{,}\) then we can take \(f = 0\) for (7.1) to hold. Thus suppose that \(\nu\) is not the zero measure.
    🔗
    Let \(\mathcal{A}\) be the collection of non-negative measurable functions \(f\) with
    🔗
    \begin{equation*} \int_E f\,d\mu \leq \nu(E) \end{equation*}
    for all \(E \in \mathcal{F}\text{.}\) Since the zero function \(0\) is in \(\mathcal{A}\text{,}\) \(\mathcal{A}\) is non-empty. Let \(a = \sup\{ \int_X f\,d\mu \colon f \in \mathcal{A}\}\text{.}\) Note that, since \(\nu\) is finite, \(a\) is finite with \(a \leq \nu(E)\text{.}\)
    🔗
    Clearly, \(a \geq 0\text{.}\) We claim that \(a > 0\text{.}\) Consider, for each real \(\alpha > 0\text{,}\) the signed measure \(\sigma_\alpha = \nu - \alpha\mu\text{.}\) For each \(\sigma_\alpha\) let \(X = P_\alpha \cup N_\alpha\) be a Hahn-Decomposition of \(X\) for \(\sigma_\alpha\text{.}\) That is \(P_\alpha\) is a positive set for \(\sigma_\alpha\text{,}\) and \(N_\alpha\) is a negative set for \(\sigma_\alpha\text{.}\) If \(\mu(P_\alpha)= 0\) for all \(\alpha\text{,}\) then
    🔗
    \begin{equation*} \nu(E) \leq \alpha \mu(E) \end{equation*}
    for all \(E \in \mathcal{F}\) and \(\alpha > 0\text{.}\) Hence, if \(\mu(E) >0\) then \(\nu(E) = 0\text{.}\) Since \(\nu \ll \mu\) we also have \(\mu(E) = 0\) implies \(\nu(E) = 0\text{,}\) and so \(\nu\) is the \(0\) measure. This contradicts the assumption that \(\nu\) is non-zero. Hence there is some \(\alpha\) for which \(\mu(P_\alpha)>0\text{.}\) Let
    🔗
    \begin{equation*} \varphi = \alpha \chi_{P_\alpha}. \end{equation*}
    Then, for any \(E \in F\)
    🔗
    \begin{equation*} \int_E \varphi\,d\mu = \alpha \mu(E \cap P_\alpha) \leq \nu(E\cap P_\alpha)\leq \nu(E). \end{equation*}
    Thus, \(\varphi \in \mathcal{F}\text{.}\) As \(\int_X \varphi\,d\mu = \alpha >0\text{,}\) we must have \(a > 0\text{.}\)
    🔗
    We will show that \(a\) is attained by a function \(f \in \mathcal{A}\text{.}\) Let \((f_n)_n\) be a sequence of functions in \(\mathcal{A}\) so that
    🔗
    \begin{equation*} \lim_n \int_X f_n\,d\mu = a. \end{equation*}
    Note also that if \(f,g \in \mathcal{A}\) and \(h = \max\{f,g\}\) then \(h \in \mathcal{A}\text{.}\) Indeed, if \(E \in F\) and \(F = \{x \in E \colon f(x) > g(x)\}\) then
    🔗
    \begin{equation*} \begin{aligned} \int_E h\,d\mu &= \int_F f \,d\mu + \int_{E\backslash F}g\,d\mu \\ &\leq \nu(F) + \nu(E\backslash F) = \nu(E). \end{aligned} \end{equation*}
    We can thus assume that the sequence \((f_n)_n\) is increasing, by replacing each \(f_n\) with \(\max\{f_1,\ldots,f_n\}\) if necessary. Let \(f(x) = \lim_n f_n(x)\) for all \(x \in X\text{.}\) By the Monotone Convergence Theorem
    🔗
    \begin{equation*} \int_E f\,d\mu = \lim_n\int_E f_n\,d\mu \leq \nu(E) \end{equation*}
    \ for all \(E \in \mathcal{F}\text{.}\) Thus \(f \in \mathcal{A}\text{.}\) Setting \(E = X\) we have
    🔗
    \begin{equation*} \int_X fd\mu = \lim_n \int_X f_n\,d\mu = a. \end{equation*}
    Our goal now is to show that \(f\) satisfies (7.1). That is, we will show that for all \(E \in \mathcal{F}\)
    🔗
    \begin{equation*} \nu(E) = \int_X f\,d\mu. \end{equation*}
    Define a measure \(\lambda\) on \((X,\mathcal{F})\) by
    🔗
    \begin{equation*} \lambda(E) = \nu(E) - \int_E f\,d\mu . \end{equation*}
    (Note that \(\lambda\) is indeed a measure, and not just a signed-measure, since \(f \in \mathcal{A}\)).
    🔗
    Since \(\nu\) is absolutely continuous with respect to \(\mu\text{,}\) and \(E \mapsto \int_E f\,d\mu\) is absolutely continuous with respect to \(\mu\text{,}\) \(\lambda\) is absolutely continuous with respect to \(\mu\text{.}\) Applying the above argument to \(\lambda\) (in place of \(\nu\)), if \(\lambda\) is a non-zero measure, then there is a non-negative measurable function \(g\) such that \(\int_X g \,d\mu > 0\) and
    🔗
    \begin{equation*} \int_E g\,d\mu \leq \lambda(E) \end{equation*}
    for all \(E \in \mathcal{F}\text{.}\)
    🔗
    Taking any \(E \in F\) we have
    🔗
    \begin{equation*} \begin{aligned} \int_E g\,d\mu \leq \lambda(E) = \nu(E) - \int_E f\,d\mu. \end{aligned} \end{equation*}
    Thus
    🔗
    \begin{equation*} \int_E(f+g) \,d\mu = \int_E f\,d\mu + \int_E g\,d\mu \leq \nu(E). \end{equation*}
    Hence \(f +g \in F\text{.}\) However
    🔗
    \begin{equation*} \int_X(f+g)\,d\mu= a + \int_Xg\,d\mu > a, \end{equation*}
    contradicting the definition of \(a\text{.}\) Hence \(\lambda(E) = 0\) for all \(E \in \mathcal{F}\text{,}\) and thus
    🔗
    \begin{equation*} \nu(E) = \int_E f\,d\mu \end{equation*}
    as desired.
    🔗
    Now suppose that \(\mu\) and \(\nu\) are both \(\sigma\)-finite. There are disjoint sets \(\{A_n\}_{n=1}^\infty\) and disjoint sets \(\{B_n\}_{n=1}^\infty\) such that
    🔗
    \begin{equation*} \bigcup_n A_n = X = \bigcup_n B_n, \end{equation*}
    \(\mu(A_n)< \infty\) and \(\nu(B_n)< \infty\) for all \(n\text{.}\) Let \(\{C_k\}_{k=1}^\infty\) be the countable collection of disjoint sets \(\{A_n\cap B_m\}_{m,n=1}^\infty\text{.}\) Then \(X = \bigcup_k C_k\text{,}\) and for all \(k\text{,}\) \(\mu(C_k)<\infty\) and \(\nu(C_k)< \infty\text{.}\)
    🔗
    For each \(k\) there is a non-negative measurable function \(f_k\) such that
    🔗
    \begin{equation*} \nu(E) = \int_E f_k \,d\mu \end{equation*}
    for all measurable \(E \subseteq C_k\text{.}\) Extend \(f_k\) to all of \(x\) by setting \(f_k(x) = 0\) for \(x \notin C_k\text{.}\) Let \(f = \sum_{k=1}^\infty f_k\text{.}\) Take any \(E \in F\text{.}\) Applying Corollary 6.1.18, we get
    🔗
    \begin{equation*} \begin{aligned} \int_E f\,d\mu &= \sum_{k=1}^\infty \int_{E \cap C_k}f\,d\mu \\ &= \sum_{k=1}^\infty \int_{E \cap C_k}f_k\,d\mu \\ &= \sum_{k=1}^\infty \nu(E \cap C_k) \\ &= \nu(E). \end{aligned} \end{equation*}
    The uniqueness (almost everywhere) follows from Exercise 6.4.1.
    🔗
    🔗
    The following corollary to the Radon-Nikodym Theorem is immediate on taking the Jordan decomposition of a signed-measure and applying the Radon-Nikodym Theorem.
    🔗

    Corollary 7.1.7.

    Let \((X,\mathcal{F},\mu)\) be a \(\sigma\)-finite measure space and let \(\nu\) be a finite signed-measure on \((X,\mathcal{F})\) such that \(\nu \ll \mu\text{.}\) Then there is a measurable function \(f\) which is integrable with respect to \(\mu\) such that, for all \(E \in \mathcal{F}\text{,}\)
    🔗
    \begin{equation*} \nu(E) = \int_E f\,d\mu. \end{equation*}
    Further, the function \(f\) is unique up to almost everywhere equivalence.
    🔗
    🔗

    Definition 7.1.8.

    Let \((X,\mathcal{F},\mu)\) be a \(\sigma\)-finite measure space and let \(\nu\) be a \(\sigma\)-finite measure or a finite signed measure on \((X,\mathcal{F})\) such that \(\nu \ll \mu\text{.}\) Let \(f\) be the measurable function on \(X\) such that
    🔗
    \begin{equation*} \nu(E) = \int_E f\,d\mu \end{equation*}
    for all \(E \in \mathcal{F}\text{.}\) The function \(f\) is called the Radon-Nikodym derivative of \(\nu\) with respect to \(\mu\text{.}\) The Radon-Nikodym derivative is denotated as
    🔗
    \begin{equation*} f = \frac{d\nu}{d\mu}. \end{equation*}
    🔗
    The word “derivative” in Radon-Nikodym will perhaps raise some eyebrows. What is being derivated here? Section~\ref{sec: ftoc}, in particular Theorem 7.3.11, goes some way to justify the term. We will satiate ourselves with a simple example now.
    🔗

    Example 7.1.9.

    Let \(f(x) = x^2\) on \([0,1]\text{.}\) Let \(\nu\) be the Carathéodory extension of the premeasure
    🔗
    \begin{equation*} \nu((a,b])= f(b) - f(a) = b^2 - a^2 \end{equation*}
    on \([0,1]\text{.}\) We saw in Example 7.1.5 that \(\nu\) is absolutely continuous with respect to \(m\text{.}\) Hence, by the Radon-Nikodym theorem there is a non-negative measurable function \(h\) on \([0,1]\) such that
    🔗
    \begin{equation*} \nu(E) = \int_E h\,dm. \end{equation*}
    In particular, for \(0\leq a < b \leq 1\) we have
    🔗
    \begin{equation*} \nu((a,b]) = \int_{(a,b]}h\,dm \end{equation*}
    By the Fundamental Theorem of Calculus we also have
     1 
    We can comfortably apply the Fundamental Theorem of Calculus since \(f\) and \(f'\) are continuous on \([0,1]\text{,}\) and thus are Riemann integrable.
    🔗
    \begin{equation*} \nu((a,b]) = \int_{[a,b]} 2x \,dm(x). \end{equation*}
    By the uniqueness of the Carathéodory Extension, from the Carathéodory-Hahn Theorem, it follows that
    🔗
    \begin{equation*} \nu(E) = \int_E f'\,dm = \int_E h\,dm \end{equation*}
    for all Borel sets \(E \subseteq [a,b]\text{.}\) It follows from Exercise 6.4.7, that \(f(x) = 2x\text{.}\)
     2 
    Technically, it means that \(h(x) = 2x\) almost everywhere, but \(h\) can be chosen so that \(h(x) = 2x\text{.}\)
    Thus, the Radon-Nikodym derivative of \(\nu\) is the derivative of \(f\text{.}\) That is
    🔗
    \begin{equation*} \frac{d\nu}{dm} = \frac{df}{dx}. \end{equation*}
    🔗
    We end this section with an important decomposition theorem due to Lebesgue.
    🔗

    Theorem 7.1.10. Lebesgue Decomposition Theorem.

    Let \((X,\mathcal{F},\mu)\) be a \(\sigma\)-finite measure space and let \(\nu\) be a \(\sigma\)-finite measure on \((X,\mathcal{F})\text{.}\) Then there are mutually orthogonal measures \(\nu_s\) and \(\nu_a\) such that \(\nu = \nu_s + \nu_a\)
    🔗
    1. \(\nu_a \ll \mu\text{,}\) and
      🔗
      🔗
    2. \(\nu_s \perp \mu\text{.}\)
      🔗
      🔗
    🔗

    Proof.

    Let \(\lambda = \mu + \nu\text{.}\) Then \(\lambda\) is a \(\sigma\)-finite measure and \(\mu \ll \lambda\text{.}\) By the Radon-Nikodym Theorem, there is a non-negative measurable function \(f\) such that, for all \(E \in \mathcal{F}\)
    🔗
    \begin{equation*} \mu(E) = \int_\lambda f \,d\mu = \int_E f\,d\mu + \int_E f\,d\nu, \end{equation*}
    with the last equality following from Exercise 7.5.1. Let
    🔗
    \begin{equation*} X_+ = \{x \in X \colon f(x) > 0\} \end{equation*}
    and
    🔗
    \begin{equation*} X_0 = \{x \in X \colon f(x) = 0\}. \end{equation*}
    Define measures \(\nu_s\) and \(\nu_a\) by
    🔗
    \begin{equation*} \nu_s(E) = \nu(E \cap X_0) \text{ and }\nu_a(E) = \nu(E \cap X_+). \end{equation*}
    Then \(\nu_s\) and \(\nu_a\) are mutually singular, with \(\nu = \nu_s + \nu_a\text{.}\)
    🔗
    That \(\mu\) and \(\nu_s\) are mutually singular is clear. It remains to show \(\nu_a \ll \mu\text{.}\) Take \(E \in \mathcal{F}\) such that \(\mu(E) = 0\text{.}\) Then
    🔗
    \begin{equation*} 0 = \mu(E) = \int_E f\,d\mu + \int_E f\,d\nu. \end{equation*}
    Thus
    🔗
    \begin{equation*} 0 = \int_E f\,d\nu = \int_{E\cap X_+} f\,d\nu = 0. \end{equation*}
    Since \(f>0\) on \(E\cap X_+\text{,}\) it follows that \(\nu(E\cap X_+) = 0\) (Exercise 6.4.5). Thus \(\nu_a(E) = 0\text{,}\) and hence \(\nu_a\ll\mu\text{.}\)
    🔗
    🔗

    🔗
    1. \(\nu_a \ll \mu\text{,}\) and
      🔗
      🔗
    2. \(\nu_s \perp \mu\text{.}\)
      🔗
      🔗

    Definition 7.1.12.

    Let \((X,\mathcal{F},\mu)\) be a \(\sigma\)-finite measure space, let \(\nu\) be a \(\sigma\)-finite measure or a finite signed-measure, and let \(\nu_s\) and \(\nu_a\) are mutually orthogonal measures such that \(\nu_s \perp \mu\text{,}\) \(\nu_a \ll \mu\text{,}\) and \(\nu = \nu_s + \nu_a\text{.}\) The measures \(\nu_s\) and \(\nu_a\) are called the Lebesgue decomposition of \(\nu\) with respect to \(\mu\).
