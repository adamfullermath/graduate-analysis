    Section 6.3 Lebesgue integration and Riemann integration

    In Chapter~1 we argued for the need of a form of integration more robust than Riemann integration. Have we achieved that? We will see that for integrals of bounded functions over closed intervals \([a,b]\) that the answer is yes.
    🔗
    When \(f\) is Riemann integrable on an interval \([a,b]\) we denote the Riemann integral of \(f\) by \(\mathrm{R}{\int}_a^b f\text{.}\)
    🔗

    Lemma 6.3.1.

    Let \(f\) be a bounded function on the closed interval \([a,b]\text{.}\) If \(f\) is Riemann integrable, then \(f\) is measurable.
    🔗
    🔗

    Proof.

    Let \(D = \{x \in [a,b] \colon f \text{ not continuous at }x\}\text{,}\) and let \(E = [a,b]\backslash D\text{.}\) Since \(f\) is continuous on \(E\text{,}\) we have that \(f|_E\) is a measurable function by Corollary 5.1.5. By Theorem 3.1.9, \(m(D) = 0\text{.}\) Since \(m\) is a complete measure on \([a,b]\text{,}\) \(f|_D\) is a measurable function. Thus \(f\) is measurable.
    🔗
    🔗
    We have defined the Lebesgue integral in terms of integrals of simple functions. The Riemann integral is usually defined in terms of Riemann sums. The following definition allows us to describe Riemann sums by integrals of simple functions.
    🔗

    Definition 6.3.2.

    A simple function \(\varphi\) on a closed interval \([a,b]\) is a step function if
    🔗
    \begin{equation*} \varphi = \sum_{k=1}^n a_k \chi_{[x_{n-1},x_n]} \end{equation*}
    for some partition \(P = \{a=x_0 < x_1<\ldots<x_n = b\}\) of \([a,b]\text{.}\)
    🔗
    🔗
    Let \(f\) be a bounded function on \([a,b]\) and let \(P = \{a=x_0 < x_1<\ldots<x_n = b\}\) be a partition of \([a,b]\text{.}\) For \(1\leq k \leq n\text{,}\) let \(a_k = \inf_{x \in[x_{k-1},x_k]} f(x)\text{.}\) Then the lower Darboux sum of \(f\) with respect to \(P\) is
    🔗
    \begin{equation*} L(f,P) := \sum_{k=1}^n a_k(x_{k-1} - x_k). \end{equation*}
    If we let \(\varphi\) be the step function \(\phi_P = \sum_{k=1}^n a_k \chi_{[x_{k-1},x_k]}\) then
    🔗
    \begin{equation*} \int_{[a,b]} \phi_P \,dm = L(f,P). \end{equation*}
    Similarly, if \(b_k = \sup_{x \in[x_{k-1},x_k]} f(x)\text{.}\) Then the upper Darboux sum of \(f\) with respect to \(P\) is
    🔗
    \begin{equation*} U(f,P) := \sum_{k=1}^n b_k(x_{k-1} - x_k). \end{equation*}
    If we let \(\psi\) be the step function \(\psi_P = \sum_{k=1}^n b_k \chi_{[x_{k-1},x_k]}\) then
    🔗
    \begin{equation*} \int_{[a,b]} \psi_P \,dm = U(f,P). \end{equation*}
    That is, the upper and lower Riemann sums are equal to the integrals of step functions.
    🔗

    Theorem 6.3.3.

    Let \(f\) be a bounded function on a closed interval \([a,b]\text{.}\) If \(f\) is Riemann integrable then \(f\) is Lebesgue integrable and
    🔗
    \begin{equation*} \mathrm{R}{\int}_a^b f = \int_{[a,b]}f\,dm. \end{equation*}
    🔗

    Proof.

    Assume \(f\) is Riemann integrable. By Lemma 6.3.1, \(f\) is measurable. Let \(M\) be an upper bound for \(|f|\text{.}\) Then
    🔗
    \begin{equation*} \int_{[a,b]}|f|\,dm \leq M m([a,b]) < \infty. \end{equation*}
    Thus \(f\) is Lebesgue integrable.
    🔗
    For each partition \(P\) let \(\phi_P\) be a step function such \(\phi_P\leq f\) and
    🔗
    \begin{equation*} \int_{[a,b]}\phi_P\,dm = L(f,P), \end{equation*}
    and let \(\psi_P\) be a step function such \(f\leq\psi_P\) and
    🔗
    \begin{equation*} \int_{[a,b]}\psi_P\,dm = U(f,P). \end{equation*}
    Note that such a step functions \(\phi_P\) and \(\psi_P\) exist by the argument preceding this theorem.
    🔗
    Since \(f\) is Riemann integrable we have that
    🔗
    \begin{equation*} \mathrm{R}{\int}_a^b = \sup_P L(f,P) = \inf_P U(f,P). \end{equation*}
    Since, for every partition \(P\text{,}\) \(\phi_P \leq f \leq \psi_P\text{,}\) the monotonicity of Lebesgue integral tell us
    🔗
    \begin{equation*} \begin{aligned} \sup_P L(f,P) = \sup_P \int_{[a,b]}\phi_P\,dm &\leq \int_{[a,b]} f\,dm \\ &\leq \sup_P \int_{[a,b]}\psi_P\,dm = \sup_P U(f,P). \end{aligned} \end{equation*}
    Hence \(\mathrm{R}{\int}_a^b f = \int_{[a,b]}f\,dm\text{.}\)
    🔗
    🔗

    Remark 6.3.4.

    Let \([a,b]\) be a closed interval. We have shown that the set of bounded Riemann integrable functions on \([a,b]\) is contained in the set of bounded Lebesgue integrable functions on \([a,b]\text{.}\) When \([a,b]\) is not a singleton (i.e. when \(a<b\)), this inclusion is strict. If \(A = \mathbb{Q} \cap [a,b]\) then \(\chi_A\) is a Lebesgue integrable function on \([a,b]\text{,}\) but \([a,b]\) is not Riemann integrable.
    🔗
    🔗
    The following example shows that the statements analogous to the Monotone Convergence Theorem and the Lebesgue Dominated Convergence Theorem for Riemann integration do not hold.
    🔗

    Example 6.3.5.

    Let \(X = [0,1]\) and let \(A = \mathbb{Q} \cap [0,1]\text{.}\) The set \(A\) is countable and so can be enumerated as
    🔗
    \begin{equation*} A = \{a_1,a_2,\ldots\} \end{equation*}
    For each \(n\) let
    🔗
    \begin{equation*} A_n = \{a_1,\ldots,a_n\}, \end{equation*}
    and let \(f_n = \chi_{A_n}\text{.}\) Then \((f_n)_n\) is an increasing sequence of non-negative Riemann-integrable functions which converge pointwise to \(\chi_A\text{.}\) The Riemann integral of each \(f_n\) is \(0\text{.}\) However, as noted in Example 1.1.7, \(\chi_A\) is not Riemann integrable.
    🔗
    We do not have
    🔗
    \begin{equation*} \lim_n \mathrm{R}{\int}_0^1f_n = \mathrm{R}{\int}_0^1 \chi_A \end{equation*}
    since \(\mathrm{R}{\int}_0^1 \chi_A\) is not defined. Of course, using Lebesgue integration, we get
    🔗
    \begin{equation*} \lim_n \int_{[0,1]}f_n \,dm = \int_{[0,1]} \chi_A \,dm=0. \end{equation*}
    Indeed this is an easy calculation to do directly. Alternatively it follows from either the Monotone Convergence Theorem or the Lebesgue Dominated Convergence Theorem.
    🔗
    🔗
    Recall that Riemann integrals can be extended beyond closed intervals by improper Riemann integrals. That is we define
    🔗
    \begin{equation*} \begin{aligned} \mathrm{R}{\int}_a^\infty f &= \lim_{b \rightarrow \infty}\mathrm{R}{\int}_a^b f, \\ \mathrm{R}{\int}_{-\infty}^b f &= \lim_{a \rightarrow -\infty}\mathrm{R}{\int}_a^b f,\text{ and} \\ \mathrm{R}{\int}_{-\infty}^\infty f &= \lim_{b \rightarrow \infty}\lim_{a \rightarrow -\infty}\mathrm{R}{\int}_a^b f \\ \end{aligned} \end{equation*}
    The following examples show that it is possible for a function \(f\) to have a finite improper Riemann integral but not be Lebesgue integrable.
    🔗

    Example 6.3.6.

    Define a function \(f(x)\) on \([0,\infty]\) by
    🔗
    \begin{equation*} f(x) = \frac{(-1)^n}{n} \end{equation*}
    when \(n = \lceil x\rceil\text{.}\)
    Then
     1 
    The exact value this alternating series takes can be shown to be \(\ln(2)\text{,}\) but that is finite is enough for us here.
    🔗
    🔗
    \begin{equation*} \mathrm{R}{\int}_{-\infty}^\infty f = \sum_{n=1}^\infty \frac{(-1)^n}{n} < \infty \end{equation*}
    However
    🔗
    \begin{equation*} \int_{[0,\infty]} |f|\,dm = \sum_{n=1}^\infty \frac{1}{n} =\infty, \end{equation*}
    and hence \(f\) is not Lebesgue integrable.
    🔗
    🔗

    Example 6.3.8.

    Let \(g(x) = \frac{\sin(x)}{x}\) on the real line.
    🔗
    🔗
    It can be shown that
     2 
    Showing this takes some work, but it can be done using complex integration tricks or double integral tricks.
    🔗
    \begin{equation*} \mathrm{R}{\int}_{-\infty}^\infty g =\pi \end{equation*}
    Note for every integer \(n \geq 0\)
    🔗
    \begin{equation*} \begin{aligned} \int_{[n\pi,(n+1)\pi]} \left|\frac{\sin(x)}{x}\right|\,dm &\geq \frac{1}{(n+1)\pi} \int_{[n\pi,(n+1)\pi]} |\sin(x)|\, dm\\ &= \frac{2}{(n+1)\pi}. \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation*} \int_{[0,\infty]} |g|\,dm \geq \sum_{k=0}^\infty \frac{2}{(n+1)\pi}= \infty. \end{equation*}
    Thus
    🔗
    \begin{equation*} \int_\mathbb{R} |g|\, dm = \infty \end{equation*}
    and so \(g\) is not Lebesgue integrable.
    🔗
    🔗

    Remark 6.3.10.

    Example 6.3.6 and Example 6.3.8 highlight an important difference between Lebesgue integrals and improper Riemann integrals. Let \(f\) be a function on the non-negative real line \([0,\infty)\text{.}\) Then \(f\) has a finite improper Riemann integral if and only if \(f\) is Riemann integrable on every each interval \([n-1,n]\) and the series \(\sum_{n=1}^\infty \mathrm{R}{\int}_{n-1}^n f\text{.}\) Then
    🔗
    \begin{equation*} \mathrm{R}{\int}_0^\infty f = \sum_{n=1}^\infty \mathrm{R}{\int}_{n-1}^n f = \sum_{n=1}^\infty \int_{[n-1,n)}f\,dm. \end{equation*}
    For Lebesgue integration, convergence of the series \(\sum_{k=1}^\infty \int_{[n-1,n)}f\,dm\) is not sufficient. If \(f\) is Lebesgue integrable on \([0,\infty)\) then
    🔗
    \begin{equation*} \int_{[0,\infty)} f\,dm = \sum_{n=1}^\infty \int_{E_n} f\,dm \end{equation*}
    for any collection of disjoint measurable sets \(\{E_n\}_n\) with \(\bigcup_n E_n = [0,\infty)\text{.}\) Thus, reordering the collection \(\{E_n\}_n\) does not affect the value of
    🔗
    \begin{equation*} \int_{[0,\infty)}f\,dm = \sum_{n=1}^\infty \int_{E_n} f\,dm. \end{equation*}
    Thus, the series \(\sum_{n=1}^\infty \int_{E_n} f\,dm\) is not only convergent, but absolutely convergent. In Example 6.3.6 and Example 6.3.8
    🔗
    \begin{equation*} \sum_{n=1}^\infty \mathrm{R}{\int}_{n-1}^n f \text { and } \sum_{n=1}^\infty \mathrm{R}{\int}_{n-1}^n g \end{equation*}
    are conditionally convergent, not absolutely convergent.
