    Section 3.1 The integral of a derivative

    In a first real analysis course, or indeed a first calculus course, students are presented with two theorems grandly named The Fundamental Theorem of Calculus. We will focus on the first of these here (though both will be addressed later in the course).
    🔗

    Theorem 3.1.1. The Fundamental Theorem of Calculus I.

    Suppose
    🔗
    \begin{equation*} F \colon [a,b] \rightarrow \mathbb{R} \end{equation*}
    is a continuous function which is differentiable on \((a,b)\) and has continuous bounded derivative on \((a,b)\text{.}\) Then
    🔗
    \begin{equation*} \int_a^b F' = F(b) - F(a). \end{equation*}
    🔗
    This is clearly a useful, strong theorem. However, it does leave a number of questions unanswered. Is there a class of functions \(F\) so that the differentiability of \(F\) is automatic? Do we need \(F'\) to be continuous everywhere in \((a,b)\) for the theorem to hold? Do we need \(F'\) to be defined everywhere in \((a,b)\) for the theorem to hold?
    🔗
    The following simple example shows that there are functions \(F\) where the conclusion of the first fundamental theorem holds, but \(F'\) is not defined on all of the interval \((a,b)\text{.}\)
    🔗

    Example 3.1.2.

    Let \(F\) be the function \(F(x) = |x|\) on the interval \([a,b]\text{,}\) where \(a< 0 < b\text{.}\) Then \(F\) is differentiable at every point in \((a,b)\) except at \(0\text{.}\) Indeed,
    🔗
    \begin{equation*} F'(x) = \begin{cases}1 &\text{if }x>0 \\ -1 & \text{if }x<0.\end{cases} \end{equation*}
    We can choose any value for \(F'(0)\text{,}\) but we can not choose a value so that \(F'\) is continuous at \(0\text{.}\) However, we can still calculate the Riemann integral \(\int_{a}^b F'\text{,}\) since a Riemann integral is not changed by the value at a single point. We have that
    🔗
    \begin{equation*} \begin{aligned} \int_{a}^b F' &= \int_{a}^0 F' + \int_0^{b} F' \\ &= (-a)(-1)+(b)(1) \\ &= b + a \\ &= F(b) - F(a). \end{aligned} \end{equation*}
    🔗
    Example Example 3.1.2 shows that we can get away with \(F'\) not being defined everywhere, and still satisfy the conclusion of the first fundamental theorem, Theorem Theorem 3.1.1. Similar to Example Example 3.1.2, we can easily construct examples of continuous functions which are not differentiable at finitely many points. With a little thought, we can construct an example of a continuous function which is continuous and not differentiable at countably many points. We give one such example below.
    🔗

    Example 3.1.3.

    Define a function \(G\) on \([0,1]\) by
    🔗
    \begin{equation*} G(x) = \min\left\{\left|x-\frac{1}{n+1}\right|, \left|x -\frac{1}{n}\right|\right\},\ \text{when }\frac{1}{n+1} \leq x \leq \frac{1}{n}. \end{equation*}
    By construction, \(G\) is continuous. However, \(G\) is not differentiable at \(x= \frac{1}{n}\) for each integer \(n > 1\text{;}\) \(G\) is differentiable at all other \(x\) (where \(G'(x) = 1\) or \(G'(x) = -1\)).
    One can further show that
    🔗
    🔗
    \begin{equation*} \int_0^b G' = G(b) \end{equation*}
    for each \(0\leq b \leq 1.\) Thus
    🔗
    \begin{equation*} \int_a^b G' = G(b) - G(a). \end{equation*}
    🔗
    Example Example 3.1.3 shows that the conclusion of the Fundamental Theorem of Calculus holds for some functions where the derivative is undefined at infinitely many points. Can we classify/describe the functions where the Fundamental Theorem of Calculus applies?
    🔗
    These questions are closely related to another question which many 19th century mathematicians struggled with. What is the relationship between continuity and differentiability? If \(f\) is a continuous function, how non-differentiable can it be? Is a continuous function necessarily differentiable at some points? On some intervals? Is a continuous function differentiable `enough’ to apply the first fundamental theorem of calculus?
    🔗
    In Example Example 3.1.3, the function \(G\) is not differentiable at each \(x=1/n\text{.}\) Whilst this gives (countably) infinite points where \(G\) is not differentiable, each point where \(G\) is not differentiable is isolated (meaning if \(G\) is not differentiable at \(x\) then there is an interval around \(x\) in which \(G\) is differentiable at all \(y \neq x\)). In the early 1800s, it was widely believed (or suspected, at least) that all examples would satisfy this. That is, it was conjectured that if \(f\) was continuous on a closed interval \([a,b]\) then each point where \(f\) is not differentiable is isolated). André-Marie Ampère (1775–1836) went so far as to present a `proof’ of this statement.
     1 
    All mathematicians make mistakes. Most mathematicians go as far to publish mistakes. If you are to remember Ampère for anything it should not be for this. Augustin-Louis Cauchy (1789 - 1857) credits Ampère in influencing his work on the formalization of calculus.
    🔗
    The question of the relationship between continuity and differentiability was settled by Karl Weierstrass (1815–1897) and Hermann Schwarz (1843–1921) in the 1870s. Weierstrass published the first example of a continuous function \(F \colon [a,b] \rightarrow \mathbb{R}\) which is differentiable nowhere.
    🔗
    Schwarz gave an example of a continuous function \(G\) which is differentiable at all but countably many points; but the points where \(G\) is not differentiable are not isolated. Schwarz’s example has an unbounded derivative which is not Riemann integrable.
     2 
    The history is more complicated than alluded to here. As earlier as the 1830s Bernard Bolzano (1781-1848) gave an example of a nowhere differentiable function. This work was not published at the time. In Bolzano’s work he exhibited the continuity of the function and that it was not differentiable on a dense set. That Bolzano’s function is nowhere differentiable was not proved until after Bolzano’s manuscript was published in the 1920s. Charles Cellérier (1818-1889) gave an example of a nowhere differentiable continuous function in the 1860s. This was not published until after Cellérier’s death, and after Weierstrass’s work, in 1890.
    🔗
    The existence of nowhere differentiable functions means that there are continuous functions for which the first fundamental theorem does not apply. But for which functions does it apply? Which continuous functions are differentiable `enough’ to apply the fundamental theorem?
    🔗
    Here is where we run into some shortcomings of the Riemann integral. Consider the following two examples.
    🔗

    Example 3.1.5.

    Let \(A = \mathbb{Q} \cap [0,1]\text{.}\) Define a function \(f[0,1] \rightarrow \mathbb{R}\) on \([0,1]\) by
    🔗
    \begin{equation*} f(x) = \begin{cases} \frac{1}{n} & \text{if }x= \frac{m}{n}\in A\text{ where } \gcd(m,n)=1.\\ 0 & \text{if }x \notin A.\\ \end{cases} \end{equation*}
    This is function goes by several names, including Riemann’s function, Thomae’s function, and the popcorn function.
    Bernhard Riemann (1826 - 1866) introduced this function in his doctoral thesis in 1854, in which he also introduced the method of integration which bears his name. Riemann showed that this function was continuous at each irrational, \(x \in [0,1]\backslash A\text{,}\) and discontinuous at the rational points, \(x \in A\text{.}\) He further showed that \(f\) is Riemann integrable and
    🔗
    🔗
    \begin{equation*} \int_0^1 f = 0. \end{equation*}
    🔗

    Example 3.1.7.

    Let \(A = \mathbb{Q} \cap [0,1]\text{.}\) Consider the characteristic function \(\chi_A \colon [0,1] \rightarrow \mathbb{R}\text{.}\) That is
    🔗
    \begin{equation*} \chi_A(x) = \begin{cases} 1 & \text{if }x \in A \\ 0 & \text{if }x \notin A. \end{cases} \end{equation*}
    As \(A\) is dense in \([0,1]\text{,}\) it follows that the lower integral
    🔗
    \begin{equation*} \underline{\int_0^1} \chi_A = 0, \end{equation*}
    and the upper integral
    🔗
    \begin{equation*} \overline{\int_0^1} \chi_A = 1. \end{equation*}
    Hence \(\chi_A\) is not Riemann integral.
    🔗
    🔗
    Examples Example 3.1.5~and Example 3.1.7 show that Riemann integration can sometimes work well with countable dense subsets, but not always. Hence Riemann integration may not work well when dealing with the derivative of Schwarz’s example. Vito Volterra (1860 - 1940) gave an example in 1881 which gives a further complication to the theory. Volterra’s function \(V \colon [0,1] \rightarrow \mathbb{R}\) is continuous, with \(V'(x)\) bounded and defined for all \(x \in (0,1)\text{.}\) However, \(V'(x)\) is not Riemann integrable. Hence, we can not apply Theorem Theorem 3.1.1 to \(V\text{.}\)
    🔗
    These failures of the Riemann integral necessitate defining a new type of integral: the Lebesgue integral. Henri Lebesgue (1875 - 1941) introduced this more robust way integrating functions. For Riemann integrable functions, the Lebesgue integral and the Riemann integral coincide. However, the Lebesgue integral allows us to integrate many functions which are not Riemann integrable, including \(\chi_A\) from Example Example 3.1.7 and the derivative of Volterra’s function \(V'(x)\text{.}\)
    🔗
    Lebesgue also worked on the theory of differentiation, and its relationship to (both Riemann and Lebesgue) integration. Lebesgue showed that every increasing function is differentiable almost everywhere. This, in turn, can be used to show that a class of continuous functions called absolutely continuous functions, are differentiable almost everywhere. The term `almost everywhere’ is not a vague colloquialism; it has an exact technical definition that we will see later in this course. In Chapter Chapter 4 we will study measure spaces. In Chapter Chapter 5 we will introduce the Lebesgue measure on subsets the real line, a generalization of the length function on intervals in \(\mathbb{R}\text{.}\) In Chapter Chapter 8 we use these ideas to introduce Lebesgue integration (and generalizations thereof). In Chapter Chapter 9 we will study absolutely continuous functions and complete our story on the relationship between (Lebesgue) integration and differentiation. In Theorem Theorem 9.3.13 we give Lebesgue’s version of the fundamental theorem of calculus:
    🔗
    \begin{equation*} \int_a^b f' = f(b) - f(a) \end{equation*}
    for all absolutely continuous functions \(f\) (where we use the Lebesgue integral to integrate \(f'\)).
