    Section 1.1 Nowhere differentiable functions

    In 1872, Karl Weierstrass gave an example of a real valued function which is nowhere differentiable. This put an end to several decades of attempts by mathematicians to determine to what extent continuity determined differentiability. Whilst nowhere differentiable functions were difficult to discover, they are, in fact, very common. If we put the \(\|\cdot\|_\infty\) norm on \(C([a,b])\text{,}\) then the nowhere differentiable functions form a dense subset in \(C([a,b])\text{.}\) Further, there is a natural probability measure on \(C([a,b])\) (the classical Wiener measure) with which the nowhere differentiable functions have measure \(1\text{.}\)
    🔗
    Before we begin, we should discuss what it means for a function to be differentiable. Let \(f\) be a function whose domain includes a point \(x_0\in\mathbb{R}\) and an open interval containing \(x_0\text{.}\) The derivative \(f'\) at the point \(x_0\) is defined as
    🔗
    \begin{equation*} f'(x_0) = \lim_{h\rightarrow \infty} \frac{f(x_0+h) - f(x_0)}{h}, \end{equation*}
    when the limit exists. If the limit does not exist, or if \(f'(x_0)=\pm \infty,\) we say \(f\) is not differentiable at \(x_0\text{.}\)
    🔗
    Thus, there is some ambiguity in the literature on what it means for a function to be nowhere differentiable. The Weierstrass function in Section~\ref{subsec: weierstrass} is nowhere differentiable in our sense. That is,
    🔗
    \begin{equation*} \lim_{h\rightarrow \infty} \frac{f(x_0+h) - f(x_0)}{h} \end{equation*}
    does not exist at any \(x_0\text{.}\) The example in Section~\ref{subsec: Tak} is nowhere differentiable in the sense of the more restrictive sense of the derivative:
    🔗
    \begin{equation*} \lim_{h\rightarrow \infty} \frac{f(x_0+h) - f(x_0)}{h} \end{equation*}
    is either \(\pm\infty\text{,}\) or the limit does not exist at each \(x_0\text{.}\)
    🔗

    Subsection The Weierstrass function

    In this section we study the Weierstrass function, introduced by Weierstrass in 1872. This is, perhaps, the most historically significant example of a nowhere differentiable function: whilst it was not the first example ever written down, it was the first widely known example. For our discussion we will closely follow Weierstrass’s own description of the function, see e.g. [20]. The rough idea is to create a function that oscillates wildly at every point.
    🔗
    Fix an odd integer \(a\) and real value \(b\) with \(0<b<1\) such that
    🔗
    \begin{equation*} ab > 1 + \frac{3}{2}\pi. \end{equation*}
    Define, for all \(x\in \mathbb{R}\text{,}\)
    🔗
    \begin{equation*} f(x) = \sum_{n=0}^\infty b^n\cos(a^n \pi x). \end{equation*}
    We call \(f\) the Weierstrass function. A straightforward application of the Weierstrass \(M\)-test shows that \(f\) is uniformly convergent, and hence continuous.
    🔗
    Now fix any \(x_0 \in \mathbb{R}\text{.}\) For each positive integer \(m\geq0\text{,}\) there is an integer \(\alpha_m\) such that
    🔗
    \begin{equation*} -1/2<2^mx_0 - \alpha_m \leq 1/2. \end{equation*}
    Let \(x_{m+1} =2^mx_0 - \alpha_m.\) We define values, \(y_m'\) and \(y_m''\) by
    🔗
    \begin{equation*} y_m' = \frac{\alpha_m - 1}{a^m} \text{ and }y_m'' = \frac{\alpha_m + 1}{a^m}. \end{equation*}
    Then
    🔗
    \begin{equation*} y_m' - x_0 = - \frac{1 +x_{m+1}}{a^m} < 0, \end{equation*}
    and
    🔗
    \begin{equation*} y_m'' - x_0 = \frac{1+x_{m+1}}{a^m} >0. \end{equation*}
    Thus \(y_m' < x_0 < y_m''.\) Further, \(\lim_{m\rightarrow \infty}y_m' = \lim_{m\rightarrow \infty}y_m'' = x_0.\)
    🔗
    Now,
    🔗
    \begin{equation*} \begin{aligned} \frac{f(y_m') - f(x_0)}{y_m' - y_0} =& \sum_{n=0}^\infty b^n \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{y_m' - x_0}\\ =& \sum_{n=0}^{m-1} b^n \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{y_m' - x_0}\\ &+ \sum_{n=m}^\infty b^n \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{y_m' - x_0}\\ =& \sum_{n=0}^{m-1} (ab)^n \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{a^n(y_m' - x_0)}\\ &+ \sum_{n=0}^\infty b^{n+m} \frac{\cos(a^{n+m}\pi y_m') - \cos(a^{n+m}\pi x_0')}{y_m' - x_0} \end{aligned} \end{equation*}
    Fix \(0\leq n \leq m-1\text{.}\) Using the trigonometric identity
    🔗
    \begin{equation*} \cos(\alpha + \beta) - \cos(\alpha - \beta) = -2 \sin(\alpha)\sin(\beta), \end{equation*}
    we have
    🔗
    \begin{equation*} \begin{aligned} \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{a^n(y_m' - x_0)} &= -2 \frac{\sin\left(a^n\pi \frac{y_m'-x_0}{2}\right)\sin\left(a^n\pi \frac{y_m'+x_0}{2}\right)}{a^n(y_m'-x_0)}\\ &= -\pi \sin\left(a^n\pi \frac{y_m'-x_0}{2}\right)\frac{\sin\left(a^n\pi \frac{y_m'+x_0}{2}\right)}{a^n\pi\frac{y_m'+x_0}{2}}. \end{aligned} \end{equation*}
    It follows that
    🔗
    \begin{equation*} \left|\sum_{n=0}^{m-1} (ab)^n \frac{\cos(a^n\pi y_m') - \cos(a^n\pi x_0')}{a^n(y_m' - x_0)}\right| \leq \pi \sum_{n=0}^{m-1} (ab)^n= \frac{\pi}{ab-1}(ab)^m. \end{equation*}
    Since \(a\) is an odd number, we have
    🔗
    \begin{equation*} \begin{aligned} \cos(a^{m+n}\pi y_m')&=\cos(a^n\pi(\alpha_m-1))=(-1)^{\alpha_m-1},\text{ and}\\ \cos(a^{m+n}\pi x_0)&=\cos(a^n\pi\alpha_m+a^n\pi x_{m+1})=(-1)^{\alpha_m}\cos(a^n\pi x_{m+1}). \end{aligned} \end{equation*}
    Hence
    🔗
    \begin{equation*} \begin{aligned} \sum_{n=0}^\infty & b^{n+m} \frac{\cos(a^{n+m}\pi y_m') - \cos(a^{n+m}\pi x_0')}{y_m' - x_0}\\ &= (-1)^{\alpha_m}(ab)^m\sum_{n=0}^\infty b^n \frac{1+\cos(a^n\pi x_{m+1})}{1+x_{m+1}}. \end{aligned} \end{equation*}
    For each \(n\text{,}\)
    🔗
    \begin{equation*} \frac{1+\cos(a^n\pi x_{m+1})}{1+x_{m+1}} \geq 0. \end{equation*}
    Further, since \(-1/2<x_{m+1}\leq 1/2\text{,}\) \(\cos(\pi x_{m+1}) \geq0\text{.}\) Hence
    🔗
    \begin{equation*} \frac{1+\cos(\pi x_{m+1})}{1+x_{m+1}}\geq \frac{2}{3}. \end{equation*}
    Putting all of this together, there is a \(\eta>0\) and \(-1<\varepsilon<1\) such that
    🔗
    \begin{equation*} \frac{f(y_m') - f(x_0)}{y_m' - y_0} = (-1)^{\alpha_m}(ab)^n\eta\left(\frac{2}{3} + \varepsilon \frac{\pi}{ab-1} \right)=:A_m. \end{equation*}
    By a similar analysis, there is a \(\eta'>1\) and \(-1<\varepsilon'<1\) such that
    🔗
    \begin{equation*} \frac{f(y_m'') - f(x_0)}{y_m'' - y_0} = -(-1)^{\alpha_m}(ab)^n\eta'\left(\frac{2}{3} + \varepsilon' \frac{\pi}{ab-1} \right)=:B_m. \end{equation*}
    Since we have assumed \(ab>1+\frac{3}{2}\pi\text{,}\)
    🔗
    \begin{equation*} \frac{2}{3} > \frac{\pi}{ab-1}. \end{equation*}
    Thus, \(A_m\) and \(B_m\) have opposite signs for each \(m\text{.}\) Now, since
    🔗
    \begin{equation*} \lim_m |A_m| = \infty = \lim_m |B_m|, \end{equation*}
    it follows that \(f\) is not differentiable at \(x_0\text{.}\)
    🔗
    🔗

    Subsection Takagi’s example

    We present now an example of a function which has does not have a finite derivative at any point. Unlike the Weierstrass function discussed in Section~\ref{subsec: weierstrass}, this function does have infinite derivative at some points (rather than the derivative not being defined). This example was first described by Takagi in 1903. It can be found in [15], see also [2]. A similar function was independently discovered by Van Der Waerden in 1930 [18]. Riesz and Sz.-Nagy’s text [8] begins with a presentation of Van Der Waerden’s example.
    🔗
    We present Takagi’s example because it is more intuitive than the Weierstrass function. The simplest example of a function not differentiable at a point is \(f(x) = |x|\text{.}\) This function is not differentiable at \(x=0\text{;}\) there is a sharp point at \(0\text{.}\) The idea behind Takagi’s example is to create a function made from a series of increasingly many `sharp points’.
    🔗
    Let
    🔗
    \begin{equation*} g(x) = \min\{x-\lfloor{x}\rfloor, \lceil{x}\rceil -x\}, \end{equation*}
    where \(\lfloor x \rfloor\) is the floor function (i.e. \(x\) rounded down to the nearest integer) and \(\lceil{x}\rceil\) is the ceiling function (i.e. \(x\) rounded up to the nearest integer). The function \(g\) is not differentiable at any point \(x = n/2\) where \(n \in \mathbb{Z}\text{.}\) At all other points,
    🔗
    \begin{equation*} g'(x) = \begin{cases} 1 &,\ x-\lfloor{x}\rfloor < 1/2\\ -1 &,\ x-\lfloor{x}\rfloor > 1/2. \end{cases} \end{equation*}
    Define a function \(f\) on the closed interval \([0,1]\) by
    🔗
    \begin{equation*} f(x) = \sum_{n=0}^\infty \frac{g(2^nx)}{2^n}. \end{equation*}
    The series \(f\) is uniformly convergent by the Weierstrass \(M\)-test; and hence \(f\) is continuous.
    🔗
    Fix \(x_0 \in (0,1)\text{.}\) We can write \(x_0\) as a series
    🔗
    \begin{equation*} x_0 = \sum_{k=1}^\infty \frac{c_k}{2^k}, \end{equation*}
    where each \(c_k \in \{0,1\}.\) Note that this series expression may not be unique, e.g.
    🔗
    \begin{equation*} \frac{1}{2} = \sum_{k=2}^\infty \frac{1}{2^k}. \end{equation*}
    In these circumstances, we choose the finite sum instead of the infinite. Note that, for each \(n\text{,}\)
    🔗
    \begin{equation*} g(2^nx_0) = \begin{cases} 2^nx_0 - \lfloor2^n x_0\rfloor & \text{if }c_n = 0,\\ \lceil2^nx_0\rceil - x_0 & \text{if }c_n = 1. \end{cases} \end{equation*}
    For each \(m\geq 0\) let
    🔗
    \begin{equation*} p(m) = \sum_{k=1}^m c_k \text{ and }z(m) = \sum_{k=1}^m1-c_k. \end{equation*}
    That is, \(p(m)\) is the number of ones in the list \(c_1,\ldots,c_m\text{,}\) and \(z(m)\) is the number of zeros in the list \(c_1,\ldots,c_m\text{.}\)
    🔗
    Suppose \(c_m = 0\text{.}\) Let \(h = \frac{1}{2^m}.\) Then, for \(n \geq m\text{,}\)
    🔗
    \begin{equation*} g(2^n(x_0 + h)) - g(2^nx_0) = 0. \end{equation*}
    Let
    🔗
    \begin{equation*} K_m = \frac{f(x_0+h) - f(x_0)}{h}. \end{equation*}
    Then
    🔗
    \begin{equation*} K_m= 10^m \sum_{n=0}^{m-1} \frac{g(2^n(x_0 + h)) - g(2^nx_0)}{2^n} = z(m) - p(m). \end{equation*}
    For \(m > 2\text{,}\) if \(c_m =1\text{,}\) \(c_{m-1} =0\text{,}\) and \(c_{m-2}=1\text{,}\) let \(h = \frac{1}{2^m}.\) Then, again
    🔗
    \begin{equation*} K_m = \frac{f(x_0+h) - f(x_0)}{h} = z(m) - p(m). \end{equation*}
    We these values for
    🔗
    \begin{equation*} \frac{f(x_0+h) - f(x_0)}{h} \end{equation*}
    we can find a subsequence \((h_{m_k})_k\) of \((1/2^m)_m\) such that
    🔗
    \begin{equation*} \lim_{k \rightarrow \infty}\frac{f(x_0+h_{m_k}) - f(x_0)}{h_{m_k}} \end{equation*}
    either does not exist, or is \(\infty.\) Thus \(f\) does not have a finite derivative at any point.
    🔗
    There is nothing special about the role of \(2\) in this example. Van der Waerden used base 10 for his example:
    🔗
    \begin{equation*} f(x) = \sum_{n=0}^\infty \frac{g(10^nx)}{10^n}. \end{equation*}
    Similarly analysis can be done for other bases.
