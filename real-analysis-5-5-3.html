    Section 4.1 Signed measures

    We have defined measures to only take non-negative values. However, there are some points where it is beneficial to extend the definition of a measure to include negative values (and we will see more as we proceed). For example, Exercise 4.3.1 shows that the collection of measures on \((X,\mathcal{F})\) is closed under addition and multiplying by non-negative scalars. The set of measures on \((X,\mathcal{F})\) is almost a vector space; we just can’t multiply be negative scalars. Allowing for measures which take positive and negative values would allow us to create a vector space (which may, or may not, be useful).
    🔗
    Less abstractly, in Section~\ref{sec: borel on R}, we showed that if \(\mu\) is a Borel measure on \([a,b]\) then the cumulative distribution function for \(\mu\text{,}\) \(g_\mu\text{,}\) is an increasing, right-continuous function, Thereom~\ref{thm: cdf}. Conversely, given an increasing, right-continuous function \(g\) on \([a,b]\text{,}\) \(g\) uniquely determines a Borel measure on \([a,b]\text{,}\) Theorem 3.3.7. Cumulative distribution functions are increasing precisely because measures take non-negative values. In Theorem 3.3.7, we need an increasing function in order to define a measure. If we want to use measure theory to study real functions, other than increasing functions, we will need to allow for measures which allow for possibly negative values. This leads us to the following definition.
    🔗

    Definition 4.1.1.

    Let \((X,\mathcal{F})\) be a measurable space. A function
    🔗
    \begin{equation*} \nu \colon \mathcal{F} \rightarrow [-\infty,\infty] \end{equation*}
    is a signed measure if
    🔗
    1. \(\nu\) takes at most value from \(\{-\infty,\infty\}\text{;}\)
      🔗
      🔗
    2. \(\nu(\emptyset) = 0\text{;}\) and
      🔗
      🔗
    3. if \(\{E_k\}_{k=1}^\infty\) are disjoint sets in \(\mathcal{F}\text{,}\) then
      \begin{equation*} \nu(\bigcup_{k=1}^\infty E_k) = \sum_{k=1}^\infty \nu(E_k). \end{equation*}
      🔗
      🔗
    The elements of \(\mathcal{F}\) are called the measurable sets, or \(\nu\)-measurable sets. A signed measure \(\nu\) on \((X,\mathcal{F})\) is finite if \(|\nu(X)|< \infty\text{.}\)
    🔗
    🔗
    The definition of a signed measure is very similar to that of a measure, but a couple of points are worth discussing. Firstly, the condition that \(\nu\) takes at most one value from \(\{-\infty,\infty\}\) is necessary in order for the third condition (countable additivity) to make sense. Suppose \(\nu\) is a function on a \(\sigma\)-algebra \(\mathcal{F}\) satisfying conditions (ii) and (iii) of a signed measure. Suppose \(\nu(E) = \infty\) and \(\nu(F) = -\infty\text{.}\) Condition (iii) then says that
    🔗
    \begin{equation*} \infty = \nu(E) = \nu(E\backslash F) + \nu(E\cap F). \end{equation*}
    Hence, \(\nu(E \cap F) \neq - \infty\text{.}\) Further
    🔗
    \begin{equation*} -\infty = \nu(F) = \nu(F \backslash E) + \nu(E \cap F). \end{equation*}
    Since \(\nu(E\cap F) \neq - \infty\text{,}\) it follows that \(\nu(F \backslash E) = - \infty\text{.}\) Thus
    🔗
    \begin{equation*} \nu(E \cup F) = \nu(F \backslash E) + \nu(E) = -\infty + \infty. \end{equation*}
    As \(-\infty + \infty\) is not defined, \(\nu\) cannot defined to take both of the values \(-\infty\) and \(\infty\text{.}\) In particular, we could have defined signed measures to only satisfy (ii) and (iii); the above argument shows that condition (i) follows from condition (iii). We include condition (i) for clarity.
    🔗
    Secondly, there is room for some misunderstandings in condition (iii). Suppose \(\{E_k\}_{k=1}^\infty\) are disjoint measurable sets, and \(E = \bigcup_k E_k\text{,}\) and that \(|\nu(E)|<\infty\text{.}\) Then
    🔗
    \begin{equation*} \sum_{k=1}^\infty \nu(E_k) = \nu(E) < \infty. \end{equation*}
    \ That is, \(\sum_{k=1}^\infty \nu(E_k)\) is a convergent series. There is more than one way a series can converge: absolutely or conditionally. Note that the set \(E\) is independent on the order of the sets \(\{E_k\}_{k=1}^\infty\text{,}\) and so \(\nu(E)\) is independent of the order of the sets \(\{E_k\}_{k=1}^\infty\text{.}\) As \(\sum_k \nu(E_k) = \nu(E)\text{,}\) it follows that the (finite) value of \(\sum_k \nu(E_k)\) is independent of the order of the terms \(\nu(E_k)\text{.}\) It follows that the series \(\sum_k \nu(E_k)\) converges absolutely.
    🔗
    Exercise 4.3.3 says that the difference of two measures is a signed measure. In this section we will show that all signed measures are of this form. We need some preliminary definitions and results first.
    🔗

    Definition 4.1.2.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\) and let \(A\) be a measurable set.
    🔗
    1. The set \(A \subseteq X\) is positive (or \(\nu\)-positive if there is room for confusion) if \(B \subseteq A\) is measurable, then \(\nu(B) \geq 0\text{;}\)
      🔗
      🔗
    2. the set \(A \subseteq X\) is negative (or \(\nu\)-negative if there is room for confusion) if \(B \subseteq A\) is measurable, then \(\nu(B) \leq 0\text{;}\) and
      🔗
      🔗
    3. the set \(A \subseteq X\) is a null set (or \(\nu\)-null if there is room for confusion) if \(B \subseteq A\) is measurable, then \(\nu(B) = 0\text{.}\)
      🔗
      🔗
    🔗

    Remark 4.1.3.

    Note that null sets are both positive and negative.
    🔗
    🔗

    Lemma 4.1.4. Hahn.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) Suppose \(E \in \mathcal{F}\text{,}\) with \(0 < \nu(E) < \infty\text{.}\) Then there is a positive set \(A \subseteq E\) such that \(\nu(A)>0\text{.}\)
    🔗
    🔗

    Proof.

    If \(E\) is positive, then we are done. Otherwise, there are subsets \(B \subseteq E\) with \(\nu(B)<0\text{.}\) Let \(m_1\) be the smallest natural number such that there is a measurable \(B_1 \subseteq E\) such that
    🔗
    \begin{equation*} \nu(B_1) < -\frac{1}{m_1}. \end{equation*}
    Continuing, choose minimal natural number \(m_k\) such that there is a
    🔗
    \begin{equation*} B_k \subseteq E \backslash \bigcup_{j=1}^{k-1}B_j \end{equation*}
    with
    🔗
    \begin{equation*} \nu(B_k) < - \frac{1}{m_k}. \end{equation*}
    If this process ends after \(n\) steps, let
    🔗
    \begin{equation*} A = E \backslash \bigcup_{k=1}^n B_k. \end{equation*}
    Then \(A\) is positive, since it contains no subsets \(B\) such that \(\nu(B)<0\text{.}\) Further,
    🔗
    \begin{equation*} \nu(E) = \nu(A) + \sum_{k=1}^n \nu(B_k). \end{equation*}
    Since \(\nu(E)>0\) and \(\sum_{k=1}^n \nu(B_k)<0\text{,}\) it follows that \(\nu(A)>0\text{.}\)
    🔗
    If the process does not end, we get a sequence of sets \(B_1,B_2,\ldots.\) Let \(B = \bigcup_{k=1}^\infty B_k\) and let \(A = E \backslash B.\) Since \(\nu(E) < \infty\text{,}\) it follows from Exercise 4.3.5 that \(|\nu(B)|<\infty\text{.}\) Thus
    🔗
    \begin{equation*} -\infty < \nu(B) = \sum_{k=1}^\infty \nu(B_k)\leq \sum_{k=1}^\infty \frac{-1}{m_k} < 0. \end{equation*}
    Thus \(\lim_{k\rightarrow \infty} m_k = \infty\text{.}\)
    🔗
    We will show that \(A\) is positive. Suppose \(C \subseteq A\) and \(\nu(C)<0.\) Then for each \(k\) we have
    🔗
    \begin{equation*} C \subseteq A \subseteq E\backslash \bigcup_{j=1}^{k-1}B_j. \end{equation*}
    Thus, by the choice of \(m_k\text{,}\)
    🔗
    \begin{equation*} \nu(C) \geq \frac{-1}{m_k-1}. \end{equation*}
    As this holds for all \(k\text{,}\) and \(\lim_k m_k = \infty\text{,}\) it follows that \(\nu(C)\geq 0\text{.}\) Hene \(A\) is a positive set.
    🔗
    To show that \(\nu(A)>0\) we note that
    🔗
    \begin{equation*} \nu(E) = \nu(A) + \nu(B), \end{equation*}
    Since \(\nu(E)>0\) and \(\nu(B)<0\text{,}\) it follows that \(\nu(A) >0\text{.}\)
    🔗
    🔗

    Theorem 4.1.5. Hahn Decomposition Theorem.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) Then there are disjoint measurable subsets \(A,B \subseteq X\) such that \(X = A \cup B\text{,}\) \(A\) is positive, and \(B\) is negative.
    🔗
    If \(A',B'\) are also disjoint measurable sets with \(A'\) positive, \(B'\) negative, and \(X = A' \cup B'\text{,}\) then the sets
    🔗
    \begin{equation*} (A\backslash A') \cup (A'\backslash A)<fn>The set is called the <em>symmetric difference</em> of and </fn> \text{ and }(B\backslash B') \cup (B'\backslash B) \end{equation*}
    are null sets.
    🔗
    🔗

    Proof.

    Assume that \(\nu\) does not take the value \(+\infty\) (replace \(\nu\) with \(-\nu\) if it does). Let \(\mathcal{P} \subseteq \mathcal{F}\) be the collection of all positive sets. Note that \(\mathcal{P}\) is non-empty since \(\emptyset \in \mathcal{P}\text{.}\) Let
    🔗
    \begin{equation*} \lambda = \sup \{\nu(A) \colon A \in \mathcal{P}\}. \end{equation*}
    We will show that \(\lambda\) is attained, i.e. \(\lambda\) is the maximum value in \(\{\nu(A)\}_{A \in \mathcal{P}}\text{.}\) Choose a sequence \(\{A_k\}_{k=1}^\infty\) such that
    🔗
    \begin{equation*} \lim_{k\rightarrow \infty} \nu(A_k) = \lambda. \end{equation*}
    Let \(A = \bigcup_{k=1}^\infty A_k\text{.}\) Note that \(A\) is positive by Exercise 4.3.6. Thus \(\nu(A) \leq \lambda\text{.}\) Also, since \(A\) is positive, \(\nu(A \backslash A_k) \geq 0\text{,}\) and
    🔗
    \begin{equation*} \begin{aligned} \nu(A) =& \nu(A \backslash A_k) + \nu(A_k)\\ \geq& \nu(A_k). \end{aligned} \end{equation*}
    Taking the limit as \(k \rightarrow \infty\text{,}\) we get \(\nu(A) \geq \lambda\text{.}\) Thus \(\nu(A) = \lambda\text{.}\)
    🔗
    Let \(B = X \backslash A\text{.}\) We will show that \(B\) is negative. If \(B\) is not negative, then there is a measurable set \(C \subseteq B\) with \(\nu(C)>0\text{.}\) Since, by assumption, \(\nu\) does not take the value \(\infty\text{,}\) we have
    🔗
    \begin{equation*} 0 < \nu(C) < \infty. \end{equation*}
    It follows from Lemma 4.1.4 that there is a positive set \(C' \subseteq C\) with \(\nu(C') >0\text{.}\) The set \(A \cup C'\) is positive with
    🔗
    \begin{equation*} \nu(A \cup C') = \nu(A) + \nu(C') > \lambda. \end{equation*}
    This contradicts the definition of \(\lambda\text{.}\) Thus \(B\) is negative.
    🔗
    Suppose \(A'\) and \(B'\) are another pair of disjoint measurable sets, with \(X = A' \cup B'\text{,}\) \(A'\) positive, and \(B'\) negative. Then \(A\backslash A' \subseteq A\) and \(A\backslash A' \subseteq B'\text{.}\) Hence \(A \backslash A'\) is both positive and negative. Thus, \(A \backslash A'\) is a null set. Similar arguments show that \(A'\backslash A\text{,}\) \(B\backslash B'\text{,}\) and \(B'\backslash B\) are null sets.
    🔗
    🔗

    Definition 4.1.6.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) If \(A\) and \(B\) are disjoint measurable sets satisfying \(X = A \cup B\) for disjoint sets \(A\) and \(B\text{,}\) with \(A\) positive and \(B\) negative, we call \(A\) and \(B\) a Hahn decomposition for \(\nu\text{.}\)
    🔗
    🔗
    The Hahn Decomposition, Theorem 4.1.5, says that Hahn decompositions exist for all signed measures.
    🔗
    We can now, as promised, decompose signed measures as the difference of two measures. Before we do this we introduce just one more definition.
    🔗

    Definition 4.1.7.

    Let \((X,\mathcal{F})\) be a measurable space, and let \(\mu\) and \(\nu\) be two measures on \((X,\mathcal{F})\text{.}\) The measures \(\mu\) and \(\nu\) are mutually singular, denoted \(\mu \perp \nu\text{,}\) if there are disjoint measurable sets \(A,B \subseteq X\) with \(A \cup B = X\) satisfying \(\mu(A) = \mu(X)\text{,}\) \(\mu(B) = 0\text{,}\) \(\nu(A) = 0\text{,}\) and \(\nu(B) = \nu(X)\text{.}\)
    🔗
    🔗

    Example 4.1.8.

    Let \(m\) be the Lebesgue measure on the Borel sets of \([0,1]\text{,}\) and let \(\mu\) be the Borel measure on the Borel sets of \([0,1]\) whose cumulative distribution function is the Cantor-Lebesgue function (see Corollary 3.3.10). It follows from Exercise 3.4.11 that \(m\) and \(\mu\) are mutually singular.
    🔗
    🔗

    Theorem 4.1.9. The Jordan Decomposition Theorem.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) Then there are mutually orthogonal measures on \(\nu_+\) and \(\nu_-\) on \((X,\mathcal{F})\) such that \(\nu = \nu_+ - \nu_-\text{,}\) i.e.
    🔗
    \begin{equation*} \nu(E) = \nu_+(E) - \nu_-(E) \end{equation*}
    for all \(E \in \mathcal{F}\text{.}\)
    🔗
    Further, \(\nu_+\) and \(\nu_-\) are unique mutually orthogonal measures satisfying \(\nu = \nu_+ - \nu_-\text{.}\)
    🔗
    🔗

    Proof.

    Let \(X = A \cup B\) be a Hahn decomposition for \(\nu\text{,}\) with \(A\) positive and \(B\) negative. For each \(E \in \mathcal{F}\text{,}\) define
    🔗
    \begin{equation*} \begin{aligned} \nu_+(E) &= \nu(E\cap A) \text{ and }\\ \nu_-(E) &= -\nu(E \cap B). \end{aligned} \end{equation*}
    That \(\nu_+\) is a measure follows from Exercise 4.3.7. That \(\nu_-\) is a measure can be proved similarly. That \(\nu_+\) and \(\nu_-\) are mutually orthogonal follows from the fact that \(A\) and \(B\) are disjoint.
    🔗
    Suppose now that \(\mu_1\) and \(\mu_2\) are mutually orthogonal measures satisfying \(\nu = \mu_1 - \mu_2\text{.}\) Since \(\mu_1\) and \(\mu_2\) are mutually orthogonal, there are disjoint measurable sets \(C\) and \(D\) satisfying \(X = C \cup D\text{,}\) \(\mu_1(C) = \mu(X)\text{,}\) \(\mu_1(D) = 0\text{,}\) \(\mu_2(C) = 0\text{,}\) and \(\mu_2(D) = 0\text{.}\) We will show that \(C\) and \(D\) give a Hahn decomposition for \(\nu\text{.}\) That is, we will show that \(C\) is \(\nu\)-positive and \(D\) is \(\nu\)-negative. Take any \(E \subseteq C\text{.}\) Since \(E \subseteq C\text{,}\) \(\mu_2(C) = 0\text{,}\) thus
    🔗
    \begin{equation*} \nu(E) = \mu_1(E) - \mu_2(E) = \mu_1(E) \geq 0. \end{equation*}
    Similarly, if \(E \subseteq D\text{,}\) then \(\mu_1(E) =0\) and so
    🔗
    \begin{equation*} \nu(E) = \mu_1(E) - \mu_2(E) = 0 - \mu_2(E) \leq 0. \end{equation*}
    Since \(C\) and \(D\) are a Hahn decompoistion for \(\nu\text{,}\) and \(A\) and \(B\) are a Hahn decompoistion for \(\nu\text{,}\) it follows from Theorem 4.1.5 that \(A\backslash C \cup C \backslash A\) and \(B \backslash D \cup D \backslash B\) are null sets. Take any \(E \in \mathcal{F}\text{,}\) then
    🔗
    \begin{equation*} E \cap C = ([E\cap C] \backslash A) \cup ([E \cap C] \cap A) \end{equation*}
    and
    🔗
    \begin{equation*} E \cap A = ([E\cap A] \backslash C) \cup ([E \cap A] \cap C) \end{equation*}
    and so
    🔗
    \begin{equation} \nu(E\cap C) = \nu(E \cap A).\tag{4.1} \end{equation}
    Take any \(E \in \mathcal{F}\text{.}\) Then
    🔗
    \begin{equation*} \begin{aligned} \mu_1(E) &= \mu_1(E \cap C) \\ &= \mu_1(E \cap C) - \mu_2(E \cap C) \\ &= \nu(E \cap C)\\ &= \nu(E \cap A) \\ &= \nu_+(E), \end{aligned} \end{equation*}
    where the penultimate equality comes from (4.1). Hence \(\mu_1 = \nu_+\text{.}\) We can show that \(\mu_2 = \nu_-\) in a similar fashion.
    🔗
    🔗

    Definition 4.1.10.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) Let \(\nu_+\) and \(\nu_-\) be mutually orthogonal measures on \((X,\mathcal{F})\) satisfying
    🔗
    \begin{equation*} \nu = \nu_+ - \nu_. \end{equation*}
    We call \(\nu_+\) and \(\nu_-\) the Jordan decomposition of \(\nu\text{.}\)
    🔗
    We call the measure \(\nu_+\) the positive part of \(\nu\), and call the measure \(\nu_-\) the negative part of \(\nu\).
    🔗
    🔗

    Definition 4.1.11.

    Let \(\nu\) be a signed measure and let \(\nu = \nu_+ - \nu_-\) be the Jordan decomposition of \(\nu\text{.}\) The measure \(|\nu|\) defined by
    🔗
    \begin{equation*} |\nu|(E) = \nu_+(E) + \nu_-(E) \end{equation*}
    is called the measure of total variation of \(\nu\). The value \(|\nu|(X)\) is the total variation of \(\nu\text{.}\) When \(|\nu|(X)\) is finite then \(\nu\) is of bounded variation.
    🔗
    🔗

    Remark 4.1.12.

    Let \(\nu\) be a signed measure. Then \(|\nu|\text{,}\) measure of total variation of \(\nu\text{,}\) is indeed a measure by Exercise 4.3.1.
    🔗
    It is worth noting that, when \(\nu\) is a signed measure, in general it is not true that
    🔗
    \begin{equation*} |\nu|(E) = |\nu(E)| \end{equation*}
    for a \(\nu\)-measurable set. That is, the placement of the \(|\cdot|\) lines is important.
    🔗
    In general, there is no reason to expect the function \(E \mapsto |\nu(E)|\) to be a measure. Can you see why not?
    🔗
    🔗
    Note that \(|\nu|(X)\) is finite if and only if \(\nu\) is a finite signed measure. The following characterization of total variation will be useful in the sequel. We leave the proof to Exercise 4.3.8
    🔗

    Proposition 4.1.13.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\text{.}\) Then the total variation \(|\nu|(X)\) is given by
    🔗
    \begin{equation*} |\nu|(X) = \sup \left\{ \sum_{k=1}^n |\nu(E_k)| \colon \{E_k\}_{k=1}^n \text{ disjoint measurable sets} \right\}. \end{equation*}
    🔗
    We end this section with another characterization of \(|\nu|(X)\) for finite signed measures.
    🔗

    Proposition 4.1.14.

    Let \(\nu\) be a signed measure on a measurable space \((X,\mathcal{F})\) with finite total variation \(|\nu|(X)\text{.}\) Suppose \(\mu_1\) and \(\mu_2\) are measures on \((X,\mathcal{F})\) satisfying \(\nu = \mu_1 - \mu_2\text{.}\) Then \(|\nu|(X) \leq \mu_1(X) + \mu_2(X)\text{.}\)
    🔗
    Further, \(|\nu|(X) = \mu_1(X) + \mu_2(X)\) if and only if \(\mu_1 = \nu_+\) and \(\mu_2 = \nu_-\text{.}\)
    🔗
    🔗

    Proof.

    Let \(X = A \cup B\) be a Hahn decomposition for \(\nu\) so that
    🔗
    \begin{equation*} \nu(A) = \nu_+(A),\ \nu(B) = - \nu_+(B) \end{equation*}
    and
    🔗
    \begin{equation*} \nu_+(B) = 0 = \nu_-(A). \end{equation*}
    Take any measurable \(E \subseteq A\text{,}\) then
    🔗
    \begin{equation*} \begin{aligned} \nu_+(E) &= \nu(E) \\ &= \mu_1(E) - \mu_2(E) \\ &\leq \mu_1(E). \end{aligned} \end{equation*}
    Similarly, if \(F \subseteq B\) is measurable then
    🔗
    \begin{equation*} \begin{aligned} \nu_-(F) &= -\nu(F) \\ &= \mu_2(F) - \mu_1(F)\\ &\leq \mu_2(F). \end{aligned} \end{equation*}
    We have then that
    🔗
    \begin{equation*} \begin{aligned} |\nu|(X) &= \nu_+(X) + \nu_-(X) \\ &= \nu_+(A) + \nu_-(B) \\ &\leq \mu_1(A) + \mu_2(B) \\ &\leq \mu_1(X) + \mu_2(X). \end{aligned} \end{equation*}
    Now suppose that \(|\nu|(X) = \mu_1(X) + \mu_2(X)\text{.}\) If \(\mu_1(A)>\nu_+(A)\) or \(\mu_2(B) > \nu_-(B)\) then, by the same reasoning as earlier in this proof,
    🔗
    \begin{equation*} |\nu|(X)< \mu_1(X) + \mu_2(X). \end{equation*}
    Thus \(\mu_1(A) = \nu_+(A)\) and \(\mu_2(B) = \nu_+(B)\text{.}\)
    🔗
    Take any measurable \(E \subseteq A\text{.}\) We have shown that \(\mu_1(E) \geq \nu_+(E)\text{.}\) If \(\mu_1(E) > \nu_+(E)\) then
    🔗
    \begin{equation*} \mu_1(A) = \mu_1(E) + \mu_1(A\backslash E) > \nu_+(A). \end{equation*}
    As \(\mu_1(A) = \nu_+(A)\text{,}\) it follows that \(\mu_1(E) = \nu_+(E)\) for all measurable \(E \subseteq A\text{.}\)
    🔗
    Suppose \(E \in \mathcal{F}\) satisfies \(\mu_1(E) \neq \nu_+(E)\text{.}\) Then
    🔗
    \begin{equation*} \mu_1(E) = \mu_1(E \cap A) + \mu_1(E \cap B) = \nu_+(E) + \mu_1(E\cap B). \end{equation*}
    Hence, if \(\mu_1 \neq \nu_+\text{,}\) then \(\mu_1(B)>0\text{.}\)
    🔗
    We can similarly show that \(\mu_2(F) = \nu_-(F)\) for all measurable \(F \subseteq B\text{,}\) and that if \(\mu_2 \neq \nu_-\) then \(\mu_2(A)>0\text{.}\)
    🔗
    Thus, if \(\mu_1 \neq \nu_+\) or \(\mu_2 \neq \nu_-\text{,}\) we have
    🔗
    \begin{equation*} \begin{aligned} |\nu|(X) &= \mu_1(X) + \mu_2(x) \\ &= \mu_1(A) + \mu_1(B) + \mu_2(A) + \mu_2(B) \\ &> \nu_+(A) + \nu_-(B) \\ &= \nu_+(X) + \nu_-(X). \end{aligned} \end{equation*}
    This contradicts the definition of the total variation \(|\nu|(X)\text{.}\) Hence \(\mu_1 = \nu_+\) and \(\mu_2 = \nu_-\text{.}\)
