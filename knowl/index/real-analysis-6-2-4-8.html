Definition 1.2.3.

Let \((X,\mathcal{M},\mu)\) be a measure space, let \(\mathcal{F}\) be all finite almost everywhere measurable functions on \(\mathcal{F}\text{,}\) and let \(\sim\) be the equivalence relation of equivalence almost everywhere on \(\mathcal{F}\text{.}\) A function \(f \in \mathcal{F}\) is essentially bounded if there is a constant \(M \geq 0\) such that \(|f| \leq M\) almost everywhere. In this case, we call \(M\) an essential bound for \(f\text{.}\) Define \(L^\infty(X,\mu)\) as
🔗
\begin{equation*} L^\infty(X,\mu) = \{[f]\in \mathcal{F}/\sim \colon f \text{ essentially bounded}\}. \end{equation*}
We call \(L^\infty(X,\mu)\) the \(L^\infty\)-space for \((X,\mathcal{M},\mu).\)
