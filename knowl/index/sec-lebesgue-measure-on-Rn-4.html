Theorem 1.2.2. The Fubini-Tonelli Theorem on \(\mathbb{R}^n\).

Let \(k,l,n\) be non-negative integers with \(n = k + l\) and let \(f\) be a Lebesgue measurable function on \(\mathbb{R}^n\text{.}\) Then
\begin{equation*} \intd{\mathbb{R}^n}{|f|}{m_n} = \intd{\mathbb{R}^k}{\intd{\mathbb{R}^l}{|f|}{m_l}}{m_k} \intd{\mathbb{R}^l}{\intd{\mathbb{R}^k}{|f|}{m_k}}{m_l}. \end{equation*}
If any of these values are finite, then \(f\) is integrable with respect to \(m_n\) and
\begin{equation*} \intd{\mathbb{R}^n}{f}{m_n} = \intd{\mathbb{R}^k}{\intd{\mathbb{R}^l}{f}{m_l}}{m_k} \intd{\mathbb{R}^l}{\intd{\mathbb{R}^k}{f}{m_k}}{m_l}. \end{equation*}
