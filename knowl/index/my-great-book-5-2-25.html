Example 2.1.14.

Let \(X\) be any non-empty set and let \(\mathcal{P}(X)\) be the \(\sigma\)-algebra of all subsets of \(X\text{.}\) Define \(c \colon \mathcal{P}(X) \rightarrow [0,\infty]\) by
\begin{equation*} c(A) = \infty \end{equation*}
if \(A\) contains infinitely many elements, and \(c(A) = n\) if \(A\) has \(n\) elements, for \(n\) finite. Then \((X,\mathcal{P}(X),c)\) is a measure space. We call \(c\) the \emph{counting measure} on \(X\text{.}\)\index{measure!counting measure}
🔗
The measure space \((X,\mathcal{P}(X),c)\) is always complete; it is \(\sigma\)-finite if and only if \(X\) is countable; it is finite if and only if \(X\) is finite.
