Theorem 5.1.7. Outer and inner approximation of Lebesgue measure.

Let \(E \subseteq \mathbb{R}\text{.}\) The following are equivalent.
🔗
  1. \(E\) is Lebesgue measurable;
    🔗
    🔗
  2. for every \(\varepsilon >0\) there exists an open set \(U\) containing \(E\) such that \(m^*(U\backslash E) < \varepsilon\text{;}\)
    🔗
    🔗
  3. there is a \(G_\delta\)-set \(G\) contraing \(E\) such that \(m^*(G\backslash E) = 0\text{;}\)
    🔗
    🔗
  4. for every \(\varepsilon >0\) there exists a closed set \(C\) contained in \(E\) such that \(m^*(E\backslash C) < \varepsilon\text{;}\)
    🔗
    🔗
  5. there is an \(F_\sigma\)-set \(F\) contained in \(E\) such that \(m^*(E\backslash F) = 0\text{.}\)
    🔗
    🔗
