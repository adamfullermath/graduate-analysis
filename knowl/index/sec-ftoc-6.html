Definition 9.3.3.

Suppose \(f\) is an integrable function on \(\mathbb{R}\text{.}\) For each \(x \in \mathbb{R}\) and \(r >0\text{,}\) we define \(A_rf(x)\) as
🔗
\begin{equation*} A_rf(x) = \frac{1}{2r}\int_{B(r,x)} f\,dm. \end{equation*}
We will call \(A_rf\) the average value function for \(f\) with radius \(r\text{.}\)
🔗
Define the Hardy-Littlewood maximal function, \(Hf\) for \(f\) by
🔗
\begin{equation*} Hf(x) = \sup_{r>0}A_r|f|(x) = \sup_{r>0} \frac{1}{2r}\int_{\mathbb{R}}|f|\,dm. \end{equation*}
