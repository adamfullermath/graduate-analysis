Example 4.3.5.

Let \(\mathcal{S}\) be the collection of all open intervals in the real line. Let \(\mu\) be the length function on \(\mathcal{S}\text{,}\) i.e. \(\mu(I) = \ell(I)\) for all \(I \in \mathcal{S}\text{.}\) Let \(\mu^*\) be the outer measure induced by \(\mu\text{.}\) Then \(\mu^*(I) = \ell(I)\) for all intervals in \(\mathbb{R}\) (Exercised~\ref{exer: induced length outer measure}). The outer measure \(\mu^*\) is called the Lebesgue outer measure on \(\mathbb{R}\). We will see (much!) more on this later.
