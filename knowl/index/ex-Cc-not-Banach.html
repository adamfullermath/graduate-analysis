Example 1.1.12.

Let \(I \subseteq \mathbb{R}\) be an interval. Let \(C_c(I)\) denote all continuous real-valued functions on \(I\) which are compactly supported. That is, a continuous function \(f\) on \(I\) is in \(C_c(I)\) if there is a compact set \(K \subseteq I\) such that \(f(x) = 0\) for \(x \in I \backslash K\text{.}\) Then \(C_c(I)\) a vector space of functions with
