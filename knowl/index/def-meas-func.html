Definition 5.1.2.

Let \((X,\mathcal{F})\) be a measurable space, \(E \in \mathcal{M}\text{,}\) and \(f \colon E \rightarrow [-\infty,\infty]\) be an extended real-valued function. Then \(f\) is a measurable function with respect to \(\mu\) if \(f\) satisfies any of the equivalent conditions from Theorem 5.1.1. If there is no chance of confusion on what the \(\sigma\)-algebra is, we will simply say that \(f\) is a measurable function.
🔗
If \((X,\mathcal{F},\mu)\) is a measure space, and \(f\) is measurable with respect to \(\mathcal{F}\) we also say \(f\) is measurable with respect to \(\mu\text{,}\) or \(\mu\)-measurable.
🔗
If \(f\) is a measurable function with domain \(E\text{,}\) and there is a measurable set \(F \subseteq E\) such that \(\mu(F)<\infty\) and \(f(x) = 0\) for all \(x \in E \backslash F\text{,}\) then \(f\) is finitely supported.
