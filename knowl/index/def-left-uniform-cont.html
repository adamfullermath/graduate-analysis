Definition 2.1.10.

Let \(G\) be a topological group, let \(x \in G\text{,}\) and let\(f\) be a bounded real-valued function on \(G\text{.}\) Define the left translation of \(f\) by \(x\) as the function
\begin{equation*} (L_xf)(y) = f(x^{-1}y) \end{equation*}
and define the right translation of \(f\) by \(x\) as the function
\begin{equation*} (R_xf)(y) = f(yx). \end{equation*}
🔗
The function \(f\) is left uniformly continuous if, for all \(\varepsilon>0\) there is a neighborhood \(U\) of the identity such that
\begin{equation*} \|L_xf -f \|_\infty \lt\varepsilon \end{equation*}
for all \(x \in U.\)
🔗
The function \(f\) is right uniformly continuous if, for all \(\varepsilon>0\) there is a neighborhood \(U\) of the identity such that
\begin{equation*} \|R_xf -f \|_\infty \lt\varepsilon \end{equation*}
for all \(x \in U.\)
