We are now ready for the main theorem of this section: our main tool for constructing measures. The uniqueness in the following theorem is important and will be used frequently. Before stating the theorem we need one more definition. If \(\mathcal{S}\) is a collection of subsets of \(X\text{,}\) and \(\mu \colon \mathcal{S} \rightarrow [0,\infty]\) is a function, we say that \(\mu\) is \(\sigma\)-finiteif there is a countable collection \(\{E_k\}_{k=1}^\infty\) of sets in \(\mathcal{S}\) with \(\mu(E_k)< \infty\) for all \(k\text{,}\) and \(X = \bigcup_{k=1}^\infty E_k\text{.}\) Note that this definition is consistent with the definition of a \(\sigma\)-finite measure.
