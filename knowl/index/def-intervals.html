Definition 1.2.3.

Fix \(n \geq 1\text{.}\) A set \(I \subseteq \bR^n\) is an interval of \(\bR^n\) if there are intervals \(I_1,\ldots,I_n\subseteq \bR\) such that
\begin{equation*} I = I_1 \times \ldots \times I_n \end{equation*}
🔗
An interval \(I\subseteq\bR^n\) is called an open interval if each \(I_i\) is open for \(i=1,\ldots,n\text{.}\)
