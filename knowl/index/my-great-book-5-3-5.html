Theorem 2.2.2.

[Continuity of measure]\index{measure!continuity} Let \((X,\mathcal{M},\mu)\) be a measure space.
  1. If \(\{A_k\}_{k=1}^\infty\) is a countable collection of measurable sets satisfying \(A_k \subseteq A_{k+1}\) (i.e. \(\{A_k\}_k\) ascending measurable sets), then
    \begin{equation*} \mu\left(\bigcup_{k=1}^\infty A_k \right) = \lim_{k\rightarrow \infty} \mu(A_k). \end{equation*}

    🔗
  2. If \(\{B_k\}_{k=1}^\infty\) is a countable collection of measurable sets satisfying \(B_{k+1} \subseteq B_{k}\) (i.e. \(\{B_k\}_k\) descending measurable sets) and \(\mu(B_1)<\infty\text{,}\) then
    \begin{equation*} \mu\left(\bigcap_{k=1}^\infty B_k \right) = \lim_{k\rightarrow \infty} \mu(B_k). \end{equation*}
