Corollary 5.2.6. Ergoroff’s Theorem on \(\mathbb{R}\).

Let \(E \subseteq \mathbb{R}\) be a Lebesegue measurable set with \(m(E)< \infty\text{.}\) Let \((f_n)_n\) be a sequence of measurable functions on \(E\) which converge a.e. to a finite a.e. function \(f\text{.}\)
🔗
For each \(\varepsilon > 0\) there is a closed set \(F \subseteq E\) such that \(m(E \backslash F)< \varepsilon\) and \((f_n)_n\) converges uniformly to \(f\) on \(F\text{.}\)
