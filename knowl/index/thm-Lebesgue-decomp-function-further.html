Theorem 7.4.14.

If \(\nu\) is a Borel signed-measure on \([a,b]\) then there are mutually singular signed Borel measures \(\nu_d\text{,}\) \(\nu_{sc}\) and \(\nu_{ac}\) such that
🔗
  1. \(\nu = \nu_d + \nu_{sc} + \nu_{ac}\text{;}\)
    🔗
    🔗
  2. \(\nu_d\) is a discrete Borel signed-measure;
    🔗
    🔗
  3. \(\nu_{sc}\) is a singular continuous Borel signed-measure; and
    🔗
    🔗
  4. \(\nu_{ac}\) is an absolutely continuous Borel signed-measure.
