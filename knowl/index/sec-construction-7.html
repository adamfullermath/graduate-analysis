Definition 4.3.1.

Let \(X\) be a non-empty set. A function
🔗
\begin{equation*} \mu^* \colon \mathcal{P}(X) \rightarrow [0,\infty] \end{equation*}
is an outer measure on \(X\) if
🔗
  1. \(\mu^*(\emptyset) = 0\text{;}\)
    🔗
    🔗
  2. if \(\{E_k\}_{k=1}^\infty\) is a countable collection of subsets of \(X\text{,}\) and \(E \subseteq \bigcup_k E_k\) then
    \begin{equation*} \mu^*(E) \leq \sum_{k=1}^\infty \mu^*(E_k). \end{equation*}
    I.e. \(\mu^*\) is countably monotone.
    🔗
    🔗
    🔗
