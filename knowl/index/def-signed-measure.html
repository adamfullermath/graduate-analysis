Definition 4.1.1.

Let \((X,\mathcal{F})\) be a measurable space. A function
🔗
\begin{equation*} \nu \colon \mathcal{F} \rightarrow [-\infty,\infty] \end{equation*}
is a signed measure if
🔗
  1. \(\nu\) takes at most value from \(\{-\infty,\infty\}\text{;}\)
    🔗
    🔗
  2. \(\nu(\emptyset) = 0\text{;}\) and
    🔗
    🔗
  3. if \(\{E_k\}_{k=1}^\infty\) are disjoint sets in \(\mathcal{F}\text{,}\) then
    \begin{equation*} \nu(\bigcup_{k=1}^\infty E_k) = \sum_{k=1}^\infty \nu(E_k). \end{equation*}
    🔗
    🔗
The elements of \(\mathcal{F}\) are called the measurable sets, or \(\nu\)-measurable sets. A signed measure \(\nu\) on \((X,\mathcal{F})\) is finite if \(|\nu(X)|< \infty\text{.}\)
