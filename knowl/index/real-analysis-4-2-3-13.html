The collection of all subsets of a set \(X\) is called the the power set of \(X\) and is denoted by \(\mathcal{P}(X)\text{.}\) The notation \(2^X\) is also sometimes used for the power-set of \(X\text{.}\) Note that if \(X\) has \(n\) elements, then \(\mathcal{P}(X)\) has \(2^n\) elements.
