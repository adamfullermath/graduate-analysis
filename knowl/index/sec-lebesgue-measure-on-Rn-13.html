We will recall what the Borel subsets of \(\mathbb{R}^n\) are. A subset \(U \subseteq \mathbb{R}^n\) is open if given \(x \in U\text{,}\) there is a \(\varepsilon>0\) such that
\begin{equation*} \{ y \in \mathbb{R}^n \colon \|x-y\|_2 < \varepsilon \} \subseteq U. \end{equation*}
We are using the Euclidean/\(2\)-norm on \(\mathbb{R}^n\) here; this is the usual and more intuitive way to define open subsets of \(\mathbb{R}^n\text{.}\) However, the choice of the \(2\)-norm is immaterial. It follows from Theorem III.1.2.26 that the following statements are equivalent for a subset \(U \subseteq \mathbb{R}^n\text{:}\)
  1. \(U\) is an open subset of \(\mathbb{R}^n\text{;}\)
    🔗
  2. given \(x \in U\text{,}\) there is a \(\varepsilon>0\) such that \(\{ y \in \mathbb{R}^n \colon \|x-y\|_p < \varepsilon \} \subseteq U\) for some \(p\) with \(1\leq p \leq \infty\text{;}\)
    🔗
  3. given \(x \in U\text{,}\) for each \(p\text{,}\) \(1\leq p \leq \infty\text{,}\) there is an \(\varepsilon_p>0\) such that \(\{ y \in \mathbb{R}^n \colon \|x-y\|_p < \varepsilon_p \} \subseteq U\text{.}\)
    🔗
That is, to see if a set \(U \subseteq \mathbb{R}^n\) is open, it does not matter which \(p\)-norm we use. The Borel sets in \(\mathbb{R}^n\) are the members of the smallest \(\sigma\)-algebra of subsets of \(\mathbb{R}^n\) which contains all open subsets of \(\mathbb{R}^n\text{.}\) We denote the Borel sets in \(\mathbb{R}^n\) by \(\mathcal{B}(\mathbb{R}^n)\text{.}\) We leave the details of the following proposition as an exercise, Exercise 1.4.11.
