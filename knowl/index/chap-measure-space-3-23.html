Definition 2.1.11.

A measure space \((X,\mathcal{M},\mu)\) is \(\sigma\)-finite if there is a countable collection of sets \(\{A_n\}_{n=1}^\infty\) in \(\mathcal{M}\) such that
🔗
  1. \(\mu(A_n)< \infty\) for each \(n\text{;}\)
    🔗
    🔗
  2. \(\displaystyle X = \bigcup_n^\infty A_n.\)
    🔗
    🔗
If \(\mu(X) < \infty\) then \((X,\mathcal{M},\mu)\) is a finite measure space.
