Definition 2.2.1.

Let \(G\) be a locally compact Hausdorff group. A non-zero Radon measure \(\mu\) on \(G\) is a left Haar measure if for all Borel sets \(E\subseteq G\) and \(x \in G\)
\begin{equation*} \mu(xE) = \mu(E). \end{equation*}
Similarly, a Radon measure \(\mu\) on \(G\) is a right Haar measure if \(\mu(Ex)=\mu(E)\) for all Borel sets \(E \subseteq G\) and \(x \in G\text{.}\) If a measure \(\mu\) is both a left and a right Haar measure we will call \(\mu\) a Haar measure.
