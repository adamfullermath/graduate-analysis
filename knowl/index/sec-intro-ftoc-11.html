In Example 1.1.3, the function \(G\) is not differentiable at each \(x=1/n\text{.}\) Whilst this gives (countably) infinite points where \(G\) is not differentiable, each point where \(G\) is not differentiable is isolated (meaning if \(G\) is not differentiable at \(x\) then there is an interval around \(x\) in which \(G\) is differentiable at all \(y \neq x\)). In the early 1800s, it was widely believed (or suspected, at least) that all examples would satisfy this. That is, it was conjectured that if \(f\) was continuous on a closed interval \([a,b]\) then each point where \(f\) is not differentiable is isolated). André-Marie Ampère (1775–1836) went so far as to present a `proof’ of this statement.
