Definition 2.3.8.

Let \(X\) be a non-empty set and let \(\mathcal{S} \subseteq \mathcal{P}(X)\) be a non-empty collection of subsets of \(X\text{.}\) A function \(\mu \colon \mathcal{S} \rightarrow [0,\infty]\) is a premeasure if
🔗
  1. \(\mu\) is finitely additive, i.e. if \(\{E_k\}_{k=1}^n\) is a disjoint collection of sets in \(\mathcal{S}\) whose union is also in \(\mathcal{S}\text{,}\) then \(\mu\left(\bigcup_{k=1}^n E_k\right) = \sum_{k=1}^n \mu(E_k)\text{;}\)
    🔗
    🔗
  2. \(\mu\) is countably monotone, i.e. if \(E,\ \{E_k\}_{k=1}^\infty\) are in \(\mathcal{S}\) and \(E \subseteq \bigcup_k E_k\text{,}\) then \(\mu(E) \leq \sum_{k=1}^\infty \mu(E_k)\text{;}\)
    🔗
    🔗
  3. if \(\emptyset \in S\text{,}\) then \(\mu(\emptyset) =0\text{.}\)
