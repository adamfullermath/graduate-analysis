Definition 2.3.6.

Let \(X\) be a non-empty set and let \(\mu^*\) be an outer measure on \(X\text{.}\) A set \(E \subseteq X\) is measurable with respect to \(\mu^*\) if
🔗
\begin{equation*} \mu^*(A) = \mu^*(A\cap E) + \mu^*(A\cap E^c), \end{equation*}
for all \(A \subseteq X.\)
🔗
The term \(\mu^*\)-measurable is also used for sets which are measurable with respect to \(\mu^*\text{.}\)
