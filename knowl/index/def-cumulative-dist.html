Definition 3.3.1.

Let \(I\) be a closed interval in \(\mathbb{R}\text{.}\) Let \(\mu\) be a finite Borel measure on \(I\text{.}\) Define a function \(g_\mu \colon I \rightarrow \mathbb{R}\) by
🔗
\begin{equation*} g_\mu(x) = \mu(\{y \in I \colon y\leq x\}). \end{equation*}
The function \(g_\mu\) is called the cumulative distribution function for \(\mu\text{.}\)
