Example 1.1.11.

Let \(I \subseteq \mathbb{R}\) be an interval. A continuous function \(f \colon I\rightarrow \mathbb{R}\) vanishes at infinity if for all \(\varepsilon>0\) there is a closed interval \(K \subseteq I\) such that \(|f(x)|<\varepsilon\) for \(x \in I \backslash K\text{.}\) We denote the collection of all continuous functions on \(I\) that vanish at infinity by \(C_0(I)\text{.}\)
🔗
The space \(C_0(I)\) is a vector subspace of \(C_b(I)\text{,}\) thus \(C_0(I)\) is a normed vector space with the \(\sup\)-norm \(\|\cdot\|_\infty\text{.}\) With this norm \(C_0(I)\) is a Banach space, see Exercise 1.4.6.
