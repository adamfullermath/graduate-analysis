Theorem 2.3.2.

Let \(X\) be a nonempty set, let \(\mathcal{S} \subseteq \mathcal{P}(X)\) be a non-empty collection of subsets of \(X\text{,}\) and let \(\mu \colon \mathcal{S} \rightarrow [0,\infty]\) be any function. Define a function \(\mu^* \colon \mathcal{P}(X) \rightarrow [0,\infty]\) by
🔗
  1. \(\mu^*(\emptyset) = 0\text{;}\)
    🔗
    🔗
  2. if \(\emptyset\neq A\text{,}\) and there exists a countable collection \(\{E_k\}_{k=1}^\infty\) of sets in \(\mathcal{S}\) such that \(A \subseteq \bigcup_k E_k\text{,}\) then
    \begin{equation*} \mu^*(A) = \inf \left\{ \sum_{k=1}^\infty \mu(E_k) \colon \{E_k\}_{k=1}^\infty \subseteq \mathcal{S},\ A \subseteq \bigcup_{k=1}^\infty E_k \right\}; \end{equation*}
    🔗
    🔗
  3. if \(\emptyset \neq A\) and \(A\) can not be covered by a countably infinite family in \(\mathcal{S}\text{,}\) then \(\mu^*(A) = \infty.\)
    🔗
    🔗
Then \(\mu^*\) is an outer measure on \(X\text{.}\)
