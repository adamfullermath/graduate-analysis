Checkpoint 3.3.13.

Complete the proof of Theorem Theorem 3.1.7. That is, show that if \(E\) is measurable then for all \(\varepsilon >0\) there is an open set \(U\) containing \(E\) such that \(m(U\backslash E) < \varepsilon\text{.}\) (You can assume that the result holds when \(m(E)<\infty\text{;}\) the proof of that case is given in the proof of Theorem Theorem 3.1.7).
