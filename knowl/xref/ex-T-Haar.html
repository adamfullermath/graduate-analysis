Example 2.2.5.

Let \(\bT\) be the topological group described in Example 2.1.6. Note that if \(z\in \bT\) then there is a unique \(\theta \in [0,2\pi)\) such that \(z= e^{i\theta}\text{.}\) Define a function
\begin{align*} f \colon [0,2\pi) \amp\rightarrow \bT \\ \theta \amp \mapsto e^{i\theta}. \end{align*}
Let \(m\) be the Lebesgue measure on \([0,2\pi)\) and define a measure \(\mu\) on \(\bT\) by
\begin{equation*} \mu(B)= m(f^{-1}(B)). \end{equation*}
Then \(\mu\) is a Haar measure on \(\bT\text{,}\) Exercise 2.3.6
