Theorem 1.1.12. Tonelli’s Theorem.

Let \((X,\mathcal{M},\mu)\) and \((Y,\mathcal{N},\nu)\) be \(\sigma\)-finite measure spaces, with \(\nu\) a complete measure, and let \(f\) be a non-negative \((\mu\times\nu)\)-measurable function on \(X\times Y\text{.}\) Then, for almost all \(x\in X\text{,}\) the \(x\)-section \(f_x\) of \(f\) is \(\nu\)-measurable, and the (almost everywhere defined) function on \(X\)
\begin{equation*} x \mapsto \int_Y f_x\,d\nu \end{equation*}
is \(\mu\)-measurable. Further,
\begin{equation*} \int_{X \times Y}f\,d(\mu\times \nu) = \int_X \left( \int_Y f(x,y)\,d\nu(y)\right) \,d\mu(x). \end{equation*}

Proof.

Since, \((X,\mathcal{M},\mu)\) and \((Y,\mathcal{N},\nu)\) are \(\sigma\)-finite, the product measure \(\mu \times \nu\) is \(\sigma\)-finite on \(X\times Y\text{,}\) by Exercise 1.4.1. Thus, by the Simple Approximation Theorem, Theorem II.5.2.4, there is an increasing sequence of simple functions \((\phi_n)_n\) which converge pointwise to \(f\) and each \(\phi_n\) is supported in a set of finite measure. For each \(n\text{,}\) since \(\phi_n\) is a supported in a set of finite measure, \(\phi_n\) is integrable with respect to \(\mu\times \nu\text{.}\) Hence, we can apply Fubini’s Theorem to each function \(\phi_n\text{,}\) to get
\begin{equation*} \int_{X\times Y}{\phi_n}\,d{(\mu\times\nu)} = \int_{X}{\left(\intd{Y}\,d{\phi_n(x,y)}{\nu(y)}\right)}{\mu(x)}. \end{equation*}
The rest of the proof proceeds in a similar vein to that of Fubini’s Theorem.
