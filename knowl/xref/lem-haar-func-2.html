Lemma 2.2.12.

Let \(G\) be a locally compact Hausdorff group, \(f_1,f_2,g \in C_c(G)\) be non-zero positive functions. Then
  1. \(I_g(L_xf_1) = I_g(f_1) \) for all \(x\in G\text{;}\)
    🔗
  2. \(I_g(cf_1) = cI_g(f_1)\) for all scalars \(c \gt 0\text{;}\)
    🔗
  3. \(I_g(f_1 + f_2) \leq I_g(f_1) + I_g(f_2)\text{;}\)
    🔗
  4. \([f_0:f_1]^{-1} \leq I_g(f_1) \leq [f_1\colon f_0]\text{.}\)
    🔗
  5. if \(\varepsilon \gt 0\text{,}\) there exists an open neighborhood \(U\) of the identity such that
    \begin{equation*} I_g(f_1) + I_g(f_2) \leq I_g(f_1 + f_2) + \varepsilon \end{equation*}
    if the compact support of \(g\) is in \(U\text{;}\)
    🔗
🔗
🔗

Proof.

Note that, for all \(x \in G\)
\begin{equation*} f_1 \leq \sum_{i=1}^n c_i L_{x_i}g \end{equation*}
if and only if
\begin{equation*} L_xf_1 \leq \sum_{i=1}^n c_i L_{xx_i}g. \end{equation*}
It follows that \([L_xf_1 \colon g] = [f_1 \colon g]\text{,}\) and hence \(I_g(L_xf_1) = I_g(f_1).\)
🔗
Part 2 and Part 3 are (near) immediate, so we will turn our attention to Part 4. If there are positive scalars \(c_1,\ldots,c_n,d_1,\ldots,d_m \lt 0\text{,}\) and \(x_1,\ldots,x_n,y_1,\ldots,y_m \in G\) such that
\begin{equation*} f_1 \leq \sum_{i=1}^n c_i L_{x_i}f_0 \text{ and }f_0 \leq \sum_{j=1}^m d_j L_{y_j}g, \end{equation*}
then
\begin{equation*} f_1 \leq \sum_{i=1}^n \sum_{j=1}^m c_id_j L_{x_iy_j}g. \end{equation*}
Thus \([f_1 \colon g] \leq [f_1 \colon f_0] [f_0\colon g],\) and so \(I_g(f) \leq [f_1\colon f_0].\) Similarly, \([f_0\colon g] \leq [f_0\colon f_1][f_1 \colon g],\) and so \(I_g(f_1) \geq [f_0\colon f_1]^{-1}\text{,}\) completing Part 4.
🔗
We will now prove Part 5 Let \(K\) be the compact support of \(f_1 + f_2\text{,}\) and let \(k\in C_c(G)\) be a positive function such that \(k(x) =1 \) for all \(x \in K.\) Note that such a \(k\) exists by Urysohn’s Lemma. Take \(\delta \gt 0\) and let \(h = f_1 + f_2 + \delta k.\) For \(i = 1,2\text{,}\) let \(h_i = f_i/h\text{.}\) By Proposition 2.1.12 \(h_1\) and \(h_2\) are right uniformly continuous. Thus, there is an open neighborhood \(U\) of the identity such that \(\|R_x h_i - h_i\|_\infty \lt \delta\) for all \(x \in U.\) Thus, if \(y^{-1}x \in U\) then \(|h_i(x) - h(y)|\lt\delta.\)
🔗
Assume the the compact support of \(g\) is in \(U\text{.}\) Take scalars \(c_1,\ldots,c_n \gt 0\) and \(x_1,\ldots,x_n \in G\) such that \(h \leq \sum_{j=1}^n c_j L_{x_j}g.\) Then, for \(i = 1,2\text{,}\) for \(x \in G\)
\begin{equation*} f_i(x) = h(x)h_i(x) \leq \sum_{j=1}^n c_j g(x_j^{-1}x) h_i(x). \end{equation*}
Note that, if \(g(x_j^{-1}x)\neq 0 \text{,}\) then \(x_j^{-1}x \in U\text{,}\) and hence \(|h_i(x_j)-h_i(x)|\lt \delta\text{,}\) for \(i = 1,2.\) Thus,
\begin{align*} f_i(x) \amp \leq \sum_{j=1}^n c_j g(x_j^{-1}x) h_i(x) \\ \amp \sum_{j=1}^n c_j g(x_j^{-1}x)(h_i(x_j) + \delta). \end{align*}
Hence \([f_i\colon g] \leq \sum_{j=1}^n c_j(h_i(x_j)+\delta)\text{,}\) for \(i=1,2\text{.}\) Since, \(h_1 + h_2 \leq 1\) it follows that
\begin{equation*} [f_1\colon g] + [f_2\colon g] \leq \sum_{j=1}^n c_j(1 + 2 \delta). \end{equation*}
It follows now that
\begin{equation*} I_g(f_1) + I_g(f_2)\leq I_g(h)(1+2\delta) \leq (I_g(f_1+f_2) + \delta I_g(k))(1+2\delta), \end{equation*}
where the last inequality is an application of Part 2 and Part 3. By Part 4, the result follows if
\begin{equation*} 2\delta[f_1+f_2\colon f_0] + \delta(1+\delta)[k\colon f_0]\lt\varepsilon \end{equation*}
