Proposition 4.2.1.

Let \((X,\mathcal{M},\mu)\) be a measure space. Then the following hold
🔗
  1. (Monotonicity) If \(A, B \in \mathcal{M}\) with \(B \subseteq A\text{,}\) then \(\mu(B)\leq \mu(A)\text{;}\)
    🔗
    🔗
  2. (Excision) If \(A,B \in \mathcal{M}\) with \(B\subseteq A\) and \(\mu(A)<\infty\text{,}\) then \(\mu(A \backslash B) = \mu(A) - \mu(B)\text{;}\)
    🔗
    🔗
  3. (Countable monotonicity) If \(\{E_k\}_{k=1}^\infty\) is a countable collection of measurable sets, and \(E\) is a measurable set such that \(E \subseteq \bigcup_{k=1}^\infty E_k\text{,}\) then \(\mu(E) \leq \sum_{k=1}^\infty \mu(E_k).\)
