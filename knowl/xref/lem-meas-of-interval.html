Lemma 1.3.3.

Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map. Then
\begin{equation*} m_n^*(T(E)) \leq |\det(T)| m_n^*(E) \end{equation*}
for all \(E \subseteq \mathbb{R}^n\text{.}\)
🔗
🔗

Proof.

Let \(I = I_1 \times \ldots \times I_n\) be an interval in \(\mathbb{R}^n\text{.}\) If \(T\) is of Type 1 for some non-zero constant \(c\text{,}\) then \(T(I)\) is an interval and \(m_n(T(I)) = |c| m_n(I) = |\det(T)| m_n(I)\text{.}\) Similarly, if \(T\) is of Type 3, then \(T(I)\) is an interval and \(m_n(T(I)) = m_n(I) = |\det(T)| m_n(I)\text{.}\)
🔗
Suppose \(T\) is of Type 2 and \(n=2\text{.}\) As \(T\) is linear, we can assume that \(I = [0,a]\times[0,b]\) for some \(a,b\geq 0\text{.}\) Then
\begin{equation*} T(I) = \{ (x,y) \colon 0\leq x \leq a,\ ax \leq y \leq ax + b\} \end{equation*}
is a parallelogram in \(\bR^2\text{.}\) By Exercise 1.4.8, \(T(I)\) is measurable and
\begin{equation*} m_2(T(I)) =ab = m_2(I). \end{equation*}
Note that, if \(n > 2\) and \(T\) is of Type 2, then \(T\) only changes on the \(i\) and \(i+1\) coordinates of an \(x \in \bR^n\text{,}\) and leaves all other coordinates fixed. Thus, by the \(n = 2\) case, \(T(I)\) is measurable and \(m_n(T(I)) = m_n(I) = |\det(T)| m_n(I)\text{.}\)
🔗
Suppose now, that \(T\) is of Type 1, Type 2, or Type 3. Take any \(E \subseteq \bR^n\) with \(m_n^*(E)<\infty\text{.}\) Let \(\{I_k\}_{k=1}^\infty\) be a countable collection of intervals in \(\bR^n\) such that \(E \subseteq \bigcup_{k=1}^\infty I_k\text{.}\) Then
\begin{align*} m_n^*(T(E)) \amp \leq \sum_{k=1}^\infty m_n(T(I_k)) \\ \amp = |\det(T)| \sum_{k=1}^\infty m_n(I_k). \end{align*}
Taking an infimum over all such collections of intervals, we have that \(m_n^*(T(E)) \leq |\det(T)| m_n^*(E)\text{.}\)
🔗
Now suppose that \(T\) is an arbitrary invertible linear map on \(\bR^n\text{.}\) Then \(T = T_1\ldots T_k\) where each \(T_i\) is of Type 1, Type 2, or Type 3. Thus, for \(E \subseteq \bR^n\)
\begin{align*} m_n^*(T(E)) \amp \leq |\det(T_1)| m_n^*(T_2T_3\ldots T_k (E)) \\ \amp \leq |\det(T_1)| \ldots|det(T_k)| m_n^*(E) \\ \amp = |\det(T)| m_n^*(E), \end{align*}
completing the proof.
