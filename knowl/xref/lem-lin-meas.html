Lemma 1.3.4.

Let \(T \colon \mathbb{R}^n \rightarrow \mathbb{R}^n\) be an invertible linear map. Then, if \(E \subseteq \bR^n\) is Lebesgue measurable, \(T(E)\) is Lebesgue measurable.
🔗
🔗

Proof.

Let \(E \subseteq \mathbb{R}\) be a bounded Lebesgue measurable set. By Corollary 1.2.9 there is an \(F_\sigma\)-set \(F \subseteq E\) such that \(m_n(E) = m_n(F) = 0\text{.}\) Let \(B = E\backslash F\text{.}\) Then \(m_n(B) = 0\text{.}\)
🔗
Note that, since \(T\) is linear, \(T^{-1}\) is linear, and therefore continuous. Hence \(T\) sends closed sets to closed sets, and hence \(F_\sigma\) sets to \(F_\sigma\) sets. Further, since \(T\) is continuous, there is an \(M \geq 0\text{,}\) such that \(\|T(x)\|_2 \leq M \|x\|_2\text{.}\) Hence, \(T\) sends bounded sets to bounded sets. Hence \(T(F)\) is a bounded \(F_\sigma\) set. In particular, \(T(F)\) is Lebesgue measurable.
🔗
By Lemma 1.3.3,
\begin{equation*} m_n^*(T(B))\leq |\det(T)|m_n(B) = 0. \end{equation*}
Hence, \(m_n^*(T(B))=0\text{,}\) and hence \(T(B)\) is Lebesgue measurable. Thus,
\begin{equation*} T(E) = T(F) \cup T(B) \end{equation*}
is Lebesgue measurable. As any measurable set is a countable union of bounded measurable sets, the result follows for all Lebesgue measurable sets.
