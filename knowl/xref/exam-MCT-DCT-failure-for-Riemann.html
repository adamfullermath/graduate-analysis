Example 6.3.5.

Let \(X = [0,1]\) and let \(A = \mathbb{Q} \cap [0,1]\text{.}\) The set \(A\) is countable and so can be enumerated as
🔗
\begin{equation*} A = \{a_1,a_2,\ldots\} \end{equation*}
For each \(n\) let
🔗
\begin{equation*} A_n = \{a_1,\ldots,a_n\}, \end{equation*}
and let \(f_n = \chi_{A_n}\text{.}\) Then \((f_n)_n\) is an increasing sequence of non-negative Riemann-integrable functions which converge pointwise to \(\chi_A\text{.}\) The Riemann integral of each \(f_n\) is \(0\text{.}\) However, as noted in Example 1.1.7, \(\chi_A\) is not Riemann integrable.
🔗
We do not have
🔗
\begin{equation*} \lim_n \mathrm{R}{\int}_0^1f_n = \mathrm{R}{\int}_0^1 \chi_A \end{equation*}
since \(\mathrm{R}{\int}_0^1 \chi_A\) is not defined. Of course, using Lebesgue integration, we get
🔗
\begin{equation*} \lim_n \int_{[0,1]}f_n \,dm = \int_{[0,1]} \chi_A \,dm=0. \end{equation*}
Indeed this is an easy calculation to do directly. Alternatively it follows from either the Monotone Convergence Theorem or the Lebesgue Dominated Convergence Theorem.
