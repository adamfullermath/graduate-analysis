Definition 4.1.9.

Let \((X,\mathcal{M})\) be a measure space. A function \(\mu \colon \mathcal{M} \rightarrow [0,\infty]\) is a measure on \((X,\mathcal{M})\) if
🔗
  1. \(\mu(\emptyset) = 0\text{;}\) and
    🔗
    🔗
  2. if \(\{A_n\}_{n=1}^\infty\) is a countable collection of disjoint sets in \(\mathcal{M}\) then
    \begin{equation*} \mu\left( \bigcup_{n=1}^\infty A_n \right) = \sum_{n=1}^\infty \mu(A_n). \end{equation*}
    🔗
    🔗
If \(\mu\) is a measure on a measurable space \((X,\mathcal{M})\text{,}\) we call the triple \((X,\mathcal{M},\mu)\) a measure space. The sets \(A \in \mathcal{M}\) are called measurable sets in the measure space \((X,\mathcal{M},\mu)\text{.}\)
