Corollary 1.2.6.

For \(E \subseteq \mathbb{R}^n\) the following are equivalent.
🔗
  1. \(E\) is Lebesgue measurable;
    🔗
    🔗
  2. there is a \(G_\delta\) set \(G \subseteq \mathbb{R}^n\) such that \(E \subset G\) and \(m_n^*(G\backslash E)=0\text{;}\)
    🔗
    🔗
  3. there is a \(F_\sigma\) set \(F \subseteq \mathbb{R}^n\) such that \(F \subseteq E\) and \(m_n^*(E \backslash F)=0.\)
    🔗
    🔗
🔗
