Proposition 5.1.4.

Let \((X,\mathcal{F})\) be a measurable space.
🔗
If \(f\) is an extended real-valued measurable function on \(X\text{,}\) then \(f^{-1}(U)\) is measurable for all open \(U \subseteq \mathbb{R}\text{.}\)
🔗
If \(f\) is a real-valued function and \(f^{-1}(U)\) is measurable for all open \(U \subseteq \mathbb{R}\text{,}\) then \(f\) is a measurable function.
