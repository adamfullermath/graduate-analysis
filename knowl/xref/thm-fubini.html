Theorem 1.1.10. Fubini’s Theorem.

Let \((X,\mathcal{M},\mu)\) be a measure space, let \((Y,\mathcal{N},\nu)\) be a complete measure space and let \(f\) be a measurable function \(X\times Y\) which is integrable with respect to \(\mu\times \nu\text{.}\)
🔗
Then for almost all \(x\in X\text{,}\) the \(x\)-section \(f_x\) of \(f\) is integrable with respect to \(\nu\) and
\begin{equation*} \int_{X\times Y} f\,d(\mu\times\nu) = \int_X\left(\int_Y f(x,y)\,d\nu(y)\right)d\mu(x). \end{equation*}
🔗
🔗

Proof.

First suppose that \(f =\chi_E\) where \(E\subseteq X\times Y\) is a measurable set with \((\mu\times\nu)(E)<\infty\text{.}\) Note that, for each \(x \in X\text{,}\) the \(x\)-section \(f_x\) is given by
\begin{equation*} f_x = \chi_{E_x}. \end{equation*}
By Lemma 1.1.9, \(E_x\) is measurable for almost all \(x\in X\text{,}\) hence \(f_x\) is measurable for almost all \(x\in X\text{.}\) Further, again by Lemma 1.1.9,
\begin{equation*} (\mu\times\nu)(E) = \int_X \nu(E_x)\,d\mu(x) <\infty. \end{equation*}
Hence \(\nu(E_x)<\infty\) for almost all \(x\in X\text{.}\) Thus, for almost all \(x\in X\text{,}\)
\begin{equation*} \infty > \nu(E_x) = \int_Y f_x\,d\mu. \end{equation*}
Thus \(f_x\) is integrable for almost all \(x\text{.}\) We have now that
\begin{equation*} \begin{aligned} \int_{X\times Y} f\,d(\mu\times \nu)& = (\mu\times\nu)(E)\\ & = \int_X \nu(E_x)\,d\mu(x)\\& = \int_X\left(\int_Y f(x,y)\,d\nu(y)\right)d\mu(x). \end{aligned} \end{equation*}
By the linearity of integration, the result holds when \(f\) is an integrable simple function on \(X\times Y\text{.}\)
🔗
Now suppose that \(f\) is a non-negative integrable function on \(X \times Y\text{.}\) Let \((\phi_n)_n\) be an increasing sequence of simple functions which converge pointwise to \(f\text{.}\) By the above arguments, and the Monotone Convergence Theorem, it follows that
\begin{equation*} \begin{aligned} \lim_{n\rightarrow\infty}\int_X\left(\int_Y \phi_n(x,y)\,d\nu(y)\right)\,d\mu(x) &= \lim_{n\rightarrow\infty}\int_{X\times Y} \phi_n \,d(\mu\times \nu)\\ & = \int_{X\times Y} f\,d(\mu\times\nu). \end{aligned} \end{equation*}
🔗
To complete the proof we will show that
\begin{equation*} \lim_{n\rightarrow\infty}\int_X\left(\int_Y \phi_n(x,y)\,d\nu(y)\right)\,d\mu(x) = \int_X\left(\int_Y f(x,y)\,d\nu(y)\right)d\mu(x). \end{equation*}
By removing a set of measure zero if necessary, we can assume that the \(x\)-sections \((\phi_n)_x\) are integrable for all \(x\in X\) for all \(n\text{.}\) Thus, for each \(x\in X\text{,}\) \(((\phi_n)_x)_n\) is an increasing sequence of measurable functions, converging pointwise to \(f_x\text{.}\) Thus \(f_x\) is measurable for each \(x\in X\text{.}\) For each \(n\text{,}\) let define a function \(h_n\) on \(X\) by
\begin{equation*} h_n(x) = \int_Y (\phi_n)_x\,d\nu, \end{equation*}
and define a function \(h\) on \(X\) by
\begin{equation*} h(x) = \int_Y f_x\,d\nu. \end{equation*}
Since each \(\phi_n\) is simple, it follows that \(h_n\) is integrable for all \(n\text{.}\) Thus sequence \((h_n)_n\) is an increasing sequence of non-negative measurable functions. By the Monotone Covergenece Theorem applied to \(((\phi_n)_x)_n\text{,}\) \((h_n)_n\) converges pointwise to \(h\text{.}\) Applying the Monotone Convergence Theorem to the sequence \((h_n)_n\text{,}\) we get
\begin{equation*} \lim_{n\rightarrow\infty}\int_X h_n \,d\mu = \int_X h\,d\mu; \end{equation*}
i.e.
\begin{equation*} \lim_{n\rightarrow\infty} \int_X \left(\int_Y \phi_n(x,y)\,d\nu(y) \right)\,d\mu(x) = \int_X \left(\int_Y f(x,y)\,d\nu(y) \right)\,d\mu(x). \end{equation*}
🔗
If \(f\) is an arbitrary integrable function on \(X\times Y\text{,}\) then \(f = f_+ - f_-\) where \(f_+\) is the positive part of \(f\text{,}\) and \(f_-\) is the negative part of \(f\text{.}\) Since the result holds for \(f_+\) and \(f_-\text{,}\) the result holds for \(f\) by the linearity of integration.
