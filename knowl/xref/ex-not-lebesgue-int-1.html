Example 8.3.6.

Define a function \(f(x)\) on \([0,\infty]\) by
🔗
\begin{equation*} f(x) = \frac{(-1)^n}{n} \end{equation*}
when \(n = \lceil x\rceil\text{.}\)
Then
 1 
The exact value this alternating series takes can be shown to be \(\ln(2)\text{,}\) but that is finite is enough for us here.
🔗
🔗
\begin{equation*} \mathrm{R}{\int}_{-\infty}^\infty f = \sum_{n=1}^\infty \frac{(-1)^n}{n} < \infty \end{equation*}
However
🔗
\begin{equation*} \int_{[0,\infty]} |f|\,dm = \sum_{n=1}^\infty \frac{1}{n} =\infty, \end{equation*}
and hence \(f\) is not Lebesgue integrable.
