Example 4.2.4.

If \(f \colon [a,b] \rightarrow \mathbb{R}\) is monotone (i.e. \(f\) is increasing or decreasing), then \(f\) is of bounded variation. Indeed, if \(f\) is increasing and \(P = \{a= x_0 < \ldots < x_n = b\}\) is any partition of \([a,b]\) then
🔗
\begin{equation*} V(f,P) = \sum_{k=1}^\infty |f(x_i) -f(x_{i-1})| = f(b) - f(a). \end{equation*}
Thus, \(TV(f) = f(b)-f(a)\text{.}\) If \(f\) is decreasing then we can show that
🔗
\begin{equation*} TV(f) = f(a) - f(b). \end{equation*}
