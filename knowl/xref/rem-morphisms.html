Remark 7.1.16.

We have only considered extended real-valued measurable functions. This is what we will be interested in for the majority of these notes. In more generality, if \((X,\mathcal{M})\) and \((Y,\mathcal{N})\) are two measurable spaces, a function \(f \colon X \rightarrow Y\) is measurable with respect to \(\mathcal{M}\) and \(\mathcal{N}\) if \(f^{-1}(E) \in \mathcal{M}\) for all \(E \in \mathcal{N}\text{.}\) This definition is consistent with Definition Definition 7.1.2 when \(Y = \mathbb{R}\cup\{-\infty,\infty\}\) and \(\mathcal{N} = \mathcal{B}(\mathbb{R} \cup \{-\infty,\infty\})\text{,}\) the Borel sets in the extended real line. See Exercise Exercise 4.4.5 for more on \(\mathcal{B}(\mathbb{R} \cup \{-\infty,\infty\})\text{.}\) This fact almost follows from Proposition Proposition 7.1.4. The full details are left to Exercise Exercise 7.3.12.
🔗
When we study measurable functions from \(\mathbb{R}\) to \(\mathbb{R}\text{,}\) we study measurable functions from with respect to \(\mathcal{M}\) and \(\mathcal{B}\text{,}\) \(\mathcal{M}\) is the Lebesgue measurable sets on \(\mathbb{R}\) and \(\mathcal{B}\) is the Borel sets on \(\mathbb{R}\text{.}\) I.e. we study measurable functions from the measure space \((\mathbb{R},\mathcal{M},m)\) to the measure space \((\mathbb{R},\mathcal{B},m)\text{.}\) The lopsidedness of this naturally raises questions. Firstly, the advantage of the domain being \((\mathbb{R},\mathcal{M},m)\) is that \((\mathbb{R},\mathcal{M},m)\) is a complete measure space. See, e.g., Proposition Proposition 7.1.6 for an advantage of having a complete measure space in the domain.
🔗
One reason for taking the Borel sets in the range, instead of all Lebesgue measurable sets, is the following. As described in the proof of Lemma Lemma 5.2.7 and Example Example 7.1.14, there is a continuous function \(f \colon \mathbb{R} \rightarrow \mathbb{R}\) and a set \(B \subseteq \mathbb{R}\) which is not Lebesgue measurable, such that \(A=f(B)\) is Lebesgue measurable but \(A\) is not a Borel set, and \(f^{-1}(A) = B\text{.}\) I.e. there is a continuous function \(f\) and a Lebesgue measurable set \(A\) such that \(f^{-1}(A)\) is not Lebesgue measurable. In conclusion, the measurable functions from \((\mathbb{R},\mathcal{M})\) to \((\mathbb{R},\mathcal{M})\) do not include all continuous functions. We do, however, want to study the continuous functions. Hence the measurable functions from \((\mathbb{R},\mathcal{M})\) to \((\mathbb{R},\mathcal{M})\) is not the right class to study.
