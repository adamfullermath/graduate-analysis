Example 1.1.3.

Denote by \(C_b(\mathbb{R})\) all continuous, bounded functions on \(\mathbb{R}\text{.}\) Then \(C_b(\mathbb{R})\) is a vector space. Define \(\|\cdot\|_\infty\) on \(C_b(\mathbb{R})\) by
🔗
\begin{equation*} \|f\|_\infty = \sup_{x \in \mathbb{R}} |f(x)|. \end{equation*}
Then \(\|\cdot\|_\infty\) defines a norm on \(C_b(\mathbb{R})\) called the infinity norm or the \(\sup\)-norm.
🔗
More generally, if \(X\) is a topological space, the \(C_b(X)\text{,}\) the space of all continuous bounded functions on \(\mathbb{R}\text{,}\) is a vector space and
🔗
\begin{equation*} \|f\|_\infty = \sup_{x\in X}|f(x)| \end{equation*}
defines a norm on \(C_b(X)\text{.}\)
🔗
Denote by \(C(X)\) all continuous functions on a topological space \(X\text{.}\) In general, \(\|\cdot\|_\infty\) will not define a norm on \(C(X)\) (since we could hace \(\sup_{x\in X}|f(x)|=\infty\) for some \(f \in C(X)\)). If \(X\) is compact then all continuous functions on \(X\) are bounded. In this case \(C(X) = C_b(X)\text{.}\)
