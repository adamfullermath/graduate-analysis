Example 11.4.2.

Let \(X\) be a locally compact Hausdorff space with Borel sets \(\mathcal{B}(X)\text{,}\) and let \(\mu\) be a finite Radon measure on \((X,\mathcal{B}(X))\text{.}\) By Corollary Corollary 7.1.5, every continuous real-valued function \(f\) on \(X\) is a measurable function on \((X,\mathcal{B}(X))\text{.}\) Since \(\mu\) is a finite measure, any bounded measurable function on \(X\) is integrable. Thus, if \(f \in C_0(X)\text{,}\) then \(f\) is integrable. Define a linear map from \(C_0(X)\) to \(\mathbb{R}\text{,}\) by
🔗
\begin{equation*} \phi_\mu(f) = \int_X f\,d\mu. \end{equation*}
Note that, if \(f \in C_0(X)\text{,}\) then
🔗
\begin{equation*} |\phi_\mu(f)| = \left|\int_X f\,d\mu\right| \leq \|f\|_\infty \mu(X). \end{equation*}
Hence, \(\phi_\mu\) is a linear functional on \(C_0(X)\) and \(\|\phi_\mu\|_\mathrm{op} \leq \mu(X)\text{.}\)
🔗
Since \(\mu\) is a Radon measure, there is an increasing sequence \(\{K_n\}_n\) of compact subsets of \(X\) such that
🔗
\begin{equation*} \mu(X) = \lim_{n\rightarrow\infty}\mu(K_n). \end{equation*}
By Urysohn’s Lemma, for each \(n\) there is a compactly supported positive function \(f_n\) on \(X\) such that
🔗
\begin{equation*} f_n(x) = 1 \text{ for all }x\in K_n \text{ and }\|f_n\|_\infty = 1. \end{equation*}
For each \(n\text{,}\)
🔗
\begin{equation*} |\phi_\mu(f_n)| =\int_X f_n\,d\mu \geq \int_{K_n} f_n\,d\mu = \mu(K_n). \end{equation*}
Hence, for each \(n\text{,}\)
🔗
\begin{equation*} \|\phi_\mu\|_\mathrm{op} = \sup\{|\phi_\mu(f)|\colon f\in C_0(X),\ \|f\|_\infty = 1\} \geq \mu(K_n). \end{equation*}
Thus
🔗
\begin{equation*} \|\phi_\mu\|_\mathrm{op} = \mu(X). \end{equation*}
