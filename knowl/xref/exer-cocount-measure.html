Exercise 1.

Let \(X\) be an uncountable set. A set \(A \subseteq X\) is cocountable if its complement \(A^c\) is countable.
🔗
  1. Show that the collection \(\mathcal{M}\) of all subsets of \(X\) which are either countable or cocountable forms a \(\sigma\)-algebra.
    🔗
    🔗
  2. Define a function \(\mu \colon \mathcal{M} \rightarrow \mathbb{R}\) by \(\mu(A) = 0\) if \(A\) is countable, and \(\mu(A) = 1\) if \(A\) is cocountable. Show that \(\mu\) is a measure.
