Example 2.2.4.

Let \(G\) be the group \(\bR\backslash \{0\}\) as in Example 2.1.5. Define a measure \(\mu\) on the Borel sets in \(G\) by
\begin{equation*} \mu(B) = \int_B \frac{1}{|x|}dm(x) \end{equation*}
for each Borel set \(B \subseteq \bR\backslash\{0\}\text{,}\) where \(m\) is the Lebesgue measure. The measure \(\mu\) is a Haar measure, see Exercise 2.3.5.
