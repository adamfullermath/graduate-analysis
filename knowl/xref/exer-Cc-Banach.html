Exercise 10.

Let \(I \subseteq \mathbb{R}\) be an interval and let \(C_c(I)\) be the collection of all continuous real-valued compactly supported functions on \(I\text{.}\)
🔗
Show that \(C_c(I)\) is a Banach space if and only if \(I\) is a closed bounded interval, in which case
🔗
\begin{equation*} C_c(I) = C_0(I) = C_b(I). \end{equation*}
