Example 5.1.14.

In the proof of Lemma 3.2.7, we prove the existence of a non-Lebesgue measurable set \(B \subseteq [0,2]\) and a bijective continuous function \(f\colon [0,2] \rightarrow [0,1]\) such that \(f(B)\) is Lebesgue measurable. By a simple adjustment, we can assume that \(f(B) \subseteq (0,1).\) We extend \(f\) to a continuous function on all of \(\mathbb{R}\) by setting \(f(x) = 1\) for \(x \geq 2\) and \(f(x) = 0\) for \(x \leq 0\text{.}\) Let \(A = f(B)\text{.}\)
🔗
Consider the measure space \((\mathbb{R},\mathcal{M})\text{,}\) where \(\mathcal{M}\) is all Lebesgue measurable sets in \(\mathbb{R}\text{.}\) The function \(\chi_A\) is a measurable function since \(A\) is measurable, Exercise 5.3.3, and \(f\) is measurable since it is continuous, Corollary 5.1.5. However the function \(g = \chi_A \circ f\) is not measurable. Indeed
🔗
\begin{equation*} \begin{aligned} \{x \in \mathbb{R} \colon g(x) > 0 \} &= \{x \in \mathbb{R} \colon \chi_A(f(x)) >0 \} \\ &= \{x \in \mathbb{R} \colon f(x) \in A\} \\ &= B, \end{aligned} \end{equation*}
is not Lebesgue measurable, and hence \(g\) is not a measurable function.
