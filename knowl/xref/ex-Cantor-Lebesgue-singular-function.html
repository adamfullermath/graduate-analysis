Example 7.4.2.

Let \(\varphi\) be the Cantor-Lebesgue function on \([0,1]\) (see Example 3.2.5), let \(C\) be the Cantor set in \([0,1]\) and let \(U = [0,1]\backslash U.\) By the definition of \(\varphi\text{,}\) there are disjoint open intervals \(\{I_k\}_{k=1}^\infty\) such that \(U = \bigcup_{k=1}^\infty I_k\) such that \(\varphi\) is constant on each \(I_k\text{.}\) It follows that \(\varphi'(x)= 0\) for all \(x \in U\text{.}\) By Example 3.2.3, \(m(U) = 1 =m([0,1])\text{.}\) Hence \(\varphi'=0\) a.e., and so \(\varphi\) is a continuous singular function.
🔗
Recall that, by Example 7.1.2, if \(\nu\) is the Borel measure on \([0,1]\) satisfying \(\varphi((c,d]) = \varphi(d) - \varphi(c)\text{,}\) then \(\nu\) is a singular measure.
