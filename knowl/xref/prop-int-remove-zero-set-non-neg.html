Proposition 8.1.8.

Let \((X,\mathcal{F},\mu)\) be a measure space, let \(f\) be non-negative measurable function on \(X\text{,}\) and let \(E \subseteq X\) be a measurable set with \(\mu(E) = 0\text{.}\) Then
🔗
\begin{equation*} \int_X f\, d\mu = \int_{X\backslash E} f\, d\mu. \end{equation*}
