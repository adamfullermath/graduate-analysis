Exercise 8.

Let \((X,\mathcal{F},\mu)\) be a measure space.
🔗
  1. Let \(f\) be a non-negative measurable function on \(X\text{.}\) Define \(\nu \colon \mathcal{F} \rightarrow [0,\infty]\) by
    \begin{equation*} \nu(E) = \int_E f\, d\mu. \end{equation*}
    Show that \(\nu\) is a measure on \((X,\mathcal{F})\) satisfying \(\nu(E) = 0\) when \(\mu(E) = 0\text{.}\)

  2. Let \(g\) an integrable function on \(X\) with respect to \(\mu\text{.}\) Define \(\sigma \colon \mathcal{F} \rightarrow \mathbb{R}\) by
    \begin{equation*} \sigma(E) = \int_E g \,d\mu. \end{equation*}
    Show that \(\sigma\) is a signed-measure on \((X,\mathcal{F})\) satisfying \(\sigma(E) = 0\) when \(\mu(E) = 0\text{.}\)
